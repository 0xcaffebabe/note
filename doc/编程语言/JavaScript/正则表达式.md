# 正则表达式

## 创建

```javascript
// 对象创建
var reg = new RegExp("正则表达式");
// 字面量创建
var reg = /正则表达式/;
```

## 使用

```javascript
reg.test('some text');
```

## 正则表达式参数

```js
/reg/[g] // 全局匹配
/reg/[i] // 忽略大小写
/reg/[gi] // 全局匹配+忽略大小写
```

## 正则替换

```js
'我的电话:0596-3766183'.replace(/\d{3,4}-\d{7,8}/,'****') // 我的电话:****
```
