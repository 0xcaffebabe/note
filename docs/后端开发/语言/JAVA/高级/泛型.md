# 自定义

```java
class Map<K>{
    // 修饰成员变量
    private K key;

    // 修饰参数
    public Map(K key){}

    // 修饰返回值
    public K get(){
        // 修饰局部变量
        K key1 = key;
        return key1;
    }
}
```

## 泛型方法

```java
    public static <T> T run(T obj){
        return obj;
    }
```

## 泛型限定

```java
// 约定T必须是Comparable的子类
<T extends Comparable> 
// 可同时指定多个父接口
<T extends Comparable&Serializable> 
```

## 通配符

```java
// 只能接受S的自身或子类
<? extends S>
// 能接收S自身及其超类
<? super S>
// 不限制类型，只能使用object接收
<?>
```
