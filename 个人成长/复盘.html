复盘

一、复盘的第一性原理

1.1 复盘的本质

复盘不是总结过去，而是更新认知结构。

其根本作用是：

将经验转化为可迁移的认知模型，并据此优化未来决策。

复盘是一个典型的认知反馈控制系统。

行动系统 → 结果 → 解释机制 → 认知模型 → 决策规则 → 新行动

复盘的价值不在于解释过去，而在于改变未来行为生成机制。

1.2 复盘解决的三类问题

层级关键问题本质类型
执行层如何把事情做得更好单环学习
决策层是否应该这样做双环学习
认知层我们理解世界的方式是否正确心智模型修正

真正有效的复盘必须至少触达决策层，优秀复盘必须进入认知层。

1.3 复盘的理论基础

复盘是多个学习理论的工程化实现：

  • 控制论（反馈调节）
  • 双环学习理论（Argyris）
  • 经验学习循环（Kolb）
  • 心智模型理论（Senge）
  • 行动学习理论（Revans）

因此复盘本质是：

组织学习系统的运行机制。

二、复盘系统总体架构

复盘不是一个流程，而是一个系统。

认知更新系统├ 方法模型├ 运行流程├ 组织结构├ 知识沉淀├ 治理机制└ 文化环境

任何缺失都会导致复盘失效。

三、复盘方法分层模型

不同复盘方法解决不同层级问题。

3.1 方法层级结构

层级目标典型方法输出
执行改进层提升效率PDCA流程优化
因果解释层找根因CLAP因果模型
认知校准层修正认知教练技术（GROW）心智更新
战略校准层调整目标PDF战略修正

3.2 各模型功能定位

PDCA —— 行动优化循环

关注：如何把既定目标做得更好。

适用：

  • 稳定业务
  • 流程优化
  • 执行改进

局限：不质疑目标合理性。

PDF —— 目标校准循环

关注：目标是否合理。

核心过程：目标 → 结果 → 分析 → 认知 → 新目标

作用：战略学习。

CLAP —— 因果认知循环

关注：解释机制。

过程：事实对比 → 因果逻辑 → 认知修正 → 行动规划

作用：建立可迁移规律。

教练技术（GROW）—— 心智干预机制

关注：

  • 情绪
  • 动机
  • 信念
  • 认知偏差

作用：提升认知质量。

3.3 方法组合原则

执行问题 → PDCA策略问题 → CLAP目标问题 → PDF认知阻塞 → 教练技术
