  2. еҒңжӯўе’Ңй”ҖжҜҒ Context е®№еҷЁе…іиҒ”зҡ„ Listener е’Ң Filter
  3. еҒңжӯўе’Ңй”ҖжҜҒ Context дёӢзҡ„ Pipeline е’Ңеҗ„з§Қ Valve
  4. еҒңжӯўе’Ңй”ҖжҜҒ Context зҡ„зұ»еҠ иҪҪеҷЁпјҢд»ҘеҸҠзұ»еҠ иҪҪеҷЁеҠ иҪҪзҡ„зұ»ж–Үд»¶иө„жәҗ
  5. еҗҜеҠЁ Context е®№еҷЁпјҢеңЁиҝҷдёӘиҝҮзЁӢдёӯдјҡйҮҚж–°еҲӣе»әеүҚйқўеӣӣжӯҘиў«й”ҖжҜҒзҡ„иө„жәҗ
