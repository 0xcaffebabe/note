InnoDB жңүдёҖдёӘеҗҺеҸ°зәҝзЁӢпјҢжҜҸйҡ” 1 з§’пјҢе°ұдјҡжҠҠ redo log buffer дёӯзҡ„ж—Ҙеҝ—пјҢи°ғз”Ё write еҶҷеҲ°ж–Үд»¶зі»з»ҹзҡ„ page cacheпјҢз„¶еҗҺи°ғз”Ё fsync жҢҒд№…еҢ–еҲ°зЈҒзӣҳпјҢйҷӨжӯӨд№ӢеӨ–пјҢredo log buffer еҚ з”Ёзҡ„з©әй—ҙеҚіе°ҶиҫҫеҲ° innodb_log_buffer_size дёҖеҚҠзҡ„ж—¶еҖҷпјҢеҗҺеҸ°зәҝзЁӢдјҡдё»еҠЁеҶҷзӣҳпјҢеҸҰдёҖз§ҚжҳҜпјҢе№¶иЎҢзҡ„дәӢеҠЎжҸҗдәӨзҡ„ж—¶еҖҷпјҢйЎәеёҰе°ҶиҝҷдёӘдәӢеҠЎзҡ„ redo log buffer жҢҒд№…еҢ–еҲ°зЈҒзӣҳ

  • и®ҫзҪ®дёә 1 зҡ„ж—¶еҖҷпјҢиЎЁзӨәжҜҸж¬ЎдәӢеҠЎжҸҗдәӨж—¶йғҪе°Ҷ redo log зӣҙжҺҘжҢҒд№…еҢ–еҲ°зЈҒзӣҳ
  • и®ҫзҪ®дёә 2 зҡ„ж—¶еҖҷпјҢиЎЁзӨәжҜҸж¬ЎдәӢеҠЎжҸҗдәӨж—¶йғҪеҸӘжҳҜжҠҠ redo log еҶҷеҲ° page cache

DDL жңәеҲ¶

еңЁдҪҝз”Ё alter table иҜӯеҸҘж—¶пјҢеҸҜд»ҘйҖҡиҝҮе…ій”®еӯ— algorithm жқҘжҢҮе®ҡдҪҝз”Ёд»Җд№Ҳзұ»еһӢзҡ„ DDLпјҡ

  • DEFAULTпјҡй»ҳи®Өж–№ејҸпјҢдёҚеҗҢзҡ„ DDL зұ»еһӢпјҢй»ҳи®Өзҡ„жү§иЎҢж–№ејҸеҸҜиғҪдёҚдёҖж ·гҖӮMySQL дјҡд»ҘиҜҘ DDL ејҖй”ҖжңҖдҪҺзҡ„ж–№ејҸжқҘйҖүжӢ©й»ҳи®Өзҡ„жү§иЎҢж–№ејҸгҖӮ
  • INSTANTпјҡMySQL 8.0 ж–°еҠ зҡ„жү§иЎҢж–№ејҸпјҢдҪҝз”Ёиҝҷз§Қжү§иЎҢж–№ејҸж—¶пјҢж·»еҠ еӯ—ж®өе’ҢеҲ йҷӨеӯ—ж®өж—¶дёҚйңҖиҰҒйҮҚе»әиЎЁгҖӮALGORITHM жҢҮе®ҡдёә INSTANT ж—¶пјҢдёҚиғҪеҶҚжҢҮе®ҡ LOCK е…ій”®еӯ—гҖӮ
  • INPLACEпјҡOnline DDLгҖӮдҪҝз”Ё INPLACE ж—¶пјҢй»ҳи®ӨдёҚй”ҒиЎЁгҖӮиҝҷйҮҢзҡ„дёҚй”ҒиЎЁпјҢжҳҜжҢҮйҮҚе»әиЎЁж—¶еӨҚеҲ¶ж•°жҚ®зҡ„иҝҮзЁӢдёӯпјҢжҲ–еҲӣе»әдәҢзә§зҙўеј•ж—¶иҜ»еҸ–е…ЁиЎЁж•°жҚ®иҝӣиЎҢжҺ’еәҸгҖҒз”ҹжҲҗзҙўеј•зҡ„иҝҮзЁӢдёӯдёҚй”ҒиЎЁгҖӮдҪҶжҳҜеңЁеӨҚеҲ¶ж•°жҚ®жҲ–з”ҹжҲҗзҙўеј•зҡ„иҝҮзЁӢдёӯпјҢиЎЁдёҠдјҡжңүж–°зҡ„ DML дҝ®ж”№ж•°жҚ®пјҢиҝҷдәӣдҝ®ж”№дјҡи®°еҪ•еҲ°дёҖдёӘеңЁзәҝзҡ„еҸҳжӣҙж—Ҙеҝ—дёӯгҖӮInnoDB йңҖиҰҒе°ҶеҸҳжӣҙж—Ҙеҝ—дёӯзҡ„еҶ…е®№жӣҙж–°еҲ°ж–°зҡ„иЎЁжҲ–зҙўеј•дёӯпјҢиҖҢиҝҷдёӘиҝҮзЁӢдёӯжҳҜдјҡй”ҒиЎЁзҡ„гҖӮ
  • COPYпјҡдј з»ҹзҡ„ DDL жү§иЎҢж–№ејҸпјҢжү§иЎҢиҝҮзЁӢдёӯдјҡй”ҒиЎЁпјҢй»ҳи®Өй”ҒжЁЎејҸдёә SHAREDпјҢеә”з”ЁзЁӢеәҸеҸҜд»ҘиҜ»еҸ–иЎЁдёӯзҡ„ж•°жҚ®пјҢдҪҶжҳҜдёҚиғҪеҶҷе…Ҙж•°жҚ®гҖӮеҰӮжһң LOCK жҢҮе®ҡдёә EXCLUSIVEпјҢйӮЈд№ҲиҜ»ж“ҚдҪңд№ҹдјҡиў«йҳ»еЎһ
