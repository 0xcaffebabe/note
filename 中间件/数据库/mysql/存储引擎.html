1.1 еӯҳеӮЁеј•ж“Һзҡ„еӣӣдёӘж ёеҝғй—®йўҳпјҲзЁіе®ҡжЁЎеһӢпјү

з»ҙеәҰ ж ёеҝғй—®йўҳ жң¬иҙЁзҹӣзӣҫ
ж•°жҚ®з»„з»Ү ж•°жҚ®еҰӮдҪ•еңЁзЈҒзӣҳдёҠз»„з»Ү йЎәеәҸжҖ§ vs йҡҸжңәжҖ§
е№¶еҸ‘жҺ§еҲ¶ еӨҡдәӢеҠЎеҰӮдҪ•еҗҢж—¶иҜ»еҶҷ е№¶еҸ‘жҖ§ vs дёҖиҮҙжҖ§
еҙ©жәғжҒўеӨҚ е®•жңәеҗҺеҰӮдҪ•дҝқиҜҒж•°жҚ®жӯЈзЎ® жҖ§иғҪ vs е®үе…Ё
жҖ§иғҪжқғиЎЎ еҰӮдҪ•з”ЁеҶ…еӯҳжҚўзЈҒзӣҳ IO жҲҗжң¬ vs ж•ҲзҺҮ

йҮҚиҰҒи®ӨзҹҘпјҡ еӯҳеӮЁеј•ж“Һд№Ӣй—ҙзҡ„е·®ејӮпјҢжң¬иҙЁдёҠдёҚжҳҜвҖңеҠҹиғҪеӨҡеҜЎвҖқпјҢиҖҢжҳҜ еңЁиҝҷдәӣзҹӣзӣҫдёҠзҡ„еҸ–иҲҚдёҚеҗҢгҖӮ

дәҢгҖҒMySQL еӯҳеӮЁеј•ж“Һе…ЁжҷҜпјҲиғҪеҠӣе®ҡдҪҚи§Ҷи§’пјү

2.1 еј•ж“Һи§’иүІе®ҡдҪҚпјҲиҖҢйқһз®ҖеҚ•жҸҸиҝ°пјү

еӯҳеӮЁеј•ж“Һ ж ёеҝғе®ҡдҪҚ е…ій”®еҸ–иҲҚ
InnoDB йҖҡз”ЁдәӢеҠЎеһӢеј•ж“Һ жҖ§иғҪжҚўдёҖиҮҙжҖ§
MyISAM еҸӘиҜ» / иҜ»еӨҡеҶҷе°‘ ж”ҫејғдәӢеҠЎжҚўз®ҖеҚ•й«ҳж•Ҳ

InnoDB е№¶дёҚжҳҜдёҖе Ҷйӣ¶ж•ЈжңәеҲ¶пјҢиҖҢжҳҜдёҖдёӘ й«ҳеәҰеҲҶеұӮзҡ„зі»з»ҹгҖӮ

  • дё»й”®и®ҫи®ЎжһҒе…¶йҮҚиҰҒ

иҝҷдёҚжҳҜ InnoDB зҡ„е®һзҺ°йҖүжӢ©пјҢиҖҢжҳҜ дәӢеҠЎеһӢеӯҳеӮЁеј•ж“Һзҡ„еҝ…з„¶йҖүжӢ©гҖӮ

5.1 Buffer Pool зҡ„жң¬иҙЁ

Buffer Pool дёҚжҳҜзј“еӯҳпјҢиҖҢжҳҜ InnoDB зҡ„вҖңдё»жҲҳеңәвҖқгҖӮ

  • жүҖжңүиҜ»еҶҷжңҖз»ҲйғҪиҰҒз»ҸиҝҮ Buffer Pool
  • зЈҒзӣҳеҸӘжҳҜжҢҒд№…еҢ–еӘ’д»Ӣ

5.2 ж”№иҝӣ LRU зҡ„и®ҫи®Ўе“ІеӯҰ

ж ёеҝғй—®йўҳпјҡ

еҰӮдҪ•йҳІжӯўдёҖж¬Ўе…ЁиЎЁжү«жҸҸеҶІеһ®зғӯзӮ№ж•°жҚ®пјҹ

и§ЈеҶіж–№ејҸпјҡеҲҶд»Ј + дёӯзӮ№жҸ’е…Ҙ

  • NewпјҲзғӯеҢәпјүпјҡзңҹжӯЈзҡ„зғӯзӮ№ж•°жҚ®
  • OldпјҲеҶ·еҢәпјүпјҡдёҖж¬ЎжҖ§гҖҒжү«жҸҸеһӢж•°жҚ®

и®ҫи®ЎжқғиЎЎпјҡ

  • еҫ—еҲ°пјҡзғӯзӮ№ж•°жҚ®зЁіе®ҡе‘Ҫдёӯ
  • д»Јд»·пјҡе®һзҺ°еӨҚжқӮеәҰдёҠеҚҮ

  • еҶҷеӨҡиҜ»е°‘

  • йў‘з№ҒиҜ»еҸ–еҗҢдёҖдәҢзә§зҙўеј•
