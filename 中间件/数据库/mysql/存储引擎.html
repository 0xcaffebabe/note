еҪ“йңҖиҰҒжӣҙж–°дёҖдёӘж•°жҚ®йЎөж—¶пјҢеҰӮжһңж•°жҚ®йЎөеңЁеҶ…еӯҳдёӯе°ұзӣҙжҺҘжӣҙж–°пјҢеҰӮжһңдёҚеңЁеҶ…еӯҳдёӯпјҢеңЁдёҚеҪұе“Қж•°жҚ®дёҖиҮҙжҖ§зҡ„жғ…еҶөдёӢпјҢдјҡе°Ҷжӣҙж–°зҡ„ж“ҚдҪңзј“еӯҳеңЁ Change Buffer йҮҢйқўпјҢеҪ“иҜ»еҸ–иҝҷдәӣеҸ—еҪұе“Қзҡ„ж•°жҚ®ж—¶пјҢдјҡжү§иЎҢ merge ж“ҚдҪңпјҢеҜ№ж•°жҚ®еә”з”Ёиҝҷдәӣж“ҚдҪңпјҢеҗҢж—¶ MySQL д№ҹдјҡе®ҡжңҹе°Ҷиҝҷдәӣжӣҙж–°иҪ¬дёәеҜ№зү©зҗҶж•°жҚ®зҡ„зңҹжӯЈжӣҙж–°

Change Buffer дјҡиў«жҢҒд№…еҢ–еҲ°зЈҒзӣҳдёӯпјҢдәӢеҠЎжҸҗдәӨзҡ„ж—¶еҖҷпјҢChange Buffer зҡ„ж“ҚдҪңд№ҹдјҡиў«и®°еҪ•еҲ° redo log йҮҢ

иҝҷдёӘзј“еҶІпјҢеҜ№дәҺеҶҷеӨҡиҜ»е°‘зҡ„дёҡеҠЎж•ҲжһңжңҖеҘҪпјҢеҰӮжһңиҜ»зҡ„жҜ”иҫғеӨҡпјҢжҜҸж¬ЎиҜ»еҸ–йғҪиҰҒи§ҰеҸ‘ merge ж“ҚдҪңпјҢд№ҹе°ұжҳҜд»ҺзЈҒзӣҳеҶҚеҺ»иҜ»ж•°жҚ®

Double Write Buffer

жңүдәӣзі»з»ҹдёӯпјҢInnoDB зј“еӯҳйЎөеҶҷеҲ°зЈҒзӣҳи®ҫеӨҮж—¶пјҢдёҚжҳҜдёҖдёӘеҺҹеӯҗж“ҚдҪңгҖӮдёҖдёӘзј“еҶІйЎөеҜ№еә”дәҶеӨҡдёӘзЈҒзӣҳжүҮеҢәпјҢеҸҜиғҪдјҡеӯҳеңЁйғЁеҲҶжүҮеҢәеҶҷжҲҗеҠҹпјҢйғЁеҲҶжүҮеҢәеҶҷеӨұиҙҘзҡ„жғ…еҶөгҖӮ

дёәдәҶйҒҝе…ҚеҮәзҺ°иҝҷз§Қжғ…еҶөпјҢInnoDB дҪҝз”ЁдәҶ Double Write BufferгҖӮи„ҸйЎөеҲ·ж–°ж—¶пјҢе…ҲеҶҷеҲ° Double Write BufferпјҢз„¶еҗҺеҶҚе°Ҷ Double Write Buffer дёӯзҡ„йЎөйқўжү№йҮҸеҲ·ж–°еҲ° Double Write ж–Үд»¶дёӯпјҢеҰӮжһңиҝҷдёӘиҝҮзЁӢеӨұиҙҘдәҶпјҢйӮЈд№Ҳж•°жҚ®ж–Үд»¶дёӯзҡ„йЎөйқўиҝҳжҳҜеҘҪзҡ„гҖӮеҰӮжһңеҶҷ Double Write ж–Үд»¶жҲҗеҠҹдәҶпјҢдҪҶжҳҜеҗҺз»ӯеңЁеҶҷж•°жҚ®ж–Үд»¶ж—¶еӨұиҙҘдәҶпјҢйӮЈд№Ҳе°ұеҸҜд»Ҙз”Ё Double Write ж–Үд»¶дёӯзҡ„еҶ…е®№жқҘжҒўеӨҚж•°жҚ®
