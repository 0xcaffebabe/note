дёҖдёӘеӯ—ж®өйҖүжӢ©дәҶд»Җд№Ҳзұ»еһӢпјҢе®һйҷ…дёҠеҗҢж—¶еҶіе®ҡдәҶпјҡ

  • **ж•°жҚ®еҰӮдҪ•иў«еӯҳеӮЁ**пјҲе®ҡй•ҝ / еҸҳй•ҝ / еӨ–йғЁеӯҳеӮЁпјү
  • **ж•°жҚ®еҰӮдҪ•иў«жҜ”иҫғдёҺжҺ’еәҸ**пјҲеӯ—иҠӮжҜ”иҫғ / иҜӯд№үжҜ”иҫғ / жҺ’еәҸи§„еҲҷпјү
  • **ж•°жҚ®еҰӮдҪ•еҸӮдёҺи®Ўз®—**пјҲзІҫеәҰгҖҒжәўеҮәгҖҒйҡҗејҸиҪ¬жҚўпјү
  • **ж•°жҚ®жҳҜеҗҰгҖҒд»ҘеҸҠеҰӮдҪ•иў«зҙўеј•**

з»“и®әпјҡ ж•°жҚ®зұ»еһӢ вү  вҖңеҖјзҡ„ж јејҸвҖқ ж•°жҚ®зұ»еһӢ = еӯҳеӮЁжЁЎеһӢ + и®Ўз®—иҜӯд№ү + жҜ”иҫғи§„еҲҷ + зҙўеј•иғҪеҠӣ

иҝҷд№ҹжҳҜдёәд»Җд№Ҳж•°жҚ®зұ»еһӢйҖүжӢ©й”ҷиҜҜпјҢеҫҖеҫҖдјҡеңЁжҖ§иғҪгҖҒжӯЈзЎ®жҖ§е’Ңзі»з»ҹжј”иҝӣдёӯеҸҚеӨҚд»ҳеҮәд»Јд»·гҖӮ

дәҢгҖҒж•°жҚ®зұ»еһӢзҡ„з»ҹдёҖжҠҪиұЎжЁЎеһӢпјҲж ёеҝғпјү

жүҖжңүе…ізі»еһӢж•°жҚ®еә“зҡ„ж•°жҚ®зұ»еһӢпјҢйғҪеҸҜд»Ҙжҳ е°„еҲ°д»ҘдёӢеӣӣдёӘзЁіе®ҡз»ҙеәҰпјҡ

1пёҸвғЈ еӯҳеӮЁжЁЎеһӢпјҲStorage Modelпјү

  • е®ҡй•ҝеӯҳеӮЁпјҲFixed-lengthпјү
  • еҸҳй•ҝеӯҳеӮЁпјҲVariable-lengthпјү
  • еӨ–йғЁеӯҳеӮЁ / жҢҮй’Ҳеј•з”ЁпјҲOff-page storageпјү

еӯҳеӮЁжЁЎеһӢеҶіе®ҡдәҶ з©әй—ҙж•ҲзҺҮгҖҒIO иЎҢдёәгҖҒUPDATE жҲҗжң¬

2пёҸвғЈ жҜ”иҫғдёҺжҺ’еәҸиҜӯд№үпјҲComparison Semanticsпјү

  • еӯ—иҠӮзә§жҜ”иҫғ
  • еҹәдәҺжҺ’еәҸи§„еҲҷпјҲcollationпјүзҡ„иҜӯд№үжҜ”иҫғ
  • жҳҜеҗҰеҸ—ж—¶еҢәгҖҒзј–з ҒгҖҒеҢәеҹҹи§„еҲҷеҪұе“Қ

жҜ”иҫғиҜӯд№үеҶіе®ҡдәҶ ORDER BYгҖҒGROUP BYгҖҒJOIN зҡ„иЎҢдёәжӯЈзЎ®жҖ§

3пёҸвғЈ и®Ўз®—иҜӯд№үпјҲComputation Semanticsпјү

  • жҳҜеҗҰзІҫзЎ®
  • жҳҜеҗҰеӯҳеңЁиҲҚе…ҘиҜҜе·®
  • жәўеҮәдёҺйҡҗејҸзұ»еһӢиҪ¬жҚўи§„еҲҷ

и®Ўз®—иҜӯд№үеҶіе®ҡдәҶ ж•°жҚ®жҳҜеҗҰвҖңеҸҜдҝЎвҖқ

4пёҸвғЈ зҙўеј•дёҺжҹҘиҜўиғҪеҠӣпјҲIndexabilityпјү

  • жҳҜеҗҰеҸҜе»әз«Ӣзҙўеј•
  • жҳҜеҗҰж”ҜжҢҒеүҚзјҖзҙўеј•
  • жҳҜеҗҰеҪұе“Қзҙўеј•еӨ§е°ҸдёҺйҖүжӢ©жҖ§

зҙўеј•иғҪеҠӣеҶіе®ҡдәҶ жҹҘиҜўжҖ§иғҪзҡ„дёҠйҷҗ

дёүгҖҒж•°еҖјзұ»еһӢпјҡзІҫеәҰгҖҒиҢғеӣҙдёҺеҸҜдҝЎи®Ўз®—

3.1 ж•°еҖјзұ»еһӢзҡ„жң¬иҙЁеҲҶзұ»

д»Һи®Ўз®—иҜӯд№үи§’еәҰпјҢж•°еҖјзұ»еһӢеҸӘжңүдёӨеӨ§зұ»пјҡ

зұ»еһӢ жң¬иҙЁ
зІҫзЎ®ж•°еҖј з”ЁдәҺвҖңдёҚиғҪй”ҷвҖқзҡ„и®Ўз®—
йқһзІҫзЎ®ж•°еҖј з”ЁдәҺвҖңеҸҜд»Ҙиҝ‘дјјвҖқзҡ„и®Ўз®—

3.2 зІҫзЎ®ж•°еҖјзұ»еһӢпјҲExactпјү

д»ЈиЎЁпјҡж•ҙж•°зұ»гҖҒе®ҡзӮ№е°Ҹж•°зұ»

зү№еҫҒпјҡ

  • иҝҗз®—з»“жһңзЎ®е®ҡ
  • дёҚеӯҳеңЁдәҢиҝӣеҲ¶иҲҚе…ҘиҜҜе·®
  • йҖӮеҗҲйҮ‘йўқгҖҒи®Ўж•°гҖҒзүҲжң¬еҸ·гҖҒеәҸеҲ—

е·ҘзЁӢе“ІеӯҰпјҡ

еҮЎжҳҜдёҡеҠЎдёҠвҖңдёҚиғҪз®—й”ҷвҖқзҡ„ж•°жҚ®пјҢеҝ…йЎ»дҪҝз”ЁзІҫзЎ®зұ»еһӢ

3.3 йқһзІҫзЎ®ж•°еҖјзұ»еһӢпјҲApproximateпјү

д»ЈиЎЁпјҡжө®зӮ№ж•°

зү№еҫҒпјҡ

  • и®Ўз®—йҖҹеәҰеҝ«
  • еӯҳеңЁзІҫеәҰиҜҜе·®
  • жҜ”иҫғз»“жһңдёҚзЁіе®ҡ

е·ҘзЁӢе“ІеӯҰпјҡ

жө®зӮ№ж•°йҖӮеҗҲз§‘еӯҰи®Ўз®—пјҢдёҚйҖӮеҗҲдёҡеҠЎиҙҰзӣ®

3.4 ж•°еҖјиҢғеӣҙдёҺ UNSIGNED зҡ„ж„Ҹд№ү

  • еӯҳеӮЁдҪҚж•°еҶіе®ҡеҖјеҹҹ
  • UNSIGNED е№¶дёҚжҳҜвҖңдјҳеҢ–вҖқпјҢиҖҢжҳҜ**иҜӯд№үеЈ°жҳҺ**

дҪҝз”Ё UNSIGNEDпјҢзӯүд»·дәҺеҗ‘зі»з»ҹеЈ°жҳҺпјҡ иҜҘеӯ—ж®өдёҚеә”еҮәзҺ°иҙҹеҖј

еӣӣгҖҒеӯ—з¬ҰдёҺдәҢиҝӣеҲ¶зұ»еһӢпјҡеӯҳеӮЁгҖҒиҜӯд№үдёҺзҙўеј•

4.1 еӯ—з¬ҰдёІзұ»еһӢзҡ„ж ёеҝғеҲҶжӯ§

еӯ—з¬Ұзұ»еһӢзҡ„ж ёеҝғе·®ејӮдёҚеңЁвҖңжҳҜеҗҰеӯ—з¬ҰдёІвҖқпјҢиҖҢеңЁпјҡ

  • жҳҜеҗҰе®ҡй•ҝ
  • жҳҜеҗҰе…·еӨҮиҜӯд№үжҜ”иҫғи§„еҲҷ

4.2 е®ҡй•ҝ vs еҸҳй•ҝ

з»ҙеәҰ е®ҡй•ҝ еҸҳй•ҝ
еӯҳеӮЁ еӣәе®ҡ жҢүйңҖ
UPDATE дҪҺжҲҗжң¬ еҸҜиғҪдә§з”ҹйўқеӨ–ејҖй”Җ
з©әй—ҙж•ҲзҺҮ дҪҺ й«ҳ
йҖӮз”ЁеңәжҷҜ й•ҝеәҰзЁіе®ҡ й•ҝеәҰдёҚзЎ®е®ҡ

жң¬иҙЁжқғиЎЎпјҡ з©әй—ҙж•ҲзҺҮ вҶ” жӣҙж–°жҲҗжң¬ вҶ” и®ҝй—®еұҖйғЁжҖ§

4.3 дәҢиҝӣеҲ¶дёҺйқһдәҢиҝӣеҲ¶еӯ—з¬ҰдёІ

  • дәҢиҝӣеҲ¶зұ»еһӢпјҡйҖҗеӯ—иҠӮжҜ”иҫғ
  • йқһдәҢиҝӣеҲ¶зұ»еһӢпјҡеҹәдәҺжҺ’еәҸи§„еҲҷжҜ”иҫғ

еҪұе“Қпјҡ

  • жҺ’еәҸ
  • зҙўеј•
  • JOIN иЎҢдёә

4.4 еӨ§еӯ—ж®өпјҲLOBпјүзҡ„е·ҘзЁӢеҗ«д№ү

еӨ§еӯ—ж®өзұ»еһӢзҡ„жң¬иҙЁжҳҜпјҡ

вҖңзүәзүІзҙўеј•иғҪеҠӣпјҢжҚўеҸ–е®№йҮҸвҖқ

е·ҘзЁӢеҺҹеҲҷпјҡ

  • е®ғд»¬дёҚжҳҜвҖңжҷ®йҖҡеӯ—ж®өвҖқ
  • иҖҢжҳҜвҖңйҷ„еұһж•°жҚ®вҖқ

дә”гҖҒж—¶й—ҙдёҺж—Ҙжңҹзұ»еһӢпјҡж—¶й—ҙиҜӯд№үиҖҢйқһж јејҸ

5.1 ж—¶й—ҙзұ»еһӢзҡ„ж ёеҝғй—®йўҳ

ж—¶й—ҙзұ»еһӢе№¶дёҚйҡҫпјҢйҡҫзҡ„жҳҜж—¶й—ҙиҜӯд№үпјҡ

  • жҳҜеҗҰдёҺж—¶еҢәз»‘е®ҡ
  • жҳҜеҗҰиЎЁзӨәвҖңдәӢд»¶еҸ‘з”ҹж—¶еҲ»вҖқ
  • жҳҜеҗҰиЎЁзӨәвҖңдёҡеҠЎж—¶й—ҙвҖқ

5.2 дёӨз§Қж—¶й—ҙе“ІеӯҰ

е“ІеӯҰ еҗ«д№ү
з»қеҜ№ж—¶й—ҙ дёҺж—¶еҢәзӣёе…і
дёҡеҠЎж—¶й—ҙ дёҺдёҡеҠЎиҜӯд№үзӣёе…і

ж—¶й—ҙзұ»еһӢйҖүжӢ©пјҢжң¬иҙЁжҳҜеҜ№ж—¶й—ҙж„Ҹд№үзҡ„е»әжЁЎйҖүжӢ©

е…ӯгҖҒжһҡдёҫдёҺдҪҚзұ»еһӢпјҡзј–з Ғж•ҲзҺҮ vs жј”иҝӣиғҪеҠӣ

6.1 жһҡдёҫзҡ„жң¬иҙЁ

жһҡдёҫе№¶дёҚжҳҜеӯ—з¬ҰдёІдјҳеҢ–пјҢиҖҢжҳҜпјҡ

вҖңз”Ёж•ҙж•°зј–з ҒжңүйҷҗзҠ¶жҖҒйӣҶеҗҲвҖқ

дјҳзӮ№пјҡ

  • еӯҳеӮЁзҙ§еҮ‘
  • жҜ”иҫғй«ҳж•Ҳ

д»Јд»·пјҡ

  • жһ¶жһ„еҲҡжҖ§
  • жј”иҝӣжҲҗжң¬й«ҳ

6.2 дҪҚдёҺйӣҶеҗҲзұ»еһӢ

  • йҖӮеҗҲиЎЁзӨәеёғе°”з»„еҗҲ
  • дёҚйҖӮеҗҲиЎЁиҫҫеӨҚжқӮиҜӯд№ү

еҪ“дҪҚеҗ«д№үи¶…иҝҮ 3пҪһ5 дёӘпјҢзі»з»ҹеӨҚжқӮеәҰдјҡжҖҘеү§дёҠеҚҮ

дёғгҖҒеәҸеҲ—дёҺиҮӘеўһпјҡе№¶еҸ‘зі»з»ҹдёӯзҡ„йЎәеәҸй—®йўҳ

7.1 еәҸеҲ—з”ҹжҲҗзҡ„жң¬иҙЁйҡҫйўҳ

з”ҹжҲҗеәҸеҲ—е№¶дёҚйҡҫпјҢйҡҫзҡ„жҳҜеңЁд»ҘдёӢзәҰжқҹдёӢз”ҹжҲҗпјҡ

  • й«ҳе№¶еҸ‘
  • еҙ©жәғжҒўеӨҚ
  • дёҚйҳ»еЎһдёҡеҠЎ

иҝһз»ӯжҖ§гҖҒе”ҜдёҖжҖ§гҖҒжҖ§иғҪпјҢдёүиҖ…дёҚеҸҜе…јеҫ—

7.2 AUTO_INCREMENT зҡ„е·ҘзЁӢе®ҡдҪҚ

AUTO_INCREMENT жҳҜдёҖз§Қпјҡ

еңЁеҚ•жңәж•°жҚ®еә“еҶ…пјҢе№іиЎЎе№¶еҸ‘жҖ§иғҪдёҺйЎәеәҸжҖ§зҡ„е·ҘзЁӢеҰҘеҚҸж–№жЎҲ

е·ҘзЁӢи®ӨзҹҘпјҡ

  • иҝһз»ӯ вү  жӯЈзЎ®
  • й—ҙйҡҷ вү  Bug

7.3 дёҚиҝһз»ӯ ID зҡ„жң¬иҙЁеҺҹеӣ 

  • е№¶еҸ‘йў„еҲҶй…Қ
  • еӣһж»ҡдёҚеӣһж”¶
  • жү№йҮҸжҸ’е…ҘдјҳеҢ–

е…ій”®з»“и®әпјҡ

ID зҡ„иҜӯд№үжҳҜвҖңе”ҜдёҖж ҮиҜҶвҖқпјҢдёҚжҳҜвҖңдёҡеҠЎйЎәеәҸвҖқ

е…«гҖҒж•°жҚ®зұ»еһӢйҖүеһӢзҡ„ж–№жі•и®әпјҲеҶізӯ–еұӮпјү

8.1 зұ»еһӢйҖүжӢ©зҡ„еӣӣй—®жі•

  1. жҳҜеҗҰиҰҒжұӮ**и®Ўз®—з»қеҜ№жӯЈзЎ®**
  2. жҳҜеҗҰеҸӮдёҺ**жҺ’еәҸ / JOIN / зҙўеј•**
  3. жҳҜеҗҰйңҖиҰҒ**й•ҝжңҹжј”иҝӣ**
  4. жҳҜеҗҰжҳҜ**дёҡеҠЎж ёеҝғеӯ—ж®ө**

8.2 еҶізӯ–зӨәдҫӢ

  • йҮ‘йўқ вҶ’ зІҫзЎ®ж•°еҖј
  • зҠ¶жҖҒ вҶ’ жһҡдёҫ or зӢ¬з«Ӣеӯ—е…ёиЎЁ
  • ж—¶й—ҙзӮ№ вҶ’ е…ҲжҳҺзЎ®вҖңж—¶й—ҙиҜӯд№үвҖқ
  • дё»й”® вҶ’ е”ҜдёҖжҖ§дјҳе…ҲдәҺиҝһз»ӯжҖ§

жҖ»з»“пјҡж•°жҚ®зұ»еһӢжҳҜж•°жҚ®еә“зҡ„вҖңиҜӯиЁҖзі»з»ҹвҖқ

еҰӮжһңжҠҠж•°жҚ®еә“зңӢжҲҗдёҖз§ҚиҜӯиЁҖпјҢйӮЈд№Ҳж•°жҚ®зұ»еһӢе°ұжҳҜе®ғзҡ„иҜӯжі•дёҺиҜӯд№үзі»з»ҹгҖӮ

  • е®ғеҶіе®ҡдәҶзі»з»ҹиғҪ"иҜҙд»Җд№Ҳ"
  • д№ҹеҶіе®ҡдәҶзі»з»ҹ"дёҚиғҪиҜҙд»Җд№Ҳ"

зңҹжӯЈжҲҗзҶҹзҡ„ж•°жҚ®еә“и®ҫи®ЎпјҢдёҚжҳҜи®°дҪҸзұ»еһӢеҲ—иЎЁпјҢиҖҢжҳҜзҗҶи§ЈжҜҸдёҖз§Қзұ»еһӢиғҢеҗҺзҡ„еҸ–иҲҚдёҺе“ІеӯҰгҖӮ
