
# CSS选择器

## 扩展选择器

```css
/* 选择所有 */
*{...} 

/* 并集选择器 */
选择器1,选择器2{}

/* 子选择器 */
选择器1 选择器2{}

/* 父选择器 */
选择器1>选择器2{}

/* 属性选择器 */
元素[属性=值]{}

/* 伪类选择器 */
元素:状态{}
```
