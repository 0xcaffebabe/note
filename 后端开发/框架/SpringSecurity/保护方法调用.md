# 使用注解保护方法

- 配置

```java
@Configuration
@EnableGlobalMethodSecurity(securedEnabled = true)
class Config1 extends GlobalMethodSecurityConfiguration{

}
```

## @Secured

```java
@Secured("ROLE_ADMIN")
    @RequestMapping("/home")
    @ResponseBody
    public String home(){
        return "home";
    }
```
