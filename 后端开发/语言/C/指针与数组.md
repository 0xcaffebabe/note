# 指针与地址

```c
p = &c; // p为指向c的指针
y = *p; // 现在y的值的c的值
```

# 指针与函数参数

```c
void swap(int *x,int *y){
    int tmp = *x;
    *x = *y;
    *y = tmp;
}
```

# 字符串与指针
```c
void reverse(char *s,int n){
    if (n<0){
        return;
    }
    putchar(*(s+n));
    reverse(s,--n);
}
```
