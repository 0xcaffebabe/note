# 循环结构

- for循环

```java
for(初始化表达式①; 布尔表达式②; 步进表达式④){
     循环体③
}
```

- while循环

```java
初始化表达式①   
while(布尔表达式②){
         循环体③     
     步进表达式④ 
}
```

- do...while循环

```java
初始化表达式①     
do{     
    循环体③     
    步进表达式④ 
}while(布尔表达式②)
```
