- 静态嵌套类，类前面有static修饰符
- 非静态嵌套类，又称内部类
  - 普通内部类
  - 局部内部类
  - 匿名内部类

```java
public class Main {

    // 静态嵌套类
    static class Inner1{ }
    
    // 普通内部类
    class Inner2{}
    
    public void f(){
        // 局部内部类
        class Inner3{}
        
        // 匿名内部类
        new Object(){};
    }
    
}
```

# 使用理由

- 细粒度访问控制
- 避免过多类定义
