дёҖгҖҒй—®йўҳиғҢжҷҜпјҡдёәд»Җд№ҲеҶ…ж ёйңҖиҰҒдёҖз§Қ eBPF жңәеҲ¶

1.1 еҶ…ж ёжј”еҢ–зҡ„ж №жң¬зҹӣзӣҫ

зҺ°д»Јж“ҚдҪңзі»з»ҹеҶ…ж ёй•ҝжңҹйқўдёҙдёүз»„з»“жһ„жҖ§зҹӣзӣҫпјҡ

  1. зЁіе®ҡжҖ§ vs еҸҜжү©еұ•жҖ§

    • еҶ…ж ёйңҖиҰҒжһҒй«ҳзЁіе®ҡжҖ§
    • дҪҶи§ӮжөӢгҖҒи°ғеәҰгҖҒзҪ‘з»ңгҖҒе®үе…Ёзӯ–з•ҘеҚҙй«ҳеәҰеҠЁжҖҒ

  2. е®үе…ЁжҖ§ vs зҒөжҙ»жҖ§

    • д»»ж„Ҹд»Јз Ғиҝӣе…ҘеҶ…ж ёжҳҜдёҚеҸҜжҺҘеҸ—зҡ„
    • дҪҶеӣәе®ҡжҺҘеҸЈж— жі•иҰҶзӣ–еҝ«йҖҹеҸҳеҢ–зҡ„йңҖжұӮ

  3. жҖ§иғҪ vs еҸҜи§ӮжөӢжҖ§

    • ptraceгҖҒд»ЈзҗҶиҝӣзЁӢзӯүж–№жЎҲжҖ§иғҪдёҚеҸҜжҺҘеҸ—
    • дҪҶеҶ…ж ёеҸҲзјәд№ҸеҠЁжҖҒи§ӮжөӢиғҪеҠӣ

1.2 дј з»ҹи§ЈеҶіж–№жЎҲзҡ„з»“жһ„жҖ§зјәйҷ·

ж–№жЎҲ ж №жң¬й—®йўҳ
еҶ…ж ёжЁЎеқ— йЈҺйҷ©й«ҳгҖҒи°ғиҜ•еӣ°йҡҫгҖҒеҚҮзә§жҲҗжң¬еӨ§
sysctl / еӣәе®ҡ hook еңәжҷҜжңүйҷҗпјҢжү©еұ•жҖ§е·®
ptrace / LD_PRELOAD жҖ§иғҪе·®гҖҒдҫөе…ҘжҖ§ејә
дё“з”Ёеӯҗзі»з»ҹпјҲеҰӮ netfilterпјү иғҪеҠӣејәдҪҶйўҶеҹҹе°Ғй—ӯ

з»“и®әпјҡеҶ…ж ёйңҖиҰҒдёҖз§ҚгҖҢеҸ—жҺ§зҡ„еҠЁжҖҒжү©еұ•жңәеҲ¶гҖҚгҖӮ

дәҢгҖҒeBPF зҡ„з¬¬дёҖжҖ§еҺҹзҗҶ

2.1 eBPF зҡ„жң¬иҙЁе®ҡд№ү

eBPF жҳҜдёҖз§ҚиҝҗиЎҢеңЁеҶ…ж ёдёӯзҡ„гҖҒеҸ—йҷҗзҡ„гҖҒеҸҜйӘҢиҜҒзҡ„гҖҒдәӢд»¶й©ұеҠЁзҡ„и®Ўз®—жЁЎеһӢгҖӮ

е®ғдёҚжҳҜпјҡ

  • еҶ…ж ёи„ҡжң¬иҜӯиЁҖ
  • йҖҡз”Ёжү§иЎҢзҺҜеўғ
  • еҶ…ж ёжЁЎеқ—жӣҝд»Је“Ғ

иҖҢжҳҜпјҡ

  • **еҶ…ж ёдёӯзҡ„зӯ–з•ҘдёҺи§ӮжөӢи®Ўз®—еұӮ**

2.2 eBPF и§ЈеҶізҡ„ж ёеҝғй—®йўҳ

й—®йўҳ eBPF зҡ„еӣһзӯ”
еҰӮдҪ•е®үе…Ёжү©еұ•еҶ…ж ёиЎҢдёә Verifier + еҸ—йҷҗжҢҮд»ӨйӣҶ
еҰӮдҪ•йҒҝе…ҚеҶ…ж ёеҙ©жәғ йқҷжҖҒиҜҒжҳҺз»ҲжӯўжҖ§дёҺеҶ…еӯҳе®үе…Ё
еҰӮдҪ•дҝқжҢҒй«ҳжҖ§иғҪ JIT + дәӢд»¶е°ұең°жү§иЎҢ
еҰӮдҪ•и·ЁжҖҒйҖҡдҝЎ жҳҫејҸ Map зҠ¶жҖҒжЁЎеһӢ

дёүгҖҒeBPF зҡ„ж•ҙдҪ“жһ¶жһ„жЁЎеһӢпјҲжҠҪиұЎи§Ҷи§’пјү

3.1 жҺ§еҲ¶йқў / жү§иЎҢйқўеҲҶзҰ»

  • з”ЁжҲ·жҖҒпјҲжҺ§еҲ¶йқўпјү

    • зЁӢеәҸз”ҹжҲҗгҖҒеҠ иҪҪгҖҒжҢӮиҪҪ
    • Map з”ҹе‘Ҫе‘Ёжңҹз®ЎзҗҶ

  • еҶ…ж ёжҖҒпјҲжү§иЎҢйқўпјү

    • eBPF зЁӢеәҸжү§иЎҢ
    • дәӢд»¶и§ҰеҸ‘дёҺж•°жҚ®йҮҮйӣҶ

иҝҷдёҖз»“жһ„зЎ®дҝқпјҡ

  • еҶ…ж ёдёҚдҝЎд»»з”ЁжҲ·йҖ»иҫ‘
  • дҪҶе…Ғи®ёе…¶еңЁдёҘж јиҫ№з•ҢеҶ…иҝҗиЎҢ

3.2 ж ёеҝғжҰӮеҝөиғҪеҠӣж ‘

дәӢд»¶жәҗ (Hook)
   вҶ“
ProgramпјҲеҸ—йҷҗйҖ»иҫ‘пјү
   вҶ“
HelperпјҲиғҪеҠӣзҷҪеҗҚеҚ•пјү
   вҶ“
MapпјҲзҠ¶жҖҒдёҺйҖҡдҝЎпјү
   вҶ“
з”ЁжҲ·жҖҒзі»з»ҹ

еӣӣгҖҒж ёеҝғжҰӮеҝөзҡ„еҺҹзҗҶзә§и§ЈйҮҠ

4.1 ProgramпјҡеҸ—йҷҗи®Ўз®—еҚ•е…ғ

  • жң¬иҙЁпјҡдәӢд»¶еӨ„зҗҶеҮҪж•°

  • зү№еҫҒпјҡ

    • ж— еҫӘзҺҜжҲ–жңүз•ҢеҫӘзҺҜ
    • ж— еҠЁжҖҒеҶ…еӯҳеҲҶй…Қ
    • ж— д»»ж„ҸеҮҪж•°и°ғз”Ё

Program дёҚжҳҜдёәдәҶвҖңиЎЁиҫҫиғҪеҠӣвҖқпјҢиҖҢжҳҜдёәдәҶвҖңеҸҜиҜҒжҳҺжҖ§вҖқгҖӮ

4.2 HookпјҡдәӢд»¶й©ұеҠЁжЁЎеһӢ

  • eBPF ж°ёдёҚдё»еҠЁиҝҗиЎҢ

  • еҸӘиғҪе“Қеә”пјҡ

    • зі»з»ҹи°ғз”Ё
    • еҶ…ж ёеҮҪж•°
    • Tracepoint
    • зҪ‘з»ңи·Ҝеҫ„

иҝҷжҳҜеҜ№еҶ…ж ёзЁіе®ҡжҖ§зҡ„ж №жң¬дҝқжҠӨгҖӮ

4.3 VerifierпјҡeBPF зҡ„ж ёеҝғ

Verifier жҳҜ eBPF иғҪиҝӣе…Ҙдё»зәҝеҶ…ж ёзҡ„ж №жң¬еҺҹеӣ гҖӮ

е®ғеңЁеҠ иҪҪйҳ¶ж®өйқҷжҖҒиҜҒжҳҺпјҡ

  1. **еҶ…еӯҳе®үе…Ё**пјҲж— и¶Ҡз•Ңи®ҝй—®пјү
  2. **жү§иЎҢеҸҜз»Ҳжӯў**пјҲж— ж— йҷҗеҫӘзҺҜпјү
  3. **еӨҚжқӮеәҰжңүз•Ң**пјҲжҢҮд»Өж•°гҖҒж Ҳж·ұеәҰпјү

eBPF зҡ„зҒөжҙ»жҖ§пјҢе»әз«ӢеңЁвҖңжҸҗеүҚжӢ’з»қеҚұйҷ©зЁӢеәҸвҖқд№ӢдёҠгҖӮ

4.4 HelperпјҡиғҪеҠӣзҷҪеҗҚеҚ•жңәеҲ¶

  • eBPF дёҚиғҪзӣҙжҺҘи°ғз”ЁеҶ…ж ёеҮҪж•°

  • еҸӘиғҪйҖҡиҝҮпјҡ

    • жҳҫејҸжҡҙйңІ
    • иҜӯд№үеҸ—жҺ§
    • еҸҜйӘҢиҜҒзҡ„ Helper

иҝҷжҳҜе…ёеһӢзҡ„ Capability-based SecurityгҖӮ

4.5 MapпјҡжҳҫејҸзҠ¶жҖҒжЁЎеһӢ

  • Map жҳҜпјҡ

    • зҠ¶жҖҒеӯҳеӮЁ
    • и·ЁжҖҒйҖҡдҝЎд»ӢиҙЁ
    • е№¶еҸ‘жҺ§еҲ¶иҫ№з•Ң

eBPF зҡ„дёҖдёӘйҮҚиҰҒи®ҫи®ЎйҖүжӢ©пјҡ йҖ»иҫ‘ж— зҠ¶жҖҒпјҢзҠ¶жҖҒеӨ–зҪ®еҢ–гҖӮ

дә”гҖҒиҝҗиЎҢж—¶жЁЎеһӢпјҲд»ҺжңәеҲ¶еҲ°еҺҹзҗҶпјү

5.1 з”ҹе‘Ҫе‘ЁжңҹжҠҪиұЎ

з”ҹжҲҗ вҶ’ ж ЎйӘҢ вҶ’ жҢӮиҪҪ вҶ’ дәӢд»¶и§ҰеҸ‘ вҶ’ жү§иЎҢ вҶ’ зҠ¶жҖҒиҫ“еҮә
  • д»»дҪ•дёҖжӯҘеӨұиҙҘпјҢзЁӢеәҸйғҪдёҚдјҡиҝӣе…ҘеҶ…ж ёжү§иЎҢи·Ҝеҫ„

5.2 жҖ§иғҪе“ІеӯҰ

  • дәӢд»¶е°ұең°жү§иЎҢпјҲж— дёҠдёӢж–ҮеҲҮжҚўпјү
  • JIT иҪ¬жҚўдёәжң¬ең°жҢҮд»Ө
  • Map йҒҝе…Қйў‘з№ҒеҶ…ж ё-з”ЁжҲ·жӢ·иҙқ

жҖ§иғҪдёҚжҳҜдјҳеҢ–еҮәжқҘзҡ„пјҢиҖҢжҳҜжһ¶жһ„йҖүжӢ©зҡ„з»“жһңгҖӮ

е…ӯгҖҒeBPF зҡ„йҖӮз”Ёиҫ№з•ҢдёҺеҸҚжЁЎејҸ

6.1 йҖӮеҗҲ eBPF зҡ„еңәжҷҜ

  • еҸҜи§ӮжөӢжҖ§пјҲTracing / Metrics / Profilingпјү
  • зҪ‘з»ңдёҺе®үе…Ёзӯ–з•Ҙ
  • еҠЁжҖҒеҶ…ж ёиЎҢдёәи°ғж•ҙ

6.2 дёҚйҖӮеҗҲ eBPF зҡ„еңәжҷҜ

  • еӨҚжқӮдёҡеҠЎйҖ»иҫ‘
  • й•ҝж—¶й—ҙиҝҗиЎҢд»»еҠЎ
  • й«ҳеәҰзҠ¶жҖҒеҢ–зЁӢеәҸ

eBPF жҳҜвҖңеҶ…ж ёдёӯзҡ„зӯ–з•ҘеұӮвҖқпјҢдёҚжҳҜеә”з”Ёе№іеҸ°гҖӮ

дёғгҖҒжһ¶жһ„еұӮжҖ»з»“

eBPF зҡ„й•ҝжңҹд»·еҖјдёҚеңЁдәҺжҹҗдёӘ APIпјҢиҖҢеңЁдәҺе®ғиҜҒжҳҺдәҶдёҖд»¶дәӢпјҡ

еҶ…ж ёеҸҜд»ҘеңЁдёҚзүәзүІзЁіе®ҡжҖ§зҡ„еүҚжҸҗдёӢпјҢиў«е®үе…Ёең°еҠЁжҖҒжү©еұ•гҖӮ
