### 使用文件系统调用
一个简单的C语言程序：
```c
int main(){

        FILE *fp;
        char c = -1;
        fp = fopen("./file.c","r");
        while ((c = fgetc(fp)) != -1){
                putchar(c);
        }
        fclose(fp);

        return 0;
}
```
