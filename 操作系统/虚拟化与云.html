иҷҡжӢҹеҢ–дёҺдә‘

иҷҡжӢҹеҢ–зҡ„еҝ…иҰҒжқЎд»¶пјҡ

  • е®үе…ЁпјҡиҷҡжӢҹжңәзЁӢеәҸеҸҜд»Ҙе®Ңе…ЁжҺҢжҺ§иҷҡжӢҹиө„жәҗ
  • дҝқзңҹпјҡиҷҡжӢҹжңәйҮҢзҡ„зЁӢеәҸиЎҢдёәиҰҒи·ҹеңЁиЈёжңәдёҠжү§иЎҢзӣёеҗҢ
  • й«ҳж•ҲпјҡиҷҡжӢҹжңәзҡ„д»Јз ҒдёҚеҸ—иҷҡжӢҹжңәзЁӢеәҸе№Іж¶ү

дёәдәҶдҝқиҜҒе®ҝдё»жңәзҡ„е®үе…ЁжҖ§пјҢиҷҡжӢҹжңәзЁӢеәҸйҮҮз”Ёи§ЈйҮҠжү§иЎҢиҷҡжӢҹжңәеҸ‘иө·зҡ„е‘Ҫд»Ө

еҸҰдёҖж–№йқўпјҢеҜ№дәҺжҹҗдәӣжҢҮд»ӨпјҢеҲҷйңҖиҰҒCPUзҡ„ж”ҜжҢҒпјҢжүҚиғҪж”ҜжҢҒиҷҡжӢҹеҢ–пјҢеҰӮIntelзҡ„VTпјҲиҷҡжӢҹеҢ–жҠҖжңҜпјү

дёӨиҖ…зҡ„еҢәеҲ«еңЁдәҺз¬¬дәҢзұ»иҷҡжӢҹжңәзЁӢеәҸиҝҗиЎҢеңЁй”ҒдҪҸж“ҚдҪңзі»з»ҹд№ӢдёҠ

з¬¬дёҖзұ»зҡ„д»ЈиЎЁжңүESX Server з¬¬дәҢзұ»зҡ„д»ЈиЎЁеҲҷжңү VMware

жңҚеҠЎеҷЁиҷҡжӢҹеҢ–жҠҖжңҜпјҡ

  • Hypervisorд№ҹеҸ«VMMпјҲvirtual machine monitorпјү
    • еңЁиҷҡжӢҹзЁӢеәҸдёҠиҝҗиЎҢж“ҚдҪңзі»з»ҹ
  • Containersе®№еҷЁеҢ–жҠҖжңҜ
    • иҷҡжӢҹзЁӢеәҸдёҠзӣҙжҺҘиҝҗиЎҢзЁӢеәҸпјҢж— ж“ҚдҪңзі»з»ҹ

й«ҳж•ҲиҷҡжӢҹеҢ–жҠҖжңҜ

зӣ®еүҚе·ІжңүдёҖдәӣз ”з©¶жҸҗеҮәйҖҡиҝҮе®ҡд№үдёҖдёӘз»ҹдёҖзҡ„иҷҡжӢҹеҢ–и°ғз”ЁжҺҘеҸЈпјҢз„¶еҗҺж“ҚдҪңзі»з»ҹзӣҙжҺҘдҪҝз”ЁиҝҷдәӣиҷҡжӢҹеҢ–жҺҘеҸЈжқҘиҝӣиЎҢеҚҠиҷҡжӢҹеҢ–жҲ–иҖ…е…ЁиҷҡжӢҹеҢ–пјҢиҖҢиҷҡжӢҹеҢ–и°ғз”ЁжҺҘеҸЈеҲҷе’ҢзңҹжӯЈзҡ„зЎ¬д»¶жҲ–иҖ…иҷҡжӢҹжңәзЁӢеәҸжү“дәӨйҒ“

еңЁдёҚж”ҜжҢҒиҷҡжӢҹеҢ–зҡ„е№іеҸ°дёҠе®һзҺ°иҷҡжӢҹеҢ–

е®ўжҲ·еҸҠзҡ„ж•Ҹж„ҹжҢҮд»Өи°ғз”Ёдјҡиў«жЁЎжӢҹдјҡеҜ№иҷҡжӢҹжңәзЁӢеәҸзҡ„жҢҮд»Өи°ғз”ЁпјҢзЎ¬д»¶дёҚдјҡзңҹжӯЈжү§иЎҢиҝҷдәӣжҢҮд»Ө

иҷҡжӢҹеҢ–зҡ„ејҖй”Җ

VTзЎ¬д»¶иҷҡжӢҹеҢ–дјҡдә§з”ҹеӨ§йҮҸйҷ·е…ҘпјҢиҖҢйҷ·е…ҘдјҡеҜ№зј“еӯҳгҖҒTLBзӯүйҖ жҲҗдёҚеҲ©еҪұе“Қ

иҪҜд»¶еҪўејҸзҡ„иҷҡжӢҹеҢ–еңЁиҜёеҰӮйЎөиЎЁз®ЎзҗҶдёҠеҲҷйңҖиҰҒдёҚе°Ҹзҡ„ејҖй”Җ

VMware еҺҹзҗҶ

  • иҷҡжӢҹеҢ–x86жһ¶жһ„пјҡйҖҡиҝҮеҶізӯ–з®—жі•жЁЎжӢҹCPUжҢҮд»ӨжҲ–иҖ…зӣҙжҺҘжү§иЎҢ
  • е®ўжҲ·ж“ҚдҪңзі»з»ҹдёәдёӯеҝғпјҡй’ҲеҜ№зү№е®ҡж“ҚдҪңзі»з»ҹеҒҡиҷҡжӢҹеҢ–ж”ҜжҢҒ
  • иҷҡжӢҹзЎ¬д»¶е№іеҸ°пјҡйҖҡиҝҮе°ҶжЁЎжӢҹеүҚз«ҜзЎ¬д»¶дёҺеҗҺз«Ҝе®һзҺ°зЎ¬д»¶еҲҶзҰ»пјҢиҝӣиЎҢзЎ¬д»¶ж— е…іе°ҒиЈ…д»Ҙзү©зҗҶзЎ¬д»¶зҡ„иҷҡжӢҹеҢ–
  • е®ҝдё»ж“ҚдҪңзі»з»ҹдҪңдёәй©ұеҠЁзЁӢеәҸзҡ„еҜ„з”ҹиҖ…пјҢиҝӣиЎҢVMXдёҺVMMзҡ„дёӯй—ҙйҖҡдҝЎеҷЁ

йҖҡиҝҮзі»з»ҹеҲҮжҚўпјҢе°Ҷж•ҙдёӘең°еқҖз©әй—ҙеңЁе®ҝдё»ж“ҚдҪңзі»з»ҹеҸҠVMMд№Ӣй—ҙеҲҮжҚўпјҢиҝҷж ·VMMжүҚиғҪжңүеҜ№иө„жәҗзҡ„е®Ңе…ЁжҺҢжҺ§жқғйҷҗ

иҷҡжӢҹеҢ–е®№еҷЁ

chrootгҖҒnamespacesгҖҒcgroups -> LXC -> Docker -> K8S

дёүж–№йқўзҡ„е…је®№жҖ§пјҡ

  1. ISAе…је®№пјҡзӣ®ж ҮжңәеҷЁжҢҮд»ӨйӣҶе…је®№жҖ§
  2. ABIе…је®№пјҡзӣ®ж Үзі»з»ҹжҲ–иҖ…дҫқиө–еә“зҡ„дәҢиҝӣеҲ¶е…је®№жҖ§
  3. зҺҜеўғе…је®№пјҡй…ҚзҪ®ж–Үд»¶ зҺҜеўғеҸҳйҮҸзӯү

еҮ з§ҚиҷҡжӢҹеҢ–жҠҖжңҜпјҡ

  • жҢҮд»ӨйӣҶиҷҡжӢҹеҢ–пјҡйҖҡиҝҮиҪҜд»¶жқҘжЁЎжӢҹдёҚеҗҢISAжһ¶жһ„зҡ„еӨ„зҗҶеҷЁе·ҘдҪңиҝҮзЁӢпјҢе°ҶиҷҡжӢҹжңәеҸ‘еҮәзҡ„жҢҮд»ӨиҪ¬жҚўдёәз¬ҰеҗҲжң¬жңәISAзҡ„жҢҮд»Ө
  • зЎ¬д»¶жҠҪиұЎеұӮиҷҡжӢҹеҢ–пјҡдҪҝз”ЁиҪҜд»¶жқҘжЁЎжӢҹзЎ¬д»¶пјҢжҲ–иҖ…зӣҙжҺҘе°ҶзЎ¬д»¶зӣҙйҖҡеҲ°иҷҡжӢҹжңәдҪҝз”Ё
  • ж“ҚдҪңзі»з»ҹеұӮиҷҡжӢҹеҢ–пјҡйҮҮз”Ёйҡ”зҰ»жүӢж®өпјҢдҪҝеҫ—дёҚеҗҢиҝӣзЁӢжӢҘжңүзӢ¬з«Ӣзҡ„зі»з»ҹиө„жәҗе’Ңиө„жәҗй…Қйўқ д№ҹе°ұжҳҜе®№еҷЁеҢ–
  • иҝҗиЎҢеә“иҷҡжӢҹеҢ–пјҡдҪҝз”ЁиҪҜд»¶зҝ»иҜ‘зҡ„ж–№жі•жқҘжЁЎжӢҹзі»з»ҹпјҢе®ғд»ҘдёҖдёӘзӢ¬з«ӢиҝӣзЁӢжқҘд»Јжӣҝж“ҚдҪңзі»з»ҹеҶ…ж ёжқҘжҸҗдҫӣзӣ®ж ҮиҪҜд»¶иҝҗиЎҢжүҖйңҖзҡ„е…ЁйғЁиғҪеҠӣ еҰӮWINE
  • иҜӯиЁҖеұӮиҷҡжӢҹеҢ–пјҡз”ұиҷҡжӢҹжңәе°Ҷй«ҳзә§иҜӯиЁҖз”ҹжҲҗзҡ„дёӯй—ҙд»Јз ҒиҪ¬жҚўдёәзӣ®ж ҮжңәеҷЁеҸҜд»ҘзӣҙжҺҘжү§иЎҢзҡ„жҢҮд»Ө

е®№еҷЁйҡ”зҰ»

  • йҡ”зҰ»ж–Үд»¶пјҡchroot
    • еҪ“жҹҗдёӘиҝӣзЁӢз»ҸиҝҮchrootж“ҚдҪңд№ӢеҗҺпјҢе®ғзҡ„ж №зӣ®еҪ•е°ұдјҡиў«й”Ғе®ҡеңЁе‘Ҫд»ӨеҸӮж•°жүҖжҢҮе®ҡзҡ„дҪҚзҪ®пјҢд»ҘеҗҺе®ғжҲ–иҖ…е®ғзҡ„еӯҗиҝӣзЁӢе°ҶдёҚиғҪеҶҚи®ҝй—®е’Ңж“ҚдҪңиҜҘзӣ®еҪ•д№ӢеӨ–зҡ„е…¶д»–ж–Үд»¶
  • йҡ”зҰ»и®ҝй—®пјҡnamespaces
    • з”ұеҶ…ж ёзӣҙжҺҘжҸҗдҫӣзҡ„е…ЁеұҖиө„жәҗе°ҒиЈ…еҰӮж–Үд»¶ иҝӣзЁӢ з”ЁжҲ· зҪ‘з»ңзӯүдҝЎжҒҜпјҢжҳҜеҶ…ж ёй’ҲеҜ№иҝӣзЁӢи®ҫи®Ўзҡ„и®ҝй—®йҡ”зҰ»жңәеҲ¶
  • йҡ”зҰ»иө„жәҗпјҡcgroups
    • жҺ§еҲ¶иҝӣзЁӢеҜ№иө„жәҗзҡ„дҪҝз”Ё

е®№еҷЁе°ҒиЈ…

  • е°ҒиЈ…зі»з»ҹпјҡLinuxX Containers
  • е°ҒиЈ…еә”з”ЁпјҡDocker
  • е°ҒиЈ…йӣҶзҫӨпјҡKubernetes

еҹәдәҺе®№еҷЁжһ„е»әзі»з»ҹ

  • йҡ”зҰ»пјҡеҗҚз§°з©әй—ҙ
  • еҚҸдҪңпјҡеҹәдәҺеҗҢдёҖдёӘpod

иҝҷйҮҢеңЁK8Sдёӯ зӣёдә’еҚҸдҪңзҡ„е®№еҷЁиҝҗиЎҢеңЁеҗҢдёҖдёӘpodдёӯ иҝҷдёӘpodе°ұжҳҜиў«и°ғеәҰзҡ„еҹәжң¬еҚ•дҪҚ

K8Sзҡ„еҚҸдҪң еҰӮжһңиҝҗиЎҢеҚ•дёӘpod йӮЈд№Ҳзі»з»ҹжҳҜеҚҒеҲҶи„Ҷејұзҡ„ дҪҝз”ЁReplicaSetзӯүжүӢж®өжқҘдҝқжҢҒдҪҸзі»з»ҹзҡ„зҠ¶жҖҒ

иҝҷз§Қз»„д»¶иў«з§°дёәжҺ§еҲ¶еӣһи·Ҝпјҡ

еә”з”Ёдёәдёӯеҝғзҡ„е°ҒиЈ…

  • KustomizeпјҡдҪҝз”ЁдёҖдёӘй…ҚзҪ®ж–Үд»¶жҸҸиҝ°й…ҚзҪ®ж–Үд»¶
  • HelmдёҺChartпјҡжЁЎд»ҝLinuxеҢ…з®ЎзҗҶеҷЁзҡ„жҖқи·ҜеҺ»з®ЎзҗҶKubernetesеә”з”Ё
  • OperatorдёҺCRDпјҡжҠҠеә”з”Ёе°ҒиЈ…дёәеҸҰдёҖз§Қжӣҙй«ҳеұӮж¬Ўзҡ„иө„жәҗпјҢеҶҚжҠҠKubernetesзҡ„жҺ§еҲ¶еҷЁжЁЎејҸд»Һйқўеҗ‘дәҺеҶ…зҪ®иө„жәҗпјҢжү©еұ•еҲ°дәҶйқўеҗ‘жүҖжңүиҮӘе®ҡд№үиө„жәҗ
  • ејҖж”ҫеә”з”ЁжЁЎеһӢ

еҶ…еӯҳиҷҡжӢҹеҢ–

иҷҡжӢҹжңәдҪңдёәдёҖдёӘдёӯй—ҙеұӮпјҢйңҖиҰҒз»ҙжҠӨе®ўжҲ·ж“ҚдҪңзі»з»ҹдёҺзңҹе®һзЎ¬д»¶еҶ…еӯҳйЎөиЎЁзҡ„жҳ е°„

  • еөҢеҘ—йЎөиЎЁзҡ„зЎ¬д»¶жү©еұ•пјҡз”ұеҺҹжқҘзҡ„иҷҡжӢҹйЎөиЎЁ-зү©зҗҶйЎөиЎЁеҸҳжҲҗе®ўжҲ·жңәиҷҡжӢҹйЎөиЎЁ-е®ўжҲ·жңәзү©зҗҶйЎөиЎЁ-е®ҝдё»жңәзү©зҗҶйЎөиЎЁ
  • еӣһж”¶еҶ…еӯҳпјҡиҝҷе°ұйңҖиҰҒиҷҡжӢҹжңәиҮӘе·ұдҪңдёәдёӯй—ҙеұӮпјҢжқҘз®ЎзҗҶеҶ…еӯҳ

IOиҷҡжӢҹеҢ–

  • IO MMUпјҡзҺ°д»ЈзЎ¬д»¶е…·еӨҮ IO MMU, еҸҜд»Ҙе®һзҺ°и®ҫеӨҮз©ҝйҖҸпјҲзӣҙжҺҘд»Һзү©зҗҶзЎ¬д»¶жҳ е°„еҲ°иҷҡжӢҹжңәпјүпјҢд№ҹеҸҜд»ҘдҝқиҜҒи®ҫеӨҮйҡ”зҰ»
  • и®ҫеӨҮеҹҹпјҡз”Ёдё“й—Ёзҡ„дёҖеҸ°иҷҡжӢҹжңәз®ЎзҗҶзЎ¬д»¶пјҢе…¶д»–иҷҡжӢҹжңәйҖҡиҝҮиҝҷеҸ°иҷҡжӢҹжңәжқҘдҪҝз”ЁзЎ¬д»¶пјҲеҚҠиҷҡжӢҹеҢ–пјү
  • еҚ•ж №IOиҷҡжӢҹеҢ–пјҡз”ұIOи®ҫеӨҮжң¬иә«жҸҗдҫӣиҷҡжӢҹеҢ–ж”ҜжҢҒпјҢйҖҡиҝҮжҸҗдҫӣVFпјҲиҷҡжӢҹеҢ–еҮҪж•°пјүз»ҷиҷҡжӢҹжңәдҪҝз”Ё

иҷҡжӢҹиЈ…зҪ®

  • е°ҒиЈ…иҷҡжӢҹжңәпјҢдәӨд»ҳиҝҗиЎҢпјҢи·ҹзҺ°йҳ¶ж®өзҡ„е®№еҷЁеҫҲеғҸ
