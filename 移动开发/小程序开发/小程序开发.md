## 视图层

### WXML

import 的依赖无法传递

- 嵌入

```html
<include src="header.wxml" />
```

### WXSS

- 单位

rpx（responsive pixel）: 可以根据屏幕宽度进行自适应

- 样式导入

```css
@import "common.wxss";
```

- 内联样式
  - style与class
  
- 选择器
  - .class
  - #id
  - element
  - element,element
  - ::after
  - ::before

- 全局样式与局部样式
  - app.wxss 是全局样式，定义在page目录下的是局部样式
