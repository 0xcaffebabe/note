  • ж•°з»„пјҡ

    • дјҳзӮ№пјҡйҖҡиҝҮдёӢж ҮзӣҙжҺҘе®ҡдҪҚпјҢж—¶й—ҙеӨҚжқӮеәҰ O(1)
    • зјәзӮ№пјҡдёӢж Үеҝ…йЎ»жҳҜиҝһз»ӯж•ҙж•°пјҢзҺ°е®һдё–з•Ңзҡ„ key е№¶дёҚж»Ўи¶і

  • й“ҫиЎЁ / ж ‘пјҡ

    • дјҳзӮ№пјҡkey дёҚиҰҒжұӮиҝһз»ӯ
    • зјәзӮ№пјҡжҹҘжүҫйңҖиҰҒжҜ”иҫғпјҢж—¶й—ҙеӨҚжқӮеәҰиҮіе°‘ O(log n)

ж №жң¬зҹӣзӣҫпјҡ жҲ‘д»¬еёҢжңӣеғҸж•°з»„дёҖж ·еҝ«пјҢдҪҶ key еҚҙеғҸеҜ№иұЎгҖҒеӯ—з¬ҰдёІгҖҒID дёҖж ·дёҚеҸҜжҺ§гҖӮ

2. ж•ЈеҲ—иЎЁзҡ„ж ёеҝғжҖқжғі

ж•ЈеҲ—иЎЁзҡ„жң¬иҙЁдёҚжҳҜ"жҹҘжүҫеҝ«"пјҢиҖҢжҳҜпјҡ

йҖҡиҝҮдёҖдёӘзЎ®е®ҡжҖ§еҮҪж•°пјҢжҠҠ"дёҚеҸҜжҺ§зҡ„ key з©әй—ҙ"пјҢжҳ е°„еҲ°"еҸҜжҺ§зҡ„ж•°з»„дёӢж Үз©әй—ҙ"гҖӮ

з”ұжӯӨеҒҡеҮәдёҖдёӘе…ій”®еҸ–иҲҚпјҡ

  • жҺҘеҸ—пјҡ

    • з©әй—ҙжөӘиҙ№
    • еҶІзӘҒзҡ„еӯҳеңЁ
    • жҰӮзҺҮж„Ҹд№үдёҠзҡ„жҖ§иғҪдҝқиҜҒ

  • жҚўеҸ–пјҡ

    • еқҮж‘Ҡ O(1) зҡ„и®ҝй—®ж—¶й—ҙ

дёҖеҸҘиҜқе®ҡд№үпјҡ

ж•ЈеҲ—иЎЁжҳҜдёҖз§Қд»ҘжҰӮзҺҮеқҮж‘ҠдёәеүҚжҸҗпјҢз”Ёз©әй—ҙжҚўж—¶й—ҙгҖҒж”ҫејғйЎәеәҸжҖ§зҡ„жҳ е°„еһӢж•°жҚ®з»“жһ„гҖӮ

дәҢгҖҒж•ЈеҲ—еҮҪж•°пјҡжҠҠзҺ°е®һдё–з•ҢеҺӢзј©иҝӣж•°з»„

1. ж•ЈеҲ—еҮҪж•°зҡ„и§’иүІе®ҡдҪҚ

ж•ЈеҲ—еҮҪж•°дёҚжҳҜдёәдәҶ"еҠ еҜҶ"пјҢиҖҢжҳҜдёәдәҶпјҡ

  • е°Ҷд»»ж„Ҹзұ»еһӢгҖҒд»»ж„ҸеҲҶеёғзҡ„ key
  • жҳ е°„дёәдёҖдёӘ **жңүйҷҗж•ҙж•°еҢәй—ҙ**
  • дҪңдёәж•°з»„зҡ„дёӢж Ү

еҪўејҸеҢ–жҸҸиҝ°пјҡ

hash : KeySpace вҶ’ [0, N)

2. ж•ЈеҲ—еҮҪж•°и®ҫи®Ўзҡ„жң¬иҙЁзәҰжқҹ

дёҖдёӘ"е·ҘзЁӢеҸҜз”Ё"зҡ„ж•ЈеҲ—еҮҪж•°пјҢжң¬иҙЁдёҠеңЁе№іиЎЎдёүд»¶дәӢпјҡ

  1. **зЎ®е®ҡжҖ§**пјҡеҗҢдёҖдёӘ key еҝ…йЎ»еҫ—еҲ°еҗҢдёҖдёӘз»“жһң
  2. **еқҮеҢҖжҖ§**пјҡе°ҪеҸҜиғҪжү“ж•Ј key зҡ„еҺҹе§ӢеҲҶеёғ
  3. **жҲҗжң¬еҸҜжҺ§**пјҡдёҚиғҪдёәдәҶеҮҸе°‘еҶІзӘҒиҖҢд»ҳеҮәиҝҮй«ҳи®Ўз®—д»Јд»·

жіЁж„Ҹпјҡ

  • "з»қдёҚеҶІзӘҒ"зҡ„ж•ЈеҲ—еҮҪж•°еңЁжңүйҷҗж•°з»„дёӯ**зҗҶи®әдёҠдёҚеҸҜиғҪеӯҳеңЁ**
  • еҶІзӘҒдёҚжҳҜзјәйҷ·пјҢиҖҢжҳҜеүҚжҸҗжқЎд»¶

дёүгҖҒеҶІзӘҒжҳҜеҝ…з„¶зҡ„пјҡйёҪе·ўеҺҹзҗҶи§Ҷи§’

1. дёәд»Җд№ҲеҶІзӘҒж— жі•йҒҝе…Қ

еҪ“пјҡ

  • key зҡ„ж•°йҮҸ > ж§ҪдҪҚж•°йҮҸ

ж №жҚ®йёҪе·ўеҺҹзҗҶпјҡ

иҮіе°‘еӯҳеңЁдёӨдёӘ key иў«жҳ е°„еҲ°еҗҢдёҖдёӘж§ҪдҪҚгҖӮ

еӣ жӯӨпјҡ

  • еҶІзӘҒдёҚжҳҜе®һзҺ°й—®йўҳ
  • иҖҢжҳҜж•°еӯҰеҝ…з„¶

2. иЈ…иҪҪеӣ еӯҗпјҡеҶІзӘҒжҰӮзҺҮзҡ„еҸҜжҺ§жҢҮж Ү

е®ҡд№үпјҡ

иЈ…иҪҪеӣ еӯҗ Оұ = е·Із”Ёж§ҪдҪҚж•° / жҖ»ж§ҪдҪҚж•°

иЈ…иҪҪеӣ еӯҗжң¬иҙЁжҸҸиҝ°зҡ„жҳҜпјҡ

  • з©әй—ҙеҲ©з”ЁзҺҮ
  • еҶІзӘҒеҸ‘з”ҹзҡ„жҰӮзҺҮеҜҶеәҰ

е·ҘзЁӢдёҠзҡ„ж ёеҝғи®ӨзҹҘпјҡ

  • Оұ иҝҮе°Ҹ вҶ’ з©әй—ҙжөӘиҙ№
  • Оұ иҝҮеӨ§ вҶ’ еҶІзӘҒжҝҖеўһпјҢжҖ§иғҪйҖҖеҢ–

иЈ…иҪҪеӣ еӯҗжҳҜж•ЈеҲ—иЎЁжҖ§иғҪдёҺз©әй—ҙд№Ӣй—ҙзҡ„и°ғиҠӮж—Ӣй’®гҖӮ

еӣӣгҖҒеҶІзӘҒи§ЈеҶізӯ–з•ҘпјҡдёҚеҗҢеҸ–иҲҚдёӢзҡ„еҝ…з„¶и®ҫи®Ў

еҶІзӘҒи§ЈеҶідёҚжҳҜ"е“Әз§Қжӣҙй«ҳзә§"пјҢиҖҢжҳҜпјҡ

еңЁдёҚеҗҢзәҰжқҹжқЎд»¶дёӢпјҢеҜ№еҗҢдёҖй—®йўҳзҡ„дёҚеҗҢеҸ–иҲҚгҖӮ

1. жӢүй“ҫжі•пјҲSeparate Chainingпјү

ж ёеҝғжҖқжғіпјҡ

  • жҜҸдёӘж§ҪдҪҚдёҚеҶҚеҸӘеӯҳдёҖдёӘе…ғзҙ 
  • иҖҢжҳҜеӯҳдёҖдёӘ"йӣҶеҗҲ"пјҲйҖҡеёёжҳҜй“ҫиЎЁжҲ–ж ‘пјү

жң¬иҙЁдјҳеҠҝпјҡ

  • еҜ№иЈ…иҪҪеӣ еӯҗдёҚж•Ҹж„ҹ
  • еҲ йҷӨж“ҚдҪңеӨ©з„¶з®ҖеҚ•
  • е·ҘзЁӢжү©еұ•жҖ§ејә

д»Јд»·пјҡ

  • йўқеӨ–жҢҮй’ҲејҖй”Җ
  • еҶ…еӯҳеұҖйғЁжҖ§иҫғе·®

еҪ“зі»з»ҹжӣҙе…іжіЁзЁіе®ҡжҖ§е’ҢеҸҜжү©еұ•жҖ§ж—¶пјҢжӢүй“ҫжі•еҫҖеҫҖжҳҜй»ҳи®ӨйҖүжӢ©гҖӮ

2. ејҖж”ҫе®ҡеқҖжі•пјҲOpen Addressingпјү

ж ёеҝғжҖқжғіпјҡ

  • жүҖжңүе…ғзҙ еҝ…йЎ»еӯҳж”ҫеңЁеҗҢдёҖдёӘж•°з»„дёӯ
  • еҶІзӘҒж—¶пјҢжҢүз…§жҹҗз§Қи§„еҲҷеҜ»жүҫдёӢдёҖдёӘз©әдҪҚ

иҝҷзұ»ж–№жі•жң¬иҙЁдҫқиө–пјҡ

  • ж§ҪдҪҚиҝһз»ӯжҖ§
  • иЈ…иҪҪеӣ еӯҗзҡ„дёҘж јжҺ§еҲ¶

2.1 зәҝжҖ§жҺўжөӢ

  • жҺўжөӢеәҸеҲ—пјҡ

    hash(key), hash(key)+1, hash(key)+2, ...

  • й—®йўҳжң¬иҙЁпјҡ

    • жҳ“еҪўжҲҗ"иҒҡйӣҶж•Ҳеә”"

2.2 дәҢж¬ЎжҺўжөӢ

  • жҺўжөӢеәҸеҲ—пјҡ

    hash(key) + i^2

  • зӣ®ж Үпјҡ

    • еҮҸиҪ»зәҝжҖ§иҒҡйӣҶ

2.3 еҸҢйҮҚж•ЈеҲ—

  • дҪҝз”ЁеӨҡдёӘж•ЈеҲ—еҮҪж•°з”ҹжҲҗжҺўжөӢжӯҘй•ҝ

  • жң¬иҙЁжҳҜпјҡ

    • з”Ёи®Ўз®—еӨҚжқӮеәҰжҚўеҸ–жӣҙеҘҪзҡ„еҲҶеёғ

3. еҶІзӘҒзӯ–з•Ҙзҡ„еҸ–иҲҚеҜ№жҜ”

з»ҙеәҰ жӢүй“ҫжі• ејҖж”ҫе®ҡеқҖ
иЈ…иҪҪеӣ еӯҗе®№еҝҚеәҰ й«ҳ дҪҺ
еҲ йҷӨеӨҚжқӮеәҰ дҪҺ й«ҳ
еҶ…еӯҳеұҖйғЁжҖ§ иҫғе·® иҫғеҘҪ
е®һзҺ°еӨҚжқӮеәҰ дёӯ й«ҳ

дә”гҖҒеҲ йҷӨпјҡдёәд»Җд№ҲиҝҷжҳҜдёҖдёӘ"еҚұйҷ©ж“ҚдҪң"

1. жӢүй“ҫжі•зҡ„еҲ йҷӨ

  • еҲ йҷӨд»…еҪұе“ҚеҪ“еүҚж§ҪдҪҚеҶ…йғЁз»“жһ„
  • дёҚз ҙеқҸж•ҙдҪ“жҹҘжүҫи·Ҝеҫ„

вҶ’ еұҖйғЁй—®йўҳпјҢеұҖйғЁи§ЈеҶі

2. ејҖж”ҫе®ҡеқҖжі•зҡ„еҲ йҷӨ

  • з®ҖеҚ•зҪ®з©әдјҡз ҙеқҸжҺўжөӢй“ҫ
  • еҗҺз»ӯе…ғзҙ еҸҜиғҪ"еҶҚд№ҹжүҫдёҚеҲ°"

е·ҘзЁӢдёҠеёёи§Ғж–№жЎҲпјҡ

  • еў“зў‘ж Үи®°пјҲDeleted Markerпјү
  • жҲ–еұҖйғЁйҮҚжҺ’

еҲ йҷӨд№ӢжүҖд»ҘеӨҚжқӮпјҢжҳҜеӣ дёәејҖж”ҫе®ҡеқҖжі•жҠҠ"и·Ҝеҫ„дҝЎжҒҜ"йҡҗеҗ«еңЁж•°з»„з»“жһ„дёӯгҖӮ

е…ӯгҖҒжү©е®№дёҺ rehashпјҡж— жі•еӣһйҒҝзҡ„йҮҚжһ„жҲҗжң¬

1. дёәд»Җд№ҲдёҚиғҪзӣҙжҺҘжҗ¬иҝҒ

  • ж§ҪдҪҚж•°йҮҸж”№еҸҳ
  • еҸ–жЁЎз»“жһңеҝ…з„¶ж”№еҸҳ

вҶ’ еҺҹжңүжҳ е°„е…ізі»ж•ҙдҪ“еӨұж•Ҳ

2. rehash зҡ„жң¬иҙЁ

rehash дёҚжҳҜжҖ§иғҪй—®йўҳпјҢиҖҢжҳҜж•°еӯҰеҝ…з„¶гҖӮ

д»»дҪ•еҹәдәҺеҸ–жЁЎжҲ–иҢғеӣҙжҳ е°„зҡ„з»“жһ„пјҢеңЁе®№йҮҸеҸҳеҢ–ж—¶пјҡ

  • еҝ…йЎ»йҮҚж–°и®Ўз®—дҪҚзҪ®

3. жёҗиҝӣејҸжү©е®№зҡ„е·ҘзЁӢж„Ҹд№үпјҲе®һзҺ°еұӮжҸҗзӨәпјү

  • жҠҠдёҖж¬ЎжҖ§жҲҗжң¬
  • ж‘ҠеҲ°еӨҡж¬Ўж“ҚдҪңдёӯ

иҝҷжҳҜе·ҘзЁӢдјҳеҢ–зӯ–з•ҘпјҢдёҚж”№еҸҳд»»дҪ•еҺҹзҗҶз»“и®әгҖӮ

дёғгҖҒж•ЈеҲ—иЎЁеңЁж•°жҚ®з»“жһ„дҪ“зі»дёӯзҡ„иҫ№з•Ң

ж•ЈеҲ—иЎЁйҖӮеҗҲпјҡ

  • Key вҶ’ Value жҳ е°„
  • дёҚе…іеҝғйЎәеәҸ
  • й«ҳйў‘жҹҘжүҫ

ж•ЈеҲ—иЎЁдёҚйҖӮеҗҲпјҡ

  • иҢғеӣҙжҹҘиҜў
  • жңүеәҸйҒҚеҺҶ
  • дёҘж јжңҖеқҸж—¶й—ҙеӨҚжқӮеәҰиҰҒжұӮ

ж•ЈеҲ—иЎЁдёҚжҳҜ"дёҮиғҪжҹҘжүҫз»“жһ„"пјҢиҖҢжҳҜ"жңҖдјҳжҳ е°„з»“жһ„"гҖӮ

е…«гҖҒзЁіе®ҡзҹҘиҜҶжҖ»з»“

ж•ЈеҲ—иЎЁзҡ„жүҖжңүи®ҫи®ЎпјҢйғҪеҸҜд»ҘиҝҳеҺҹдёәд»ҘдёӢеҮ жқЎдёҚеҸҜеҠЁж‘Үзҡ„дәӢе®һпјҡ

  1. жңүйҷҗз©әй—ҙдёӯпјҢеҶІзӘҒеҝ…з„¶еӯҳеңЁ
  2. иЈ…иҪҪеӣ еӯҗеҶіе®ҡеҶІзӘҒжҰӮзҺҮ
  3. еҶІзӘҒи§ЈеҶізӯ–з•Ҙ = е·ҘзЁӢеҸ–иҲҚ
