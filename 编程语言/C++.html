const/volatile/mutable

const

  • е®ғжҳҜдёҖдёӘзұ»еһӢдҝ®йҘ°з¬ҰпјҢеҸҜд»Ҙз»ҷд»»дҪ•еҜ№иұЎйҷ„еҠ дёҠвҖңеҸӘиҜ»вҖқеұһжҖ§пјҢдҝқиҜҒе®үе…Ё
  • е®ғеҸҜд»Ҙдҝ®йҘ°еј•з”Ёе’ҢжҢҮй’ҲпјҢвҖңconst &вҖқеҸҜд»Ҙеј•з”Ёд»»дҪ•зұ»еһӢпјҢжҳҜеҮҪж•°е…ҘеҸЈеҸӮж•°зҡ„жңҖдҪізұ»еһӢ
  • е®ғиҝҳеҸҜд»Ҙдҝ®йҘ°жҲҗе‘ҳеҮҪж•°пјҢиЎЁзӨәеҮҪж•°жҳҜвҖңеҸӘиҜ»вҖқзҡ„пјҢconst еҜ№иұЎеҸӘиғҪи°ғз”Ё const жҲҗе‘ҳеҮҪж•°

volatile

  • иЎЁзӨәеҸҳйҮҸеҸҜиғҪдјҡиў«вҖңдёҚиў«еҜҹи§үвҖқең°дҝ®ж”№пјҢзҰҒжӯўзј–иҜ‘еҷЁдјҳеҢ–пјҢжҜҸж¬ЎдҪҝз”Ёж—¶пјҢйғҪиҰҒеҺ»еҶ…еӯҳдёӯиҜ»

mutable

  • еҜ№дәҺдёҖдәӣзү№ж®Ҡзҡ„еҸҳйҮҸпјҢз»ҷе®ғеҠ дёҠ mutable дҝ®йҘ°пјҢи§ЈйҷӨ const зҡ„йҷҗеҲ¶пјҢи®©д»»дҪ•жҲҗе‘ҳеҮҪж•°йғҪеҸҜд»Ҙж“ҚдҪңе®ғ
class A {  public:    int a = 1;    mutable int b = 2;    void test() {      a = 2;    }    void test1() const {      // a = 2; дёҚиЎҢ      b = 2;    }};int main() {  const A a = A();  // a.test(); ж— жі•и°ғз”Ё  a.test1();  return 1;}
