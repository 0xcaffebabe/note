еңЁеӨҡж•°жҠҖжңҜиө„ж–ҷдёӯпјҢвҖңе°Ҷ .java зј–иҜ‘дёә .classвҖқеҫҖеҫҖиў«жҸҸиҝ°дёәдёҖзі»еҲ—зәҝжҖ§жӯҘйӘӨпјҡи§ЈжһҗгҖҒж ЎйӘҢгҖҒз”ҹжҲҗеӯ—иҠӮз ҒгҖӮдҪҶиҝҷз§ҚжҸҸиҝ°и§ЈйҮҠдәҶеҸ‘з”ҹдәҶд»Җд№ҲпјҢеҚҙжІЎжңүи§ЈйҮҠдёәд»Җд№ҲиҰҒиҝҷж ·и®ҫи®ЎгҖӮ

иҰҒзңҹжӯЈзҗҶи§Ј Java зј–иҜ‘еҷЁпјҢжҲ‘д»¬йңҖиҰҒе°Ҷи§Ҷи§’д»ҺвҖңжөҒзЁӢжү§иЎҢвҖқжҸҗеҚҮеҲ°пјҡ

зј–иҜ‘еҷЁ = дёҖеҘ—е°ҶвҖңдәәзұ»иҜӯиЁҖвҖқиҪ¬еҢ–дёәвҖңиҷҡжӢҹжңәеҸҜжү§иЎҢжЁЎеһӢвҖқзҡ„иҜӯиЁҖзҗҶи§Јзі»з»ҹ

жң¬ж–Үе°Ҷд»Ҙ Java зј–иҜ‘еүҚз«ҜпјҲjavacпјү дёәдҫӢпјҢйҮҚжһ„е…¶ж ёеҝғеҺҹзҗҶгҖҒзЁіе®ҡжһ¶жһ„жЁЎеһӢдёҺиғҢеҗҺзҡ„е·ҘзЁӢе“ІеӯҰгҖӮ

дәҢгҖҒз¬¬дёҖжҖ§еҺҹзҗҶпјҡд»Җд№ҲжҳҜвҖңзј–иҜ‘еүҚз«ҜвҖқпјҹ

д»ҺжҠҪиұЎеұӮйқўзңӢпјҢжүҖжңүзј–иҜ‘еҷЁйғҪеҸҜд»Ҙиў«жӢҶеҲҶдёәдёӨеӨ§йғЁеҲҶпјҡ

  • **еүҚз«ҜпјҲFrontendпјү**пјҡзҗҶи§ЈиҜӯиЁҖеҗ«д№ү
  • **еҗҺз«ҜпјҲBackendпјү**пјҡз”ҹжҲҗзӣ®ж ҮиЎЁзӨәпјҲеӯ—иҠӮз Ғ / жңәеҷЁз Ғпјү

зј–иҜ‘еүҚз«Ҝзҡ„жң¬иҙЁиҒҢиҙЈ

зј–иҜ‘еүҚз«Ҝзҡ„е”ҜдёҖзӣ®ж ҮпјҡжҠҠвҖңж–Үжң¬вҖқиҪ¬еҢ–дёәвҖңиҜӯд№үдёҠиҮӘжҙҪзҡ„зЁӢеәҸжЁЎеһӢвҖқ

е®ғе№¶дёҚе…іеҝғжҖ§иғҪдјҳеҢ–гҖҒжҢҮд»Өи°ғеәҰжҲ–иҝҗиЎҢж—¶жү§иЎҢпјҢеҸӘе…іжіЁпјҡ

  • д»Јз ҒвҖңжҳҜеҗҰеҗҲжі•вҖқ
  • еҗ«д№үвҖңжҳҜеҗҰжҳҺзЎ®вҖқ
  • иғҪеҗҰиў«жҳ е°„дёәдёҖз§Қжӣҙеә•еұӮгҖҒзЁіе®ҡзҡ„дёӯй—ҙиЎЁзӨә

дёүгҖҒзЁіе®ҡжҠҪиұЎжЁЎеһӢпјҡJava зј–иҜ‘еүҚз«ҜдёүеұӮз»“жһ„

дёәдәҶж‘Ҷи„ұжөҒзЁӢи®°еҝҶпјҢе…Ҳе»әз«ӢдёҖдёӘеҸҜиҝҒз§»зҡ„зЁіе®ҡжЁЎеһӢпјҡ

в”ңв”Җ 1. иҜӯжі•еұӮпјҲSyntax Layerпјү
в”Ӯ   в””в”Җ еӯ—з¬Ұ вҶ’ Token вҶ’ AST
в”ңв”Җ 2. иҜӯд№үеұӮпјҲSemantic Layerпјү
в”Ӯ   в”ңв”Җ з¬ҰеҸ·иЎЁпјҲSymbol Tableпјү
в”Ӯ   в”ңв”Җ зұ»еһӢзі»з»ҹпјҲType Systemпјү
в”Ӯ   в”ңв”Җ ж•°жҚ®жөҒ / жҺ§еҲ¶жөҒеҲҶжһҗ
в”ңв”Җ 3. иҜӯиЁҖеҸҳжҚўеұӮпјҲDesugar Layerпјү
в”Ӯ   в””в”Җ иҜӯжі•зі– вҶ’ ж ёеҝғиҜӯиЁҖз»“жһ„

иҜҘжЁЎеһӢеҗҢж ·йҖӮз”ЁдәҺ KotlinгҖҒScalaгҖҒC#гҖҒBabelгҖҒLLVM еүҚз«Ҝ

еҗҺж–ҮжүҖжңүеҶ…е®№пјҢеқҮе°Ҷжҳ е°„еҲ°иҝҷдёҖжЁЎеһӢдёӯгҖӮ

еӣӣгҖҒиҜӯжі•еұӮпјҡд»Һеӯ—з¬ҰеҲ°жҠҪиұЎиҜӯжі•ж ‘пјҲASTпјү

1. иҜӯжі•еұӮи§ЈеҶізҡ„й—®йўҳ

иҜӯжі•еұӮеҸӘеӣһзӯ”дёҖдёӘй—®йўҳпјҡ

иҝҷж®өд»Јз ҒеңЁеҪўејҸдёҠжҳҜеҗҰз¬ҰеҗҲиҜӯиЁҖе®ҡд№үпјҹ

е®ғе№¶дёҚзҗҶи§ЈвҖңзұ»еһӢжҳҜеҗҰеҢ№й…ҚвҖқпјҢд№ҹдёҚзҗҶи§ЈвҖңеҸҳйҮҸжҳҜеҗҰе·ІиөӢеҖјвҖқгҖӮ

2. ж ёеҝғдә§зү©пјҡAST

  • жәҗд»Јз Ғиў«иҪ¬жҚўдёә Token жөҒ
  • Token жөҒж №жҚ®иҜӯжі•и§„еҲҷжһ„е»ә AST

дёҖж—Ұ AST жһ„е»әе®ҢжҲҗпјҡ

жәҗз Ғеӯ—з¬ҰжөҒе°Ҷиў«еҪ»еә•жҠӣејғпјҢеҗҺз»ӯжүҖжңүж“ҚдҪңеҸӘеҹәдәҺ AST

иҝҷдҪ“зҺ°дәҶдёҖз§Қе…ій”®и®ҫи®ЎжҖқжғіпјҡ

зј–иҜ‘еҷЁдёҚеӨ„зҗҶж–Үжң¬пјҢиҖҢжҳҜеӨ„зҗҶз»“жһ„еҢ–иҜӯд№үжЁЎеһӢ

дә”гҖҒиҜӯд№үеұӮпјҡи®©зЁӢеәҸвҖңеңЁйҖ»иҫ‘дёҠжҲҗз«ӢвҖқ

иҜӯд№үеұӮжҳҜзј–иҜ‘еүҚз«Ҝзҡ„ж ёеҝғпјҢе…¶д»»еҠЎжҳҜпјҡ

йӘҢиҜҒ AST жҳҜеҗҰеңЁиҜӯиЁҖи§„еҲҷе’ҢдёҠдёӢж–ҮдёӯиҮӘжҙҪ

1. з¬ҰеҸ·иЎЁпјҡеҗҚеӯ—дёҺе®һдҪ“зҡ„жҳ е°„зі»з»ҹ

з¬ҰеҸ·иЎЁжң¬иҙЁдёҠи§ЈеҶізҡ„жҳҜпјҡ

вҖңеҗҚеӯ—вҖқ вҶ” вҖңиҜӯиЁҖе®һдҪ“вҖқ зҡ„з»‘е®ҡй—®йўҳ

е®ғж”Ҝж’‘дәҶпјҡ

  • еҸҳйҮҸ / ж–№жі• / зұ»зҡ„дҪңз”Ёеҹҹ
  • йҮҚиҪҪи§Јжһҗ
  • и®ҝй—®жҺ§еҲ¶

з¬ҰеҸ·иЎЁе№¶йқһдёҖж¬ЎжҖ§з»“жһ„пјҢиҖҢжҳҜпјҡ

иҙҜз©ҝж•ҙдёӘиҜӯд№үеҲҶжһҗйҳ¶ж®өзҡ„ж ёеҝғеҹәзЎҖи®ҫж–Ҫ

2. зұ»еһӢзі»з»ҹдёҺж ҮжіЁжЈҖжҹҘ

еңЁиҝҷдёҖйҳ¶ж®өпјҢзј–иҜ‘еҷЁйӘҢиҜҒпјҡ

  • еҸҳйҮҸжҳҜеҗҰе…ҲеЈ°жҳҺеҗҺдҪҝз”Ё
  • иөӢеҖјжҳҜеҗҰж»Ўи¶ізұ»еһӢзәҰжқҹ
  • еёёйҮҸиЎЁиҫҫејҸжҳҜеҗҰеҸҜжҠҳеҸ 

иҝҷдёҖжӯҘдҪ“зҺ°дәҶ Java зҡ„ж ёеҝғе·ҘзЁӢе“ІеӯҰд№ӢдёҖпјҡ

е°ҪеҸҜиғҪеңЁзј–иҜ‘жңҹжҡҙйңІй”ҷиҜҜпјҢиҖҢйқһиҝҗиЎҢжңҹеӨұиҙҘ

3. ж•°жҚ®жөҒдёҺжҺ§еҲ¶жөҒеҲҶжһҗ

иҝӣдёҖжӯҘйӘҢиҜҒзЁӢеәҸзҡ„вҖңйҖ»иҫ‘е®ҢеӨҮжҖ§вҖқпјҢдҫӢеҰӮпјҡ

  • еұҖйғЁеҸҳйҮҸжҳҜеҗҰеңЁдҪҝз”ЁеүҚе·ІиөӢеҖј
  • ж–№жі•жҳҜеҗҰеңЁжүҖжңүи·Ҝеҫ„дёҠиҝ”еӣһ
  • `final` иҜӯд№үжҳҜеҗҰиў«з ҙеқҸ

иҝҷдёҖеұӮе№¶дёҚз”ҹжҲҗд»Јз ҒпјҢиҖҢжҳҜеңЁзәҰжқҹзЁӢеәҸиЎҢдёәз©әй—ҙгҖӮ

е…ӯгҖҒиҜӯиЁҖеҸҳжҚўеұӮпјҡиҜӯжі•зі–зҡ„жң¬иҙЁ

1. д»Җд№ҲжҳҜиҜӯжі•зі–пјҹ

иҜӯжі•зі–е№¶дёҚжҳҜвҖңж–°иғҪеҠӣвҖқпјҢиҖҢжҳҜпјҡ

еҜ№ж ёеҝғиҜӯиЁҖиғҪеҠӣзҡ„иҜӯжі•зә§е°ҒиЈ…

Java йҖүжӢ©еңЁзј–иҜ‘еүҚз«Ҝе°Ҷе…¶иҝҳеҺҹпјҲdesugarпјүпјҢиҖҢйқһеңЁ JVM дёӯж”ҜжҢҒгҖӮ

2. жіӣеһӢпјҡзұ»еһӢзі»з»ҹдёҺиҷҡжӢҹжңәзҡ„жҠҳдёӯ

и®ҫи®ЎдәӢе®һ

  • JVM жҢҮд»ӨйӣҶдёҚж”ҜжҢҒеҸӮж•°еҢ–зұ»еһӢ
  • ж—©жңҹ Java ж— жіӣеһӢпјҢз”ҹжҖҒеәһеӨ§

и®ҫи®ЎйҖүжӢ©

зј–иҜ‘жңҹжіӣеһӢ + иҝҗиЎҢжңҹж“ҰйҷӨ

иҝҷжҳҜдёҖж¬ЎжҳҺзЎ®зҡ„е·ҘзЁӢеҸ–иҲҚпјҡ

з»ҙеәҰ йҖүжӢ©
еҗ‘еҗҺе…је®№ вң…
иҝҗиЎҢжңҹзұ»еһӢдҝЎжҒҜ вқҢ
иҷҡжӢҹжңәеӨҚжқӮеәҰ жңҖе°Ҹ

жіӣеһӢеӣ жӯӨжҲҗдёәпјҡ

еӯҳеңЁдәҺвҖңзұ»еһӢжЈҖжҹҘйҳ¶ж®өвҖқзҡ„иҜӯиЁҖиғҪеҠӣпјҢиҖҢйқһиҝҗиЎҢжңҹиғҪеҠӣ

3. иҮӘеҠЁиЈ…жӢҶз®ұдёҺ foreach

иҝҷдәӣзү№жҖ§жң¬иҙЁдёҠпјҡ

  • дёҚж”№еҸҳ JVM иғҪеҠӣ
  • еҸӘжҸҗеҚҮжәҗз ҒиЎЁиҫҫеҠӣ

йҖҡиҝҮ desugar иҪ¬жҚўдёәпјҡ

  • жҳҫејҸеҜ№иұЎеҲӣе»ә
  • иҝӯд»ЈеҷЁи°ғз”Ё

дҪ“зҺ°дәҶ Java зҡ„дёҖиҙҜз«Ӣеңәпјҡ

иҜӯиЁҖеұӮеҸӢеҘҪ вү  иҷҡжӢҹжңәеұӮеӨҚжқӮ

дёғгҖҒеӯ—иҠӮз Ғз”ҹжҲҗпјҡеүҚз«Ҝзҡ„з»ҲзӮ№

еңЁе®ҢжҲҗиҜӯд№үйӘҢиҜҒдёҺиҜӯиЁҖиҝҳеҺҹеҗҺпјҢзј–иҜ‘еүҚз«Ҝе°Ҷпјҡ

  • дёәзұ»иЎҘе…Ё `` дёҺ ``
  • е°Ҷ AST жҳ е°„дёә JVM еӯ—иҠӮз Ғз»“жһ„

иҮіжӯӨпјҢеүҚз«Ҝзҡ„дҪҝе‘Ҫе®ҢжҲҗгҖӮ

еҗҺз»ӯзҡ„жҖ§иғҪгҖҒжү§иЎҢгҖҒдјҳеҢ–пјҢе·Іе®Ңе…ЁеұһдәҺ JVM дёҺиҝҗиЎҢжңҹзі»з»ҹгҖӮ

е…«гҖҒжіЁи§ЈеӨ„зҗҶеҷЁпјҡеҸ—жҺ§зҡ„зј–иҜ‘жңҹжү©еұ•жңәеҲ¶

1. жіЁи§ЈеӨ„зҗҶеҷЁзҡ„жң¬иҙЁе®ҡдҪҚ

жіЁи§ЈеӨ„зҗҶеҷЁе№¶дёҚжҳҜвҖңй»‘йӯ”жі•вҖқпјҢиҖҢжҳҜпјҡ

зј–иҜ‘еүҚз«ҜжҡҙйңІеҮәзҡ„гҖҒеҸ—жҺ§зҡ„жҸ’д»¶жү©еұ•зӮ№

е®ғе…Ғи®ёеңЁпјҡ

  • AST жһ„е»әе®ҢжҲҗеҗҺ
  • еӯ—иҠӮз Ғз”ҹжҲҗд№ӢеүҚ

иҝӣиЎҢпјҡ

  • ж ЎйӘҢ
  • д»Јз Ғз”ҹжҲҗ
  • е…ғж•°жҚ®ж”¶йӣҶ

2. иғҪеҠӣиҫ№з•Ңи®ӨзҹҘпјҲйқһеёёе…ій”®пјү

з»ҙеәҰ жіЁи§ЈеӨ„зҗҶеҷЁ
жүҖеұһйҳ¶ж®ө зј–иҜ‘еүҚз«Ҝ
ж“ҚдҪңеҜ№иұЎ AST / з¬ҰеҸ·
йҖӮеҗҲеңәжҷҜ ж ЎйӘҢгҖҒз”ҹжҲҗжәҗз Ғ
дёҚйҖӮеҗҲ жҺ§еҲ¶жөҒйҮҚеҶҷгҖҒAOP

е®ғдҪ“зҺ°зҡ„жҳҜ Java зҡ„и®ҫи®Ўе…ӢеҲ¶пјҡ

е…Ғи®ёжү©еұ•пјҢдҪҶдёҚе…Ғи®ёз ҙеқҸиҜӯиЁҖдёҺиҷҡжӢҹжңәзҡ„зЁіе®ҡиҫ№з•Ң

д№қгҖҒж•ҙдҪ“и®ҫи®Ўе“ІеӯҰжҖ»з»“

Java зј–иҜ‘еүҚз«Ҝзҡ„жүҖжңүи®ҫи®ЎпјҢжңҖз»ҲйғҪжҢҮеҗ‘дёүжқЎй•ҝжңҹдёҚеҸҳзҡ„еҺҹеҲҷпјҡ

  1. **зј–иҜ‘жңҹеҸ‘зҺ°й—®йўҳпјҢиҝҗиЎҢжңҹдҝқжҢҒз®ҖеҚ•**
  2. **иҜӯиЁҖжј”иҝӣдёҚиғҪз ҙеқҸж—ўжңүз”ҹжҖҒ**
  3. **иҷҡжӢҹжңәдҝқжҢҒжңҖе°ҸиғҪеҠӣйӣҶпјҢеӨҚжқӮжҖ§еүҚз§»еҲ°зј–иҜ‘жңҹ**

зҗҶи§ЈиҝҷдёҖзӮ№пјҢжүҚиғҪзңҹжӯЈзҗҶи§Јпјҡ

  • дёәд»Җд№Ҳ Java жіӣеһӢжҳҜж“ҰйҷӨзҡ„
  • дёәд»Җд№ҲиҜӯжі•зі–е…ЁйғЁеңЁеүҚз«Ҝи§ЈеҶі
  • дёәд»Җд№ҲжіЁи§ЈеӨ„зҗҶеҷЁиғҪеҠӣиў«дёҘж јйҷҗеҲ¶
