JVM зј–иҜ‘зі»з»ҹе№¶йқһдёәдәҶвҖңжӣҙеҝ«вҖқиҖҢеӯҳеңЁпјҢиҖҢжҳҜдёәдәҶи§ЈеҶі Java иҝҗиЎҢжЁЎеһӢдёӯзҡ„дёүз»„ж №жң¬зҹӣзӣҫпјҡ

  1. еҗҜеҠЁйҖҹеәҰ vs еі°еҖјжҖ§иғҪ

    • зәҜи§ЈйҮҠпјҡеҗҜеҠЁеҝ«гҖҒй•ҝжңҹж…ў
    • зәҜйқҷжҖҒзј–иҜ‘пјҡеҗҜеҠЁж…ўгҖҒеі°еҖјй«ҳ

  2. зј–иҜ‘жҲҗжң¬ vs дјҳеҢ–ж”¶зӣҠ

    • ж·ұеәҰдјҳеҢ–йңҖиҰҒж—¶й—ҙ
    • е№¶йқһжүҖжңүд»Јз ҒйғҪеҖјеҫ—иў«дјҳеҢ–

  3. йқҷжҖҒзЎ®е®ҡжҖ§ vs иҝҗиЎҢж—¶дёҚзЎ®е®ҡжҖ§

    • Java зҡ„еӨҡжҖҒгҖҒеҠЁжҖҒеҠ иҪҪгҖҒеҸҚе°„
    • еңЁзј–иҜ‘жңҹж— жі•е®Ңе…ЁзЎ®е®ҡиЎҢдёә

JVM зј–иҜ‘зі»з»ҹзҡ„жң¬иҙЁзӣ®ж ҮжҳҜпјҡ еңЁиҝҗиЎҢиҝҮзЁӢдёӯпјҢз”ЁжңҖе°Ҹзҡ„йўқеӨ–жҲҗжң¬пјҢдёәжңҖжңүд»·еҖјзҡ„д»Јз ҒжҚўеҸ–жңҖеӨ§жҖ§иғҪж”¶зӣҠгҖӮ

2. еҚіж—¶зј–иҜ‘зҡ„е“ІеӯҰеҹәзЎҖ

еҚіж—¶зј–иҜ‘пјҲJITпјүе№¶дёҚжҳҜвҖңзј–иҜ‘еҫ—жӣҙжҷҡвҖқпјҢиҖҢжҳҜпјҡ

  • **з”ЁиҝҗиЎҢж—¶дҝЎжҒҜжӣҝд»ЈйқҷжҖҒеҒҮи®ҫ**
  • **з”ЁжҰӮзҺҮеҲӨж–ӯжӣҝд»Јз»қеҜ№жӯЈзЎ®**
  • **е…Ғи®ёеӨұиҙҘпјҢеҶҚеӣһйҖҖ**

иҝҷдҪҝ JVM зј–иҜ‘жҲҗдёәдёҖз§Қпјҡ

д»Ҙз»ҹи®ЎеӯҰдёҺеҸҚйҰҲдёәеҹәзЎҖзҡ„е·ҘзЁӢзі»з»ҹпјҢиҖҢйқһдј з»ҹзј–иҜ‘еҷЁгҖӮ

дәҢгҖҒжү§иЎҢжһ¶жһ„жҖ»и§Ҳпјҡи§ЈйҮҠеҷЁдёҺзј–иҜ‘еҷЁзҡ„еҚҸеҗҢзі»з»ҹ

1. дёәд»Җд№Ҳ JVM еҝ…йЎ»еҗҢж—¶еӯҳеңЁи§ЈйҮҠеҷЁдёҺзј–иҜ‘еҷЁ

и§ЈйҮҠеҷЁдёҺзј–иҜ‘еҷЁе№¶йқһеҜ№з«ӢпјҢиҖҢжҳҜеҲҶе·ҘжҳҺзЎ®пјҡ

и§’иүІ ж ёеҝғд»·еҖј
и§ЈйҮҠеҷЁ еҝ«йҖҹеҗҜеҠЁгҖҒдҪҺжҲҗжң¬жү§иЎҢ
зј–иҜ‘еҷЁ й«ҳжҖ§иғҪгҖҒж·ұеәҰдјҳеҢ–

JVM йҮҮз”Ёзҡ„жҳҜдёҖз§Қ жёҗиҝӣејҸжү§иЎҢжЁЎеһӢпјҡ

е…Ҳи§ЈйҮҠжү§иЎҢ вҶ’ еҶҚеұҖйғЁзј–иҜ‘ вҶ’ жңҖз»Ҳж·ұеәҰдјҳеҢ–

иҝҷдёҖжЁЎејҸиў«з§°дёәпјҡж··еҗҲжү§иЎҢжЁЎејҸпјҲMixed ModeпјүгҖӮ

2. жү§иЎҢжЁЎејҸзҡ„жһҒз«ҜеҪўжҖҒ

  • зәҜи§ЈйҮҠжЁЎејҸпјҲ-Xintпјүпјҡ

    • жңҖдҪҺеҗҜеҠЁжҲҗжң¬
    • жңҖе·®й•ҝжңҹжҖ§иғҪ

  • зәҜзј–иҜ‘жЁЎејҸпјҲ-Xcompпјүпјҡ

    • еҗҜеҠЁжһҒж…ў
    • е®һйҷ…еҫҲе°‘дҪҝз”Ё

е®ғд»¬зҡ„еӯҳеңЁжӣҙеӨҡжҳҜдёәдәҶпјҡ

  • и°ғиҜ•
  • е®һйӘҢ
  • жҖ§иғҪеҜ№жҜ”

дёүгҖҒеҲҶеұӮзј–иҜ‘пјҡдёҖз§ҚжёҗиҝӣејҸжҠ•иө„жЁЎеһӢ

1. еҲҶеұӮзј–иҜ‘зҡ„жң¬иҙЁ

еҲҶеұӮзј–иҜ‘пјҲTiered Compilationпјүе№¶дёҚжҳҜвҖңеӨҡзј–иҜ‘еҮ ж¬ЎвҖқпјҢиҖҢжҳҜдёҖз§Қпјҡ

еҜ№д»Јз Ғд»·еҖјйҖҗжӯҘеҠ еӨ§жҠ•иө„зҡ„еҶізӯ–жЁЎеһӢгҖӮ

жҜҸдёҖеұӮйғҪеӣһзӯ”дёҖдёӘй—®йўҳпјҡ

иҝҷж®өд»Јз ҒпјҢеҖјдёҚеҖјеҫ—иҠұжӣҙеӨҡж—¶й—ҙеҺ»дјҳеҢ–пјҹ

2. еҲҶеұӮз»“жһ„зҡ„жҠҪиұЎзҗҶи§Ј

еұӮзә§ жҠ•иө„жҲҗжң¬ еӣһжҠҘйў„жңҹ еҶізӯ–дҫқжҚ®
0 жһҒдҪҺ жһҒдҪҺ еҶ·д»Јз Ғ
1вҖ“3 дёӯзӯү дёӯзӯү зғӯеәҰеўһй•ҝ
4 жһҒй«ҳ жһҒй«ҳ зЁіе®ҡзғӯзӮ№

еҲҶеұӮзј–иҜ‘зҡ„е…ій”®дёҚжҳҜеұӮж•°пјҢиҖҢжҳҜвҖңеҸҜйҖҶжҖ§вҖқпјҡ

  • зј–иҜ‘еҸҜд»ҘеӣһйҖҖ
  • еҒҮи®ҫеҸҜд»Ҙиў«жҺЁзҝ»

еӣӣгҖҒзғӯзӮ№д»Јз ҒиҜҶеҲ«пјҡз”Ёз»ҹи®Ўжӣҝд»Јзӣҙи§ү

1. зғӯзӮ№жЈҖжөӢзҡ„ж ёеҝғжҖқжғі

JVM дёҚе°қиҜ•еҲӨж–ӯвҖңе“Әж®өд»Јз ҒйҮҚиҰҒвҖқпјҢиҖҢжҳҜеҲӨж–ӯпјҡ

е“Әж®өд»Јз Ғиў«еҸҚеӨҚиҜҒжҳҺжҳҜйҮҚиҰҒзҡ„гҖӮ

еӣ жӯӨйҮҮз”Ёпјҡ

  • и®Ўж•°еҷЁ
  • иЎ°еҮҸжңәеҲ¶

иҝҷжҳҜдёҖдёӘж—¶й—ҙзӘ—еҸЈеҶ…зҡ„з»ҹи®Ўй—®йўҳгҖӮ

2. ж–№жі•зғӯзӮ№дёҺеҫӘзҺҜзғӯзӮ№зҡ„жң¬иҙЁе·®ејӮ

  • ж–№жі•зғӯзӮ№пјҡ

    • е…іжіЁвҖңиў«йў‘з№Ғи°ғз”ЁвҖқ

  • еҫӘзҺҜзғӯзӮ№пјҲOSRпјүпјҡ

    • е…іжіЁвҖңеңЁдёҖж¬Ўи°ғз”Ёдёӯж¶ҲиҖ—еӨ§йҮҸж—¶й—ҙвҖқ

OSR зҡ„еј•е…ҘпјҢжң¬иҙЁдёҠжҳҜпјҡ

жүҝи®ӨвҖңж–№жі•зІ’еәҰиҝҮзІ—вҖқпјҢе№¶е…Ғи®ёеңЁжү§иЎҢдёӯйҖ”еҲҮжҚўжү§иЎҢзӯ–з•ҘгҖӮ

дә”гҖҒJIT зј–иҜ‘жөҒзЁӢпјҡд»ҺжҠҪиұЎиҜӯд№үеҲ°жңәеҷЁзҺ°е®һ

1. еӨҡеұӮ IR зҡ„и®ҫи®Ўе“ІеӯҰ

JVM зј–иҜ‘еҷЁйҮҮз”ЁеӨҡзә§дёӯй—ҙиЎЁзӨәпјҡ

IR еұӮзә§ е…іжіЁзӮ№
HIR иҜӯд№үгҖҒжҺ§еҲ¶жөҒ
LIR жңәеҷЁзү№жҖ§гҖҒеҜ„еӯҳеҷЁ

е…¶жң¬иҙЁжҳҜпјҡ

еҲҶзҰ»вҖңиҜӯд№үжӯЈзЎ®жҖ§вҖқдёҺвҖңзЎ¬д»¶ж•ҲзҺҮвҖқгҖӮ

2. дёәд»Җд№Ҳ JVM еҒҸеҘҪзәҝжҖ§жү«жҸҸеҜ„еӯҳеҷЁеҲҶй…Қ

  • зј–иҜ‘йҖҹеәҰдјҳе…ҲдәҺжңҖдјҳи§Ј
  • JIT жҳҜеңЁзәҝзі»з»ҹпјҢиҖҢйқһзҰ»зәҝзј–иҜ‘

иҝҷжҳҜдёҖдёӘе…ёеһӢзҡ„ е·ҘзЁӢжқғиЎЎйҖүжӢ©гҖӮ

е…ӯгҖҒзј–иҜ‘еҷЁзӯ–з•ҘеұӮпјҡC1гҖҒC2 дёҺ Graal

1. зј–иҜ‘еҷЁдёҚжҳҜзүҲжң¬жј”иҝӣпјҢиҖҢжҳҜзӯ–з•ҘеҲҶеҢ–

зј–иҜ‘еҷЁ ж ёеҝғзӯ–з•Ҙ и®ҫи®ЎеҸ–еҗ‘
C1 еҝ«йҖҹз”ҹжҲҗ еҗҜеҠЁдёҺеҸҚйҰҲ
C2 ж·ұеәҰдјҳеҢ– еі°еҖјжҖ§иғҪ
Graal еҸҜжү©еұ• IR иҜӯиЁҖдёҺжһ¶жһ„жј”иҝӣ

Graal зҡ„еҮәзҺ°пјҢе№¶дёҚжҳҜдёәдәҶвҖңжӣҙеҝ«вҖқпјҢиҖҢжҳҜдёәдәҶи§ЈеҶіпјҡ

дј з»ҹзј–иҜ‘еҷЁжһ¶жһ„йҡҫд»Ҙжј”иҝӣзҡ„й—®йўҳгҖӮ

2. JVMCI зҡ„зңҹжӯЈж„Ҹд№ү

JVMCI зҡ„еј•е…ҘпјҢж„Ҹе‘ізқҖпјҡ

зј–иҜ‘еҷЁд»Һ JVM еҶ…ж ёдёӯи§ЈиҖҰпјҢжҲҗдёәеҸҜжӣҝжҚўз»„д»¶гҖӮ

иҝҷжҳҜдёҖж¬Ўжһ¶жһ„еұӮйқўзҡ„зӘҒз ҙпјҢиҖҢйқһжҖ§иғҪдјҳеҢ–гҖӮ

дёғгҖҒзј–иҜ‘дјҳеҢ–жҠҖжңҜпјҡд»ҺжҠҖе·§еҲ°еҺҹзҗҶ

1. зј–иҜ‘дјҳеҢ–зҡ„дә”еӨ§жң¬иҙЁзұ»еҲ«

1пёҸвғЈ жҺ§еҲ¶жөҒз®ҖеҢ–

  • ж–№жі•еҶ…иҒ”
  • еҫӘзҺҜеұ•ејҖ

зӣ®ж ҮпјҡжҡҙйңІжӣҙеӨҡдјҳеҢ–з©әй—ҙ

2пёҸвғЈ ж•°жҚ®жөҒдјҳеҢ–

  • е…¬е…ұеӯҗиЎЁиҫҫејҸж¶ҲйҷӨ
  • ж ҮйҮҸжӣҝжҚў

зӣ®ж ҮпјҡеҮҸе°‘еҶ—дҪҷи®Ўз®—

3пёҸвғЈ еҶ…еӯҳдёҺе№¶еҸ‘иҜӯд№үдјҳеҢ–

  • йҖғйҖёеҲҶжһҗ
  • еҗҢжӯҘж¶ҲйҷӨ

зӣ®ж ҮпјҡеүҠеҮҸдёҚеҝ…иҰҒзҡ„еҶ…еӯҳдёҺеҗҢжӯҘжҲҗжң¬

4пёҸвғЈ зЎ¬д»¶дәІе’ҢдјҳеҢ–

  • еҗ‘йҮҸеҢ–
  • Intrinsic

зӣ®ж Үпјҡиҙҙиҝ‘ CPU иғҪеҠӣиҫ№з•Ң

5пёҸвғЈ е®үе…ЁжЈҖжҹҘж¶ҲйҷӨ

  • ж•°з»„иҫ№з•ҢжЈҖжҹҘ
  • з©әжҢҮй’ҲжЈҖжҹҘ

зӣ®ж Үпјҡз”ЁзЎ¬д»¶ејӮеёёжӣҝд»ЈиҪҜд»¶еҲҶж”Ҝ

е…«гҖҒAOTгҖҒJNI дёҺ JIT зҡ„дҪ“зі»е®ҡдҪҚ

1. дёүзұ»жҠҖжңҜзҡ„еұӮзә§е·®ејӮ

жҠҖжңҜ жүҖеӨ„йҳ¶ж®ө и§ЈеҶізҡ„й—®йўҳ
JIT иҝҗиЎҢжңҹ еі°еҖјжҖ§иғҪ
AOT йғЁзҪІжңҹ еҗҜеҠЁдёҺиө„жәҗ
JNI иҜӯиЁҖиҫ№з•Ң иғҪеҠӣеӨҚз”Ё

е®ғд»¬дёҚжҳҜз«һдәүе…ізі»пјҢиҖҢжҳҜпјҡ

еңЁдёҚеҗҢйҳ¶ж®өи§ЈеҶідёҚеҗҢзәҰжқҹжқЎд»¶гҖӮ

д№қгҖҒжј”иҝӣи¶ӢеҠҝдёҺиҫ№з•Ңи®ӨзҹҘ

1. JVM зј–иҜ‘зі»з»ҹзҡ„жј”иҝӣж–№еҗ‘

  • жӣҙејәзҡ„иҝҗиЎҢж—¶еҸҚйҰҲпјҲPGOпјү
  • жӣҙзҒөжҙ»зҡ„зј–иҜ‘еҷЁжһ¶жһ„пјҲGraalпјү
  • жӣҙж—©д»Ӣе…Ҙзҡ„дјҳеҢ–пјҲAOT + JIT еҚҸеҗҢпјү

2. еҚіж—¶зј–иҜ‘зҡ„еӨ©з„¶иҫ№з•Ң

JIT ж— жі•и§ЈеҶіпјҡ

  • IO дё»еҜјеһӢжҖ§иғҪй—®йўҳ
  • з®—жі•еӨҚжқӮеәҰй—®йўҳ
  • жһ¶жһ„зә§е№¶еҸ‘з“¶йўҲ

зј–иҜ‘еҷЁдјҳеҢ–ж°ёиҝңжҳҜ"ж”ҫеӨ§еҷЁ"пјҢиҖҢдёҚжҳҜ"еҲӣйҖ еҷЁ"гҖӮ
