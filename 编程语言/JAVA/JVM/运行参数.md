# 运行参数

## 参数类型

- 标准参数
  - -help
  - -version
- -X参数（非标准参数）
  - Xint
  - Xcomp
- -XX参数
  - -XX:newSize
  - -XX:+UseSerialGC

### -D参数

用来获取命令行传入的参数

```java
System.out.println(System.getProperty("str"));
```

```shell
java -Dstr=123 Main.java # JDK13可以直接编译并运行
```

### -XX参数

- boolean类型
  - `-XX:+name`表示启用
  - `-XX:-name`表示不启用
- 非boolean类型
  - `-XX:name=value`

### -Xms与-Xmx

```text
-Xmx2048m 设置最大堆内存为2048M
-Xms523m 设置初始堆内存为512M
```

### 查看JVM运行参数

```
-XX:+PrintCommandLineFlags
```

- 查看正在运行的JVM参数

```
jinfo -flags pid
```

- 可以通过jps命令列出JAVA进程
