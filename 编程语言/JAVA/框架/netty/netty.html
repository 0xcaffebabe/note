Netty

NettyжҳҜз”ұJBOSSжҸҗдҫӣзҡ„дёҖдёӘjavaејҖжәҗжЎҶжһ¶гҖӮNettyжҸҗдҫӣејӮжӯҘзҡ„гҖҒдәӢд»¶й©ұеҠЁзҡ„зҪ‘з»ңеә”з”ЁзЁӢеәҸжЎҶжһ¶е’Ңе·Ҙе…·пјҢз”Ёд»Ҙеҝ«йҖҹејҖеҸ‘й«ҳжҖ§иғҪгҖҒй«ҳеҸҜйқ жҖ§зҡ„зҪ‘з»ңжңҚеҠЎеҷЁе’Ңе®ўжҲ·з«ҜзЁӢеәҸгҖӮ

  • ејӮжӯҘйҖҡдҝЎйўҶеҹҹ

зү№жҖ§

еҲҶзұ»зү№жҖ§
и®ҫи®Ўз»ҹдёҖзҡ„API,ж”ҜжҢҒеӨҡз§Қдј иҫ“зұ»еһӢпјҢйҳ»еЎһзҡ„е’Ңйқһйҳ»еЎһзҡ„пјӣз®ҖеҚ•иҖҢејәеӨ§зҡ„зәҝзЁӢжЁЎеһӢпјӣзңҹжӯЈзҡ„ж— иҝһжҺҘж•°жҚ®жҠҘеҘ—жҺҘеӯ—ж”ҜжҢҒпјӣй“ҫжҺҘйҖ»иҫ‘з»„д»¶д»Ҙж”ҜжҢҒеӨҚз”Ё
жҖ§иғҪжӢҘжңүжҜ”Javaзҡ„ж ёеҝғAPIжӣҙй«ҳзҡ„еҗһеҗҗйҮҸд»ҘеҸҠжӣҙдҪҺзҡ„е»¶иҝҹпјӣеҫ—зӣҠдәҺжұ еҢ–е’ҢеӨҚз”ЁпјҢжӢҘжңүжӣҙдҪҺзҡ„иө„жәҗж¶ҲиҖ—пјӣжңҖе°‘зҡ„еҶ…еӯҳеӨҚеҲ¶
еҒҘеЈ®жҖ§дёҚдјҡеӣ дёәж…ўйҖҹгҖҒеҝ«йҖҹжҲ–иҖ…и¶…иҪҪзҡ„иҝһжҺҘиҖҢеҜјиҮҙOutofMemoryErrorпјӣж¶ҲйҷӨеңЁй«ҳйҖҹзҪ‘з»ңдёӯNIOеә”з”ЁзЁӢеәҸеёёи§Ғзҡ„дёҚе…¬е№іиҜ»/еҶҷжҜ”зҺҮ
  • йқһйҳ»еЎһзҪ‘з»ңи°ғз”ЁдҪҝеҫ—жҲ‘д»¬еҸҜд»ҘдёҚеҝ…зӯүеҫ…дёҖдёӘж“ҚдҪңзҡ„е®ҢжҲҗгҖӮе®Ңе…ЁејӮжӯҘзҡ„I/OжӯЈжҳҜеҹәдәҺиҝҷдёӘзү№жҖ§жһ„е»әзҡ„пјҢе№¶дё”жӣҙиҝӣдёҖжӯҘпјҡејӮжӯҘж–№жі•дјҡз«ӢеҚіиҝ”еӣһпјҢе№¶дё”еңЁе®ғе®ҢжҲҗж—¶пјҢдјҡзӣҙжҺҘжҲ–иҖ…еңЁзЁҚеҗҺзҡ„жҹҗдёӘж—¶й—ҙзӮ№йҖҡзҹҘз”ЁжҲ·гҖӮ
  • йҖүжӢ©еҷЁдҪҝеҫ—жҲ‘д»¬иғҪеӨҹйҖҡиҝҮиҫғе°‘зҡ„зәҝзЁӢдҫҝеҸҜзӣ‘и§Ҷи®ёеӨҡиҝһжҺҘдёҠзҡ„дәӢд»¶гҖӮ
