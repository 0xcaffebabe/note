Servlet зҡ„жң¬иҙЁжҳҜдёҖз§Қ жңҚеҠЎеҷЁз«Ҝз»„д»¶и§„иҢғпјҢе®ғе®ҡд№үдәҶдёҖдёӘиғҪеӨҹеҠЁжҖҒз”ҹжҲҗеҶ…е®№зҡ„ Java зұ»еә”йҒөеҫӘзҡ„жҺҘеҸЈдёҺз”ҹе‘Ҫе‘ЁжңҹеҚҸи®®гҖӮ е®ғдёҚзӣҙжҺҘеҸӮдёҺзҪ‘з»ңйҖҡдҝЎпјҢиҖҢжҳҜйҖҡиҝҮдёҖеҘ—ж ҮеҮҶеҢ– APIпјҢи®©еә”з”ЁејҖеҸ‘иҖ…д»Ҙз»ҹдёҖж–№ејҸзј–еҶҷеҠЁжҖҒ Web з»„д»¶гҖӮ

е…¶ж ёеҝғжҖқжғіжҳҜпјҡ

йҖҡиҝҮжҺҘеҸЈдёҺз”ҹе‘Ҫе‘Ёжңҹи§„иҢғпјҢе®һзҺ° Web иҜ·жұӮеӨ„зҗҶзҡ„ж ҮеҮҶеҢ–дёҺеҸҜз§»жӨҚжҖ§гҖӮ

1.2 з»ҹдёҖжҺҘеҸЈжҠҪиұЎдёҺз”ҹе‘Ҫе‘ЁжңҹжЁЎеһӢ

Servlet йҖҡиҝҮ javax.servlet.Servlet жҺҘеҸЈпјҲеҸҠе…¶еӯҗжҺҘеҸЈ HttpServletпјүе®ҡд№үдәҶз»ҹдёҖзҡ„з»„д»¶еҘ‘зәҰгҖӮ жүҖжңүе®һзҺ°зұ»йғҪеҝ…йЎ»е®һзҺ°д»ҘдёӢж ёеҝғж–№жі•пјҡ

  • `init(ServletConfig config)`пјҡеҲқе§ӢеҢ–иө„жәҗпјҢд»…и°ғз”ЁдёҖж¬ЎгҖӮ
  • `service(ServletRequest req, ServletResponse res)`пјҡеӨ„зҗҶиҜ·жұӮпјҢжҜҸж¬ЎиҜ·жұӮи°ғз”ЁдёҖж¬ЎгҖӮ
  • `destroy()`пјҡйҮҠж”ҫиө„жәҗпјҢд»…и°ғз”ЁдёҖж¬ЎгҖӮ

Servlet зҡ„з”ҹе‘Ҫе‘Ёжңҹз”ұе®№еҷЁе…Ёжқғз®ЎзҗҶпјҢж•ҙдҪ“жөҒзЁӢдёәпјҡ

е®һдҫӢеҢ– вҶ’ еҲқе§ӢеҢ–(init) вҶ’ еӨҡж¬ЎжңҚеҠЎ(service) вҶ’ й”ҖжҜҒ(destroy)

иҝҷз§ҚжЁЎејҸдҪҝе®№еҷЁиғҪеӨҹж ҮеҮҶеҢ–ең°жҺ§еҲ¶з»„д»¶з”ҹе‘Ҫе‘ЁжңҹпјҢзЎ®дҝқиө„жәҗзҡ„жңүеәҸеҲҶй…ҚдёҺеӣһж”¶гҖӮ

1.3 иҜ·жұӮ-е“Қеә”жЁЎеһӢдёҺе®№еҷЁеҢ–йғЁзҪІ

service() ж–№жі•жҳҜ Servlet зҡ„ж ёеҝғпјҢе®ғжҠҪиұЎдәҶвҖңжҺҘж”¶иҜ·жұӮ вҶ’ еӨ„зҗҶдёҡеҠЎ вҶ’ з”ҹжҲҗе“Қеә”вҖқзҡ„йҖҡз”ЁжЁЎеһӢгҖӮ еә•еұӮзҡ„ HTTP еҚҸи®®и§ЈжһҗгҖҒиҝһжҺҘз®ЎзҗҶзӯүеӨҚжқӮйҖ»иҫ‘пјҢйғҪз”ұе®№еҷЁе®ҢжҲҗгҖӮејҖеҸ‘иҖ…еҸӘйңҖдё“жіЁдәҺдёҡеҠЎйҖ»иҫ‘гҖӮ

Servlet жң¬иә«еҸӘжҳҜдёҖдёӘз¬ҰеҗҲи§„иҢғзҡ„ Java еҜ№иұЎпјҢиҖҢе®№еҷЁпјҲеҰӮ TomcatгҖҒJettyпјүеҲҷиҙҹиҙЈпјҡ

  • е®һдҫӢеҲӣе»әдёҺй”ҖжҜҒ
  • з”ҹе‘Ҫе‘Ёжңҹеӣһи°ғ
  • еӨҡзәҝзЁӢе№¶еҸ‘еӨ„зҗҶпјҲеҚ•е®һдҫӢеӨҡзәҝзЁӢпјү
  • зәҝзЁӢе®үе…ЁдёҺи°ғеәҰз®ЎзҗҶ

и®ҫи®ЎиҰҒзӮ№пјҡз”ұдәҺе®№еҷЁйҖҡеёёдёәжҜҸдёӘ Servlet зұ»еҸӘеҲӣе»әдёҖдёӘе®һдҫӢпјҢе»әи®®дҝқжҢҒ Servlet зҡ„ж— зҠ¶жҖҒи®ҫи®ЎпјҢйҒҝе…ҚжҲҗе‘ҳеҸҳйҮҸе…ұдә«еј•еҸ‘зҡ„зәҝзЁӢе®үе…Ёй—®йўҳгҖӮ

1.4 дёҠдёӢж–ҮдёҺй…ҚзҪ®з®ЎзҗҶжңәеҲ¶

  • **ServletContext**пјҡеә”з”Ёзә§е…ұдә«зҺҜеўғпјҢжҸҗдҫӣе…ЁеұҖй…ҚзҪ®дёҺиө„жәҗи®ҝй—®иғҪеҠӣгҖӮ
  • **ServletConfig**пјҡServlet е®һдҫӢзә§й…ҚзҪ®пјҢжіЁе…ҘеҲқе§ӢеҢ–еҸӮж•°гҖӮ

иҝҷдёӨиҖ…е…ұеҗҢжһ„жҲҗдәҶ Servlet зҡ„иҝҗиЎҢдёҠдёӢж–ҮпјҢдҪҝз»„д»¶иғҪеӨҹж ҮеҮҶеҢ–ең°и®ҝй—®зҺҜеўғдҝЎжҒҜгҖҒе…ұдә«иө„жәҗе’Ңй…ҚзҪ®гҖӮ

1.5 жһ¶жһ„е“ІеӯҰпјҡе…іжіЁзӮ№еҲҶзҰ»

Servlet и§„иҢғйҖҡиҝҮи§ЈиҖҰе®һзҺ°дәҶеӨҡеұӮиҒҢиҙЈеҲ’еҲҶпјҡ

иҒҢиҙЈ иҜҙжҳҺ
еҚҸи®®еӨ„зҗҶ е®№еҷЁе®ҢжҲҗзҪ‘з»ңйҖҡдҝЎдёҺHTTPи§Јжһҗ
з”ҹе‘Ҫе‘Ёжңҹз®ЎзҗҶ е®№еҷЁиҙҹиҙЈе®һдҫӢз®ЎзҗҶдёҺиө„жәҗеӣһж”¶
дёҡеҠЎйҖ»иҫ‘ з”ұејҖеҸ‘иҖ…е®һзҺ° service() жҲ– doXXX() ж–№жі•
зҺҜеўғй…ҚзҪ® йҖҡиҝҮServletContext/ConfigжҸҗдҫӣ

иҝҷз§Қи®ҫи®Ўе®һзҺ°дәҶе…ёеһӢзҡ„е…іжіЁзӮ№еҲҶзҰ» (Separation of Concerns)пјҢжһҒеӨ§жҸҗеҚҮдәҶ Web еә”з”Ёзҡ„ејҖеҸ‘ж•ҲзҺҮдёҺеҸҜз»ҙжҠӨжҖ§гҖӮ

2. з”ҹе‘Ҫе‘Ёжңҹдёӯзҡ„и®ҫи®ЎжЁЎејҸ

Servlet и§„иҢғдёҺе®№еҷЁи®ҫи®Ўдёӯи•ҙеҗ«дәҶеӨҡдёӘз»Ҹе…ёзҡ„йқўеҗ‘еҜ№иұЎи®ҫи®ЎжЁЎејҸпјҢиҝҷдәӣжЁЎејҸе…ұеҗҢжһ„е»әдәҶе…¶и§ЈиҖҰгҖҒзҒөжҙ»зҡ„з”ҹе‘Ҫе‘ЁжңҹдҪ“зі»гҖӮ

2.1 жЁЎжқҝж–№жі•жЁЎејҸпјҲTemplate Methodпјү

HttpServlet.service() жҳҜе…ёеһӢзҡ„жЁЎжқҝж–№жі•пјҡ е®ғе®ҡд№үдәҶ HTTP иҜ·жұӮеӨ„зҗҶзҡ„жЁЎжқҝпјҲи§ЈжһҗиҜ·жұӮж–№жі• вҶ’ еҲҶжҙҫеҲ° doGetгҖҒdoPost зӯүпјүпјҢ иҖҢе…·дҪ“дёҡеҠЎйҖ»иҫ‘еҲҷз”ұејҖеҸ‘иҖ…еңЁеӯҗзұ»дёӯйҮҚеҶҷеҜ№еә”ж–№жі•е®һзҺ°гҖӮ

2.2 еҚ•дҫӢжЁЎејҸпјҲSingletonпјү

е®№еҷЁйҖҡеёёдёәжҜҸдёӘ Servlet зұ»еҸӘеҲӣе»әдёҖдёӘе®һдҫӢгҖӮ иҝҷз§ҚеҚ•дҫӢејҸе…ұдә«иғҪеӨҹжҳҫи‘—жҸҗеҚҮеҶ…еӯҳеҲ©з”ЁзҺҮпјҢдҪҶд№ҹиҰҒжұӮејҖеҸ‘иҖ…жіЁж„ҸзәҝзЁӢе®үе…ЁжҖ§гҖӮ

е»әи®®пјҡServlet и®ҫи®Ўеә”дҝқжҢҒж— зҠ¶жҖҒпјҢжҲ–дҪҝз”ЁеҗҢжӯҘжңәеҲ¶дҝқжҠӨе…ұдә«иө„жәҗгҖӮ

2.3 е·ҘеҺӮжЁЎејҸпјҲFactoryпјү

Servlet е®№еҷЁжң¬иә«е……еҪ“е·ҘеҺӮи§’иүІпјҢдҫқжҚ®й…ҚзҪ®ж–Үд»¶жҲ–жіЁи§ЈеҲӣе»әе№¶з®ЎзҗҶ Servlet е®һдҫӢпјҢ йҡҗи—ҸдәҶеҜ№иұЎеҲӣе»әдёҺдҫқиө–з®ЎзҗҶзҡ„еӨҚжқӮжҖ§пјҢе®һзҺ°дәҶиҝҗиЎҢж—¶зҡ„зҒөжҙ»йғЁзҪІгҖӮ

2.4 и§ӮеҜҹиҖ…жЁЎејҸпјҲObserverпјү

ServletContextListenerгҖҒHttpSessionListenerгҖҒServletRequestListener зӯүзӣ‘еҗ¬еҷЁжңәеҲ¶пјҢ дҪҝеҫ—ејҖеҸ‘иҖ…еҸҜд»Ҙи®ўйҳ…е®№еҷЁдәӢд»¶пјҢе“Қеә”еҜ№иұЎзҡ„еҲӣе»әгҖҒй”ҖжҜҒзӯүз”ҹе‘Ҫе‘ЁжңҹеҸҳеҢ–гҖӮ

2.5 иҙЈд»»й“ҫжЁЎејҸпјҲChain of Responsibilityпјү

Filter дёҺ FilterChain жһ„жҲҗиҙЈд»»й“ҫз»“жһ„гҖӮ жҜҸдёӘиҜ·жұӮеңЁиҝӣе…Ҙ Servlet еүҚпјҢдјҡжҢүй“ҫејҸйЎәеәҸз»ҸиҝҮеӨҡдёӘиҝҮж»ӨеҷЁпјҲеҰӮи®ӨиҜҒгҖҒж—Ҙеҝ—гҖҒи·ЁеҹҹзӯүпјүпјҢе®һзҺ°жЁӘеҲҮйҖ»иҫ‘зҡ„з»ҹдёҖеӨ„зҗҶгҖӮ

рҹ§© жЁЎејҸжҖ»з»“

иҝҷдәӣжЁЎејҸеҚҸеҗҢдҪңз”ЁпјҢе…ұеҗҢе®һзҺ°дәҶ Servlet е®№еҷЁзҡ„ж ёеҝғзү№жҖ§пјҡ

й«ҳеҶ…иҒҡгҖҒдҪҺиҖҰеҗҲгҖҒеҸҜжү©еұ•гҖҒеҸҜжҸ’жӢ”зҡ„з”ҹе‘Ҫе‘Ёжңҹз®ЎзҗҶдҪ“зі»гҖӮ

3. Servlet е®№еҷЁжһ¶жһ„еҺҹзҗҶ

Servlet е®№еҷЁпјҲеҰӮ TomcatпјүжҳҜ Servlet и§„иҢғзҡ„иҝҗиЎҢж—¶е®һзҺ°гҖӮе®ғиҙҹиҙЈзҪ‘з»ңйҖҡдҝЎгҖҒз»„д»¶з”ҹе‘Ҫе‘ЁжңҹгҖҒиҜ·жұӮи°ғеәҰдёҺе®үе…ЁжҺ§еҲ¶пјҢжҳҜж•ҙдёӘ Java Web жһ¶жһ„зҡ„ж ёеҝғжү§иЎҢеұӮгҖӮ

3.1 е®№еҷЁеҲҶеұӮз»“жһ„

е®№еҷЁеҶ…йғЁйҮҮз”Ё з»„еҗҲжЁЎејҸ (Composite Pattern) жһ„е»әеұӮзә§жһ¶жһ„пјҡ

EngineпјҲеј•ж“Һпјү
 в””в”Җв”Җ HostпјҲиҷҡжӢҹдё»жңәпјү
      в””в”Җв”Җ ContextпјҲWeb еә”з”Ёпјү
           в””в”Җв”Җ WrapperпјҲеҚ•дёӘ Servletпјү

жҜҸдёҖеұӮе°ҒиЈ…дёҚеҗҢиҒҢиҙЈпјҢд»Һе…ЁеұҖеҲ°еҚ• Servlet йҖҗзә§з»ҶеҢ–пјҢжңҖз»Ҳе®ҢжҲҗиҜ·жұӮеҲ°дёҡеҠЎйҖ»иҫ‘зҡ„еҲҶеҸ‘гҖӮ

3.2 иҜ·жұӮеӨ„зҗҶжөҒзЁӢ

дёҖдёӘе…ёеһӢ HTTP иҜ·жұӮзҡ„жөҒиҪ¬иҝҮзЁӢеҰӮдёӢпјҡ

  1. **Connector** жҺҘж”¶зҪ‘з»ңиҜ·жұӮе№¶и§Јжһҗдёә Servlet API еҜ№иұЎпјӣ
  2. **Engine** ж №жҚ®иҷҡжӢҹдё»жңәеҢ№й…Қ **Host**пјӣ
  3. **Host** ж №жҚ®дёҠдёӢж–Үи·Ҝеҫ„е®ҡдҪҚ **Context (Web еә”з”Ё)**пјӣ
  4. **Context** жҹҘжүҫеҢ№й…Қзҡ„ **Wrapper (Servlet)**пјӣ
  5. **Wrapper** и°ғз”Ё Servlet зҡ„ `service()` ж–№жі•пјӣ
  6. еӨ„зҗҶе®ҢжҜ•еҗҺз”ҹжҲҗе“Қеә”пјҢйҖҡиҝҮ Connector иҝ”еӣһе®ўжҲ·з«ҜгҖӮ

иҝҷдёҖз®ЎзәҝдҪ“зҺ°дәҶиҒҢиҙЈжё…жҷ°зҡ„еҲҶеұӮжЁЎеһӢпјҢд№ҹдёәжӢҰжҲӘеҷЁгҖҒж—Ҙеҝ—гҖҒи®ӨиҜҒзӯүеҠҹиғҪжҸҗдҫӣдәҶжү©еұ•иҠӮзӮ№гҖӮ

3.3 ж ёеҝғз»„д»¶иҒҢиҙЈ

з»„д»¶ иҒҢиҙЈиҜҙжҳҺ
Connector зҪ‘з»ңйҖҡдҝЎдёҺеҚҸи®®и§Јжһҗ
Container з”ҹе‘Ҫе‘ЁжңҹдёҺз»„д»¶з®ЎзҗҶ
Pipeline / Valve иҜ·жұӮз®ЎзәҝдёҺйҖҡз”ЁйҖ»иҫ‘жү©еұ•
ClassLoader еә”з”Ёйҡ”зҰ»зҡ„зұ»еҠ иҪҪжңәеҲ¶
Realm е®үе…Ёи®ӨиҜҒдёҺжҺҲжқғ
SessionManager HTTP дјҡиҜқз®ЎзҗҶ

зү№еҲ«ең°пјҢTomcat жү“з ҙж ҮеҮҶзҡ„еҸҢдәІе§”жҙҫжЁЎеһӢпјҢдёәжҜҸдёӘ Web еә”з”ЁжҸҗдҫӣзӢ¬з«Ӣзұ»еҠ иҪҪеҷЁпјҢе®һзҺ°жЁЎеқ—зә§йҡ”зҰ»гҖӮ

3.4 з”ҹе‘Ҫе‘Ёжңҹз»ҹдёҖз®ЎзҗҶ

жүҖжңүж ёеҝғз»„д»¶еқҮе®һзҺ° Lifecycle жҺҘеҸЈпјҢ йҖҡиҝҮз»ҹдёҖзҡ„ start()гҖҒstop()гҖҒdestroy() ж–№жі•иҝӣиЎҢз®ЎзҗҶпјҢе®һзҺ°е®№еҷЁзә§зҡ„еҸҜжҺ§еҗҜеҠЁдёҺе…ій—ӯжөҒзЁӢгҖӮ

4. Servlet и§„иҢғзҡ„жҠҪиұЎд»·еҖјдёҺз”ҹжҖҒж„Ҹд№ү

Servlet дёҚд»…жҳҜдёҖз»„жҺҘеҸЈпјҢжӣҙжҳҜдёҖз§Қе·ҘзЁӢе“ІеӯҰпјҡ д»Ҙж ҮеҮҶеҢ–гҖҒи§ЈиҖҰеҢ–зҡ„ж–№ејҸпјҢе®ҡд№ү Web з»„д»¶зҡ„иЎҢдёәиҫ№з•ҢдёҺиҝҗиЎҢж—¶жЁЎеһӢгҖӮ

4.1 ж ҮеҮҶеҢ–зҡ„иҝҗиЎҢж—¶жҠҪиұЎ

Servlet и§„иҢғжҠҪиұЎдәҶеә”з”ЁдёҺжңҚеҠЎеҷЁд№Ӣй—ҙзҡ„дәӨдә’еұӮпјҢ и®©ејҖеҸ‘иҖ…зј–еҶҷзҡ„д»Јз ҒдёҺе…·дҪ“жңҚеҠЎеҷЁе®һзҺ°пјҲTomcatгҖҒJettyгҖҒUndertowпјүи§ЈиҖҰгҖӮ д»ҺиҖҢе®һзҺ°и·ЁжңҚеҠЎеҷЁгҖҒи·ЁеҺӮе•Ҷзҡ„еҸҜз§»жӨҚжҖ§гҖӮ

4.2 е…іжіЁзӮ№еҲҶзҰ»зҡ„е·ҘзЁӢжҖқжғі

Servlet жҳҺзЎ®еҲ’еҲҶпјҡ

  • зҪ‘з»ңеҚҸи®®з”ұе®№еҷЁеӨ„зҗҶпјӣ
  • дёҡеҠЎйҖ»иҫ‘з”ұ Servlet е®һзҺ°пјӣ
  • з”ҹе‘Ҫе‘Ёжңҹз”ұи§„иҢғе®ҡд№үпјӣ
  • зҺҜеўғдёҺй…ҚзҪ®йҖҡиҝҮдёҠдёӢж–ҮжіЁе…ҘгҖӮ

иҝҷз§ҚеҲҶзҰ»еҘ е®ҡдәҶеҗҺз»ӯжЎҶжһ¶пјҲеҰӮ Spring MVCпјүдёӯвҖңжҺ§еҲ¶еҸҚиҪ¬ (IoC)вҖқдёҺвҖңдҫқиө–жіЁе…Ҙ (DI)вҖқзҡ„еҹәзЎҖжҖқжғігҖӮ

4.3 е®Ңж•ҙзҡ„з»„д»¶жЁЎеһӢ

Servlet и§„иҢғдёҚд»…еҢ…еҗ« Servlet жң¬иә«пјҢиҝҳе®ҡд№үдәҶ FilterгҖҒListener зӯүе‘Ёиҫ№жңәеҲ¶пјҢ е…ұеҗҢжһ„е»әдәҶдёҖдёӘеӨ„зҗҶжЁӘеҲҮе…іжіЁзӮ№пјҲи®ӨиҜҒгҖҒж—Ҙеҝ—гҖҒе®үе…Ёпјүзҡ„ж ҮеҮҶз»„д»¶з”ҹжҖҒгҖӮ

4.4 еҜ№зҺ°д»ЈжЎҶжһ¶зҡ„еҪұе“Қ

Spring MVCгҖҒJakarta EEгҖҒз”ҡиҮі Spring Boot зҡ„еөҢе…ҘејҸе®№еҷЁжңәеҲ¶пјҢ йғҪзӣҙжҺҘе»әз«ӢеңЁ Servlet и§„иҢғд№ӢдёҠгҖӮ DispatcherServlet е°ұжҳҜе…ёеһӢзҡ„еүҚз«ҜжҺ§еҲ¶еҷЁпјҲFront ControllerпјүпјҢ йҖҡиҝҮ Servlet е®№еҷЁиҝҗиЎҢпјҢе®һзҺ°е®Ңж•ҙзҡ„иҜ·жұӮеҲҶжҙҫгҖҒжҺ§еҲ¶дёҺи§ҶеӣҫжёІжҹ“гҖӮ

4.5 жҢҒз»ӯжј”иҝӣзҡ„ж ҮеҮҶ

е°Ҫз®ЎеҰӮд»ҠеҮәзҺ°дәҶејӮжӯҘдёҺе“Қеә”ејҸзј–зЁӢжЁЎеһӢпјҲеҰӮ WebFluxгҖҒNettyпјүпјҢ Servlet и§„иҢғдҫқз„¶жҳҜ еҗҢжӯҘ HTTP иҜ·жұӮжЁЎеһӢзҡ„дәӢе®һж ҮеҮҶпјҢ е№¶жҢҒз»ӯеңЁ Jakarta EE з”ҹжҖҒдёӯжј”иҝӣпјҲеҰӮ Servlet 5.0 ж”ҜжҢҒ HTTP/2гҖҒејӮжӯҘ I/OпјүгҖӮ

5. жҖ»з»“

Servlet жҳҜ Java Web жҠҖжңҜдҪ“зі»зҡ„ж №еҹәгҖӮ е®ғд»ҘвҖңи§„иҢғе…ҲиЎҢгҖҒе®һзҺ°и§ЈиҖҰгҖҒиҒҢиҙЈжё…жҷ°гҖҒз»„д»¶еҢ–вҖқдёәж ёеҝғзҗҶеҝөпјҢ еҘ е®ҡдәҶж•ҙдёӘдјҒдёҡзә§ Web жһ¶жһ„зҡ„е·ҘзЁӢеҹәзЎҖгҖӮ

дёҖеҸҘиҜқжҖ»з»“пјҡ Servlet жҳҜ Web дё–з•Ңзҡ„вҖңJVM зә§жҠҪиұЎеұӮвҖқвҖ”вҖ” е®ғд»Ҙж ҮеҮҶеҢ–жҺҘеҸЈе®ҡд№үдәҶ Web з»„д»¶зҡ„з”ҹе‘ҪеҪўејҸпјҢ д»Ҙе®№еҷЁеҢ–жһ¶жһ„е®һзҺ°дәҶй«ҳеҸҜз§»жӨҚгҖҒй«ҳжү©еұ•зҡ„дјҒдёҡзә§еә”з”ЁиҝҗиЎҢзҺҜеўғгҖӮ
