  2. static getDerivedStateFromProps гҖӮеҰӮжһңзұ»з»„д»¶е®ҡд№үдәҶиҝҷдёӘйқҷжҖҒж–№жі•пјҢеңЁз»„д»¶жҢӮиҪҪиҝҮзЁӢдёӯпјҢReact дјҡи°ғз”ЁиҝҷдёӘж–№жі•пјҢж №жҚ®иҝ”еӣһеҖјжқҘи®ҫзҪ® state
  3. render пјҢзұ»з»„д»¶еҝ…йЎ»иҰҒе®һзҺ°иҝҷдёӘж–№жі•гҖӮйҖҡеёёеңЁиҝ”еӣһеҖјдёӯдјҡдҪҝз”Ё JSX иҜӯжі•пјҢReact еңЁжҢӮиҪҪиҝҮзЁӢдёӯдјҡи°ғз”Ё render ж–№жі•иҺ·еҫ—з»„д»¶зҡ„е…ғзҙ ж ‘гҖӮж №жҚ®е…ғзҙ ж ‘пјҢReact жңҖз»Ҳдјҡз”ҹжҲҗеҜ№еә”зҡ„ DOM ж ‘

жҢӮиҪҪйҳ¶ж®өгҖӮReact дјҡжү§иЎҢз»„д»¶еҮҪж•°пјҢеңЁеҮҪж•°жү§иЎҢиҝҮзЁӢдёӯйҒҮеҲ°зҡ„ useState гҖҒ useMemo зӯү Hooks дҫқж¬ЎжҢӮиҪҪеҲ° FiberNode дёҠпјҢuseEffect е…¶е®һд№ҹдјҡиў«жҢӮиҪҪпјҢдҪҶе®ғеҢ…еҗ«зҡ„еүҜдҪңз”ЁпјҲSide-effectпјҢеңЁ Fiber еј•ж“Һдёӯз§°дёә Effectпјүдјҡдҝқз•ҷеҲ°жҸҗдәӨйҳ¶ж®ө

жӣҙж–°йҳ¶ж®өгҖӮеҪ“з»„д»¶жҺҘж”¶еҲ°ж–° propsпјҢи°ғз”Ё useState иҝ”еӣһзҡ„ setter жҲ–иҖ… useReducer иҝ”еӣһзҡ„ dispatch дҝ®ж”№дәҶзҠ¶жҖҒпјҢз»„д»¶дјҡиҝӣе…Ҙжӣҙж–°йҳ¶ж®өгҖӮз»„д»¶еҮҪж•°жң¬иә«дјҡиў«еҶҚж¬Ўжү§иЎҢпјҢHooks дјҡдҫқж¬ЎдёҺ FiberNode дёҠе·Із»ҸжҢӮиҪҪзҡ„ Hooks дёҖдёҖеҢ№й…ҚпјҢе№¶ж №жҚ®йңҖиҰҒжӣҙж–°гҖӮз»„д»¶еҮҪж•°зҡ„иҝ”еӣһеҖјз”ЁжқҘжӣҙж–° FiberNode ж ‘

жҸҗдәӨйҳ¶ж®өпјҢReact дјҡжӣҙж–°зңҹе®һ DOMгҖӮйҡҸеҗҺ React дјҡе…Ҳжү§иЎҢдёҠдёҖиҪ® Effect зҡ„жё…йҷӨеҮҪж•°пјҢз„¶еҗҺеҶҚж¬Ўжү§иЎҢ EffectгҖӮиҝҷйҮҢзҡ„ Effect еҢ…жӢ¬ useEffect дёҺuseLayoutEffect пјҢдёӨиҖ…зү№жҖ§еҫҲзӣёеғҸгҖӮе…¶дёӯuseLayoutEffect зҡ„ Effect жҳҜеңЁжӣҙж–°зңҹе®һ DOM д№ӢеҗҺеҗҢжӯҘжү§иЎҢзҡ„пјҢдёҺзұ»з»„д»¶зҡ„ componentDidMountгҖҒcomponentDidUpdate жӣҙзӣёдјјдёҖдәӣпјӣиҖҢ useEffect зҡ„ Effect жҳҜејӮжӯҘжү§иЎҢзҡ„пјҢдёҖиҲ¬жҷҡдәҺ useLayoutEffect

еҚёиҪҪйҳ¶ж®өгҖӮдё»иҰҒжҳҜжү§иЎҢ Effect зҡ„жё…йҷӨеҮҪж•°

еҮҪж•°з»„д»¶зҡ„й”ҷиҜҜеӨ„зҗҶдҫқиө–дәҺзҲ¶з»„д»¶жҲ–зҘ–е…Ҳз»„д»¶жҸҗдҫӣзҡ„й”ҷиҜҜиҫ№з•Ң
