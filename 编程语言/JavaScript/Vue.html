Vue дёҚд»…жҳҜдёҖдёӘжЎҶжһ¶пјҢжӣҙжҳҜеүҚз«Ҝе·ҘзЁӢеҢ–ж—¶д»Јзҡ„жһ¶жһ„жҖқжғіиЎЁиҫҫгҖӮ е®ғзҡ„ж ёеҝғд»·еҖјдёҚеңЁдәҺ APIпјҢиҖҢеңЁдәҺд»ҘгҖҢеЈ°жҳҺејҸгҖҚгҖҢе“Қеә”ејҸгҖҚгҖҢз»„д»¶еҢ–гҖҚдёәдёӯеҝғзҡ„зЁіе®ҡзҹҘиҜҶдҪ“зі»гҖӮ

1. жң¬иҙЁи®ӨзҹҘпјҡд»Һе‘Ҫд»ӨејҸеҲ°е“Қеә”ејҸзҡ„иҢғејҸи·ғиҝҒ

еүҚз«ҜејҖеҸ‘зҡ„ж №жң¬й—®йўҳжҳҜпјҡеҰӮдҪ•и®©з•ҢйқўжӯЈзЎ®ең°еҸҚжҳ зҠ¶жҖҒпјҲUI = f(state))гҖӮ еңЁдј з»ҹе‘Ҫд»ӨејҸжЁЎеһӢдёӯпјҢејҖеҸ‘иҖ…еҝ…йЎ»жүӢеҠЁж“ҚдҪң DOM жқҘеҗҢжӯҘз•ҢйқўдёҺж•°жҚ®пјҢеёҰжқҘдәҶеӨҚжқӮзҡ„еүҜдҪңз”ЁдёҺзҠ¶жҖҒдёҚдёҖиҮҙй—®йўҳгҖӮ

Vue зҡ„иҜһз”ҹжәҗиҮӘеҜ№жӯӨй—®йўҳзҡ„ж №жң¬жҠҪиұЎвҖ”вҖ” е®ғжҸҗеҮәдәҶдёҖдёӘж–°зҡ„и®ӨзҹҘжЁЎеһӢпјҡ

еЈ°жҳҺе…ізі»пјҢиҖҢйқһжү§иЎҢжӯҘйӘӨгҖӮ

ејҖеҸ‘иҖ…еҸӘйңҖеЈ°жҳҺгҖҢж•°жҚ®дёҺи§Ҷеӣҫзҡ„дҫқиө–е…ізі»гҖҚпјҢжЎҶжһ¶иҙҹиҙЈиҮӘеҠЁз»ҙжҠӨдёӨиҖ…д№Ӣй—ҙзҡ„еҗҢжӯҘгҖӮ иҝҷдёҖжҖқжғіжј”еҢ–еҮә Vue зҡ„дёүеӨ§зЁіе®ҡеҶ…ж ёпјҡ

  • **еЈ°жҳҺејҸжёІжҹ“** вҶ’ д»Һж“ҚдҪңиҪ¬еҗ‘жҸҸиҝ°
  • **е“Қеә”ејҸзі»з»ҹ** вҶ’ д»Һе‘Ҫд»ӨиҪ¬еҗ‘ж„ҹзҹҘ
  • **з»„д»¶еҢ–жЁЎеһӢ** вҶ’ д»ҺиҝҮзЁӢиҪ¬еҗ‘з»“жһ„

иҝҷдәӣзҗҶеҝөжһ„жҲҗдәҶ Vue зҡ„е“ІеӯҰж №еҹәпјҢд№ҹжһ„жҲҗдәҶзҺ°д»ЈеүҚз«ҜжЎҶжһ¶зҡ„зЁіе®ҡжЁЎејҸгҖӮ

2. ж ёеҝғи®ҫи®Ўе“ІеӯҰ

2.1 еЈ°жҳҺејҸжёІжҹ“пјҡи®©ж•°жҚ®вҖңиҜҙиҜқвҖқ

еңЁеЈ°жҳҺејҸиҢғејҸдёӯпјҢUI дёҚеҶҚжҳҜж“ҚдҪңз»“жһңпјҢиҖҢжҳҜж•°жҚ®зҠ¶жҖҒзҡ„иҮӘз„¶жҠ•еҪұгҖӮ ејҖеҸ‘иҖ…еҸӘйңҖжҸҸиҝ°гҖҢи§ҶеӣҫеҰӮдҪ•з”ұж•°жҚ®з”ҹжҲҗгҖҚпјҢжЎҶжһ¶иҙҹиҙЈиҮӘеҠЁжҺЁеҜјжӣҙж–°гҖӮ

иҝҷз§ҚжЁЎејҸдҪ“зҺ°дәҶдёүеұӮзЁіе®ҡжҖқжғіпјҡ

  • **жҠҪиұЎжҸҗеҚҮ**пјҡд»ҺвҖңж“ҚдҪңз•ҢйқўвҖқеҲ°вҖңеЈ°жҳҺе…ізі»вҖқ
  • **еҚ•еҗ‘дҫқиө–**пјҡи§Ҷеӣҫдҫқиө–ж•°жҚ®пјҢиҖҢйқһзӣёдә’з»‘е®ҡ
  • **иҮӘеҠЁеҗҢжӯҘ**пјҡйҖҡиҝҮдҫқиө–иҝҪиёӘж¶ҲйҷӨдәәе·ҘеҗҢжӯҘжҲҗжң¬

иҝҷдёҚд»…жҳҜдёҖз§Қзј–зЁӢж–№ејҸзҡ„иҪ¬еҸҳпјҢжӣҙжҳҜдёҖз§Қи®ӨзҹҘжЁЎејҸзҡ„з®ҖеҢ–гҖӮ

2.2 з»„д»¶еҢ–пјҡз»“жһ„зҡ„жҠҪиұЎдёҺеӨҚз”Ё

з»„д»¶еҢ–жҖқжғізҡ„ж ёеҝғпјҢдёҚеңЁдәҺе°ҒиЈ…д»Јз ҒпјҢиҖҢеңЁдәҺеҲҶзҰ»е…іжіЁзӮ№пјҲSeparation of ConcernsпјүгҖӮ жҜҸдёӘз»„д»¶жҳҜдёҖдёӘеҸҜеӨҚз”Ёзҡ„гҖҢиҮӘжІ»еҚ•е…ғгҖҚпјҢжӢҘжңүзӢ¬з«Ӣзҡ„жЁЎжқҝгҖҒйҖ»иҫ‘дёҺж ·ејҸгҖӮ

Vue д»ҘгҖҢз»„еҗҲдјҳдәҺз»§жүҝгҖҚдёәе“ІеӯҰеҹәзЎҖпјҢйҖҡиҝҮпјҡ

  • `props` е®һзҺ°ж•°жҚ®дёӢиЎҢ
  • дәӢд»¶ (`$emit`) е®һзҺ°иЎҢдёәдёҠиЎҢ
  • дҪңз”ЁеҹҹжҸ’ж§ҪдёҺ Composition API е®һзҺ°еҠҹиғҪзҡ„жЁӘеҗ‘з»„еҗҲ

з”ұжӯӨпјҢеӨҚжқӮзі»з»ҹиў«йҮҚжһ„дёәеҸҜз»„еҗҲгҖҒеҸҜжү©еұ•гҖҒеҸҜжј”еҢ–зҡ„жЁЎеқ—еҢ–зҪ‘з»ңгҖӮ иҝҷжҳҜдёҖз§Қ**з»„еҗҲејҸжһ¶жһ„пјҲComposition-Oriented Architectureпјү**зҡ„дҪ“зҺ°гҖӮ

2.3 жёҗиҝӣејҸпјҡеҶ…ж ёдёҺз”ҹжҖҒзҡ„еұӮж¬ЎеҢ–

Vue йҮҮз”ЁвҖңжёҗиҝӣејҸвҖқи®ҫи®ЎпјҢиҝҷжҳҜдёҖз§ҚеҲҶеұӮдҫқиө–е“ІеӯҰпјҡ

жЎҶжһ¶зҡ„ж ёеҝғеә”еҪ“зӢ¬з«ӢеӯҳеңЁпјҢиҖҢе‘Ёиҫ№иғҪеҠӣеә”д»ҘжҸ’д»¶еҪўејҸжј”еҢ–гҖӮ

иҝҷз§Қи®ҫи®ЎдҪ“зҺ°дәҶдёӨдёӘзЁіе®ҡеҺҹеҲҷпјҡ

  • **еҶ…ж ёзЁіе®ҡжҖ§**пјҡж ёеҝғиғҪеҠӣеә”й•ҝжңҹеҸҜйў„жңҹпјҲе“Қеә”ејҸгҖҒз»„д»¶еҢ–гҖҒжёІжҹ“жңәеҲ¶пјү
  • **з”ҹжҖҒзҒөжҙ»жҖ§**пјҡеҠҹиғҪеҸҜжҢүйңҖз»„еҗҲпјҲRouterгҖҒPiniaгҖҒSSRзӯүпјү

жёҗиҝӣејҸдёҚд»…жҳҜдёҖз§ҚдҪҝз”Ёзӯ–з•ҘпјҢжӣҙжҳҜдёҖдёӘз”ҹжҖҒеҸҜжј”еҢ–жЁЎеһӢгҖӮ

3. жһ¶жһ„еҺҹзҗҶдёҺжңәеҲ¶жҠҪиұЎ

3.1 е“Қеә”ејҸзі»з»ҹпјҡиҮӘеҠЁдҫқиө–иҝҪиёӘжңәеҲ¶

Vue зҡ„е“Қеә”ејҸзі»з»ҹеҸҜи§ҶдёәдёҖз§ҚвҖңж•°жҚ®жөҒж„ҹзҹҘеј•ж“ҺвҖқгҖӮ е…¶ж ёеҝғжңәеҲ¶еҹәдәҺи§ӮеҜҹиҖ…жЁЎејҸ (Observer Pattern)пјҡ

  • ж•°жҚ®иў« `getter` / `setter` жҲ– `Proxy` д»ЈзҗҶ
  • еҪ“ж•°жҚ®иў«и®ҝй—®ж—¶ж”¶йӣҶдҫқиө–
  • еҪ“ж•°жҚ®еҸҳеҢ–ж—¶и§ҰеҸ‘дҫқиө–жӣҙж–°

жң¬иҙЁдёҠпјҢиҝҷжҳҜд»ҺжҳҫејҸдәӢд»¶й©ұеҠЁеҲ°йҡҗејҸж•°жҚ®й©ұеҠЁзҡ„иҪ¬еҸҳгҖӮ иҝҷз§ҚиҮӘеҠЁеҢ–жңәеҲ¶еҪўжҲҗдәҶвҖңжҺЁеҜјејҸзҠ¶жҖҒжЁЎеһӢвҖқвҖ”вҖ”зҠ¶жҖҒж”№еҸҳеҚійҖ»иҫ‘жөҒеҠЁгҖӮ

3.2 иҷҡжӢҹ DOMпјҡжҠҪиұЎжёІжҹ“ж ‘дёҺе·®еҲҶжӣҙж–°

иҷҡжӢҹ DOM е№¶йқһжҖ§иғҪдјҳеҢ–зҡ„еҒ¶з„¶дә§зү©пјҢиҖҢжҳҜи§ҶеӣҫеұӮжҠҪиұЎзҡ„еҝ…иҰҒдёӯй—ҙеұӮгҖӮ е®ғжҸҗдҫӣдәҶпјҡ

  • **з»“жһ„жҠҪиұЎ**пјҡе°Ҷ UI иЎЁзӨәдёәеҸҜжҜ”иҫғзҡ„ж ‘
  • **еҸҳжӣҙи®Ўз®—**пјҡйҖҡиҝҮ diff з®—жі•и®Ўз®—жңҖе°Ҹжӣҙж–°и·Ҝеҫ„
  • **еҸҜз§»жӨҚжҖ§**пјҡжёІжҹ“йҖ»иҫ‘зӢ¬з«ӢдәҺе№іеҸ°пјҲWebгҖҒNativeгҖҒServerпјү

жң¬иҙЁдёҠпјҢиҷҡжӢҹ DOM жҳҜдёҖз§ҚжёІжҹ“и§ЈиҖҰеұӮпјҢдҝқиҜҒдәҶе“Қеә”ејҸдёҺи·Ёе№іеҸ°жёІжҹ“зҡ„з»ҹдёҖиҜӯд№үгҖӮ

3.3 з”ҹе‘Ҫе‘ЁжңҹпјҡеҸҜзј–зЁӢзҡ„зҠ¶жҖҒжңә

жҜҸдёӘз»„д»¶зҡ„еӯҳеңЁиҝҮзЁӢеҸҜд»Ҙиў«и§ҶдёәдёҖдёӘжңүйҷҗзҠ¶жҖҒжңәпјҲFSMпјүгҖӮ з”ҹе‘Ҫе‘Ёжңҹй’©еӯҗжҳҜиҝҷдёӘзҠ¶жҖҒжңәзҡ„вҖңдәӢд»¶еӣһи°ғжҺҘеҸЈвҖқпјҢи®©ејҖеҸ‘иҖ…иғҪеңЁзҠ¶жҖҒиҪ¬жҚўж—¶жү§иЎҢзү№е®ҡйҖ»иҫ‘гҖӮ

иҝҷз§ҚжЁЎејҸзҡ„жң¬иҙЁжҳҜпјҡ

е°Ҷж—¶й—ҙиҪҙеҸҜзј–зЁӢеҢ–пјҢдҪҝиө„жәҗгҖҒж•°жҚ®дёҺжёІжҹ“еңЁз”ҹе‘Ҫе‘ЁжңҹеҶ…еҫ—еҲ°жңүеәҸз®ЎзҗҶгҖӮ

иҝҷдҪ“зҺ°дәҶдёҖз§ҚйҖҡз”Ёзҡ„зі»з»ҹжҖқжғіпјҡ д»ҘзҠ¶жҖҒдёәж ёеҝғпјҢд»Ҙй’©еӯҗдёәжҺҘеҸЈпјҢе®һзҺ°ж—¶еәҸеҸҜжҺ§зҡ„з»“жһ„еҢ–жү§иЎҢгҖӮ

3.4 зҠ¶жҖҒз®ЎзҗҶпјҡдёҖиҮҙжҖ§дёҺеҸҜйў„жөӢжҖ§жЁЎеһӢ

еңЁеӨҚжқӮзі»з»ҹдёӯпјҢеұҖйғЁзҠ¶жҖҒйҡҫд»ҘдҝқиҜҒж•ҙдҪ“дёҖиҮҙжҖ§гҖӮ Vuex дёҺ Pinia жҸҗдҫӣдәҶдёҖз§ҚйӣҶдёӯејҸзҠ¶жҖҒдёҖиҮҙжҖ§жҺ§еҲ¶жЁЎеһӢпјҡ

  • зҠ¶жҖҒйӣҶдёӯеӯҳеӮЁпјҢйҒҝе…Қе…ұдә«ж··д№ұ
  • жүҖжңүзҠ¶жҖҒеҸҳжӣҙйҖҡиҝҮжҳҫејҸи·Ҝеҫ„пјҲmutations/actionsпјү
  • ејӮжӯҘеүҜдҪңз”ЁдёҺеҗҢжӯҘеҸҳжӣҙйҡ”зҰ»

иҝҷжҳҜдёҖз§ҚвҖңзәҜеҢ–еүҜдҪңз”ЁпјҢжҳҫејҸзҠ¶жҖҒжөҒвҖқзҡ„жһ¶жһ„е“ІеӯҰгҖӮ е…¶ж ёеҝғзӣ®ж ҮдёҚжҳҜе·Ҙе…·жҖ§дҫҝеҲ©пјҢиҖҢжҳҜзЎ®е®ҡжҖ§дёҺеҸҜиҝҪиёӘжҖ§гҖӮ

4. Vue иғҢеҗҺзҡ„зЁіе®ҡи®ҫи®ЎжЁЎејҸ

жЁЎејҸ зЁіе®ҡжҖқжғі Vue е®һзҺ°
и§ӮеҜҹиҖ…жЁЎејҸ иҮӘеҠЁдҫқиө–ж„ҹзҹҘ е“Қеә”ејҸзі»з»ҹ
еҸ‘еёғи®ўйҳ…жЁЎејҸ жқҫиҖҰеҗҲдәӢд»¶йҖҡдҝЎ $emitгҖҒдәӢд»¶жҖ»зәҝ
з»„еҗҲжЁЎејҸ жЁЎеқ—зҡ„жЁӘеҗ‘еӨҚеҗҲ з»„д»¶гҖҒComposition API
зҠ¶жҖҒжңәжЁЎејҸ з”ҹе‘Ҫе‘Ёжңҹз®ЎзҗҶ з”ҹе‘Ҫе‘Ёжңҹй’©еӯҗ
жЁЎеқ—еҢ–и®ҫи®Ў еҠҹиғҪи§ЈиҖҰдёҺеӨҚз”Ё RouterгҖҒStore жЁЎеқ—
е‘Ҫд»Өйҡ”зҰ» зәҜеҢ–еүҜдҪңз”Ё Actions / Computed

иҝҷдәӣжЁЎејҸжһ„жҲҗдәҶ Vue зҡ„гҖҢзЁіе®ҡеә•еұӮжҖқжғігҖҚпјҢеҸҜиҝҒз§»иҮід»»дҪ•е…·жңүзҠ¶жҖҒдёҺUIжҳ е°„е…ізі»зҡ„зі»з»ҹгҖӮ

5. е·ҘзЁӢеҢ–е“ІеӯҰпјҡеҸҜжҢҒз»ӯжј”еҢ–зҡ„жһ¶жһ„е№іиЎЎ

5.1 еҸҜз»ҙжҠӨжҖ§

з»„д»¶еҢ–дёҺеҚ•еҗ‘ж•°жҚ®жөҒдҪҝеҫ— Vue еә”з”Ёзҡ„иЎҢдёәз©әй—ҙеҸҜиў«жӢҶи§ЈгҖҒжөӢиҜ•дёҺжҺЁзҗҶгҖӮ вҶ’ еӨҚжқӮеәҰеҸҜеҲҶи§ЈпјҢжҳҜй•ҝжңҹеҸҜз»ҙжҠӨзҡ„еүҚжҸҗгҖӮ

5.2 еҸҜжү©еұ•жҖ§

жёҗиҝӣејҸдёҺжЁЎеқ—еҢ–жңәеҲ¶дҝқиҜҒдәҶзі»з»ҹзҡ„еұӮж¬Ўеўһй•ҝдёҚдјҡз ҙеқҸеҶ…ж ёзЁіе®ҡжҖ§гҖӮ вҶ’ еұҖйғЁеҸҜжј”иҝӣпјҢж•ҙдҪ“еҸҜе»¶з»ӯгҖӮ

5.3 еҸҜжөӢиҜ•жҖ§

еЈ°жҳҺејҸдёҺе“Қеә”ејҸзҡ„зЎ®е®ҡжҖ§пјҢдҪҝеҫ—жөӢиҜ•дёҚеҶҚдҫқиө–еүҜдҪңз”ЁпјҢиҖҢдҫқиө–зҠ¶жҖҒеҸҳеҢ–гҖӮ вҶ’ еҸҜйӘҢиҜҒеҚіеҸҜдҝЎиө–гҖӮ

6. зЁіе®ҡзҹҘиҜҶзҡ„жҠҪиұЎжҖ»з»“

зЁіе®ҡзҹҘиҜҶ жҰӮиҰҒи§ЈйҮҠ
еЈ°жҳҺејҸжҖқжғі д»Һе‘Ҫд»ӨејҸжҺ§еҲ¶иҪ¬еҗ‘ж•°жҚ®жҳ е°„пјҢйҷҚдҪҺеҝғжҷәиҙҹжӢ…
е“Қеә”ејҸжЁЎеһӢ иҮӘеҠЁдҫқиө–иҝҪиёӘдёҺжҺЁеҜјејҸзҠ¶жҖҒжөҒ
з»„д»¶еҢ–дёҺз»„еҗҲ йқўеҗ‘з»„еҗҲзҡ„з»“жһ„еӨҚз”ЁдёҺжЁЎеқ—иҫ№з•Ңжё…жҷ°еҢ–
еҚ•еҗ‘ж•°жҚ®жөҒ еҸҜйў„жөӢзҡ„ж•°жҚ®дёҖиҮҙжҖ§и®ҫи®Ў
зҠ¶жҖҒдёҖиҮҙжҖ§жЁЎеһӢ е°ҶзҠ¶жҖҒдёҺеүҜдҪңз”ЁеҲҶзҰ»пјҢдҝқиҜҒзі»з»ҹзәҜеҮҖжҖ§
жёҗиҝӣејҸжһ¶жһ„е“ІеӯҰ дҝқжҢҒеҶ…ж ёзЁіе®ҡгҖҒз”ҹжҖҒеҸҜжј”иҝӣ
жёІжҹ“и§ЈиҖҰеұӮжҖқжғі йҖҡиҝҮиҷҡжӢҹDOMе®һзҺ°и·Ёе№іеҸ°дёҖиҮҙиҜӯд№ү

7. з»“иҜӯпјҡVue дҪңдёәжһ¶жһ„е“ІеӯҰзҡ„иЎЁиҫҫ

Vue зҡ„д»·еҖјпјҢдёҚеңЁдәҺжЁЎжқҝиҜӯжі•жҲ– API ж•°йҮҸпјҢиҖҢеңЁдәҺе®ғе°ҶеүҚз«ҜејҖеҸ‘д»ҺвҖңе‘Ҫд»ӨејҸе®һзҺ°вҖқжҸҗеҚҮеҲ°дәҶвҖңе“Қеә”ејҸе»әжЁЎвҖқгҖӮ е®ғд»ЈиЎЁдәҶдёҖз§Қжӣҙй«ҳеұӮж¬Ўзҡ„зі»з»ҹжҖқз»ҙпјҡ

  • **еЈ°жҳҺејҸ вҶ’ з”ұж„Ҹеӣҫй©ұеҠЁиҖҢйқһиҝҮзЁӢй©ұеҠЁ**
  • **е“Қеә”ејҸ вҶ’ з”ұж•°жҚ®еҸҳеҢ–й©ұеҠЁиҖҢйқһдәӢд»¶и§ҰеҸ‘**
  • **з»„д»¶еҢ– вҶ’ з”ұз»„еҗҲеҪўжҲҗеӨҚжқӮзі»з»ҹиҖҢйқһз»§жүҝеҸ еҠ **

иҝҷдәӣжҖқжғідёҚд»…йҖӮз”ЁдәҺ VueпјҢд№ҹйҖӮз”ЁдәҺд»»ж„ҸеӨҚжқӮзі»з»ҹзҡ„и®ҫи®ЎгҖӮ е®ғд»¬жһ„жҲҗдәҶеүҚз«Ҝд№ғиҮіиҪҜд»¶зі»з»ҹи®ҫи®ЎдёӯдёҚйҡҸжҠҖжңҜжӣҙиҝӯиҖҢж¶ҲеӨұзҡ„зЁіе®ҡзҹҘиҜҶгҖӮ
