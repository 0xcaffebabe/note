Linux зҪ‘з»ңиҷҡжӢҹеҢ–

вҖ”вҖ”д»Һз¬¬дёҖжҖ§еҺҹзҗҶеҲ°жһ¶жһ„жЁЎеһӢзҡ„зі»з»ҹжҖ§зҗҶи§Ј

дёҖгҖҒй—®йўҳиө·зӮ№пјҡдёәд»Җд№ҲйңҖиҰҒзҪ‘з»ңиҷҡжӢҹеҢ–пјҹ

еңЁд»»дҪ•и®Ўз®—зі»з»ҹдёӯпјҢзҪ‘з»ңзҡ„жң¬иҙЁзӣ®ж ҮеҸӘжңүдёҖдёӘпјҡ

еңЁйҖ»иҫ‘дёҠзӢ¬з«Ӣзҡ„йҖҡдҝЎе®һдҪ“д№Ӣй—ҙпјҢжҸҗдҫӣеҸҜжҺ§гҖҒеҸҜжү©еұ•гҖҒеҸҜйҡ”зҰ»зҡ„йҖҡдҝЎи·Ҝеҫ„

дҪҶ Linux йқўдёҙзҡ„зҺ°е®һзәҰжқҹжҳҜпјҡ

  • дёҖдёӘзү©зҗҶжңәпјҢеҸӘжңүпјҡ

    • дёҖеҘ—зү©зҗҶзҪ‘еҚЎ
    • дёҖжқЎзңҹе®һзҡ„зү©зҗҶй“ҫи·Ҝ

  • еҚҙйңҖиҰҒеҗҢж—¶ж”ҜжҢҒпјҡ

    • еӨҡиҝӣзЁӢ
    • еӨҡе‘ҪеҗҚз©әй—ҙ
    • еӨҡиҷҡжӢҹжңә / е®№еҷЁ
    • еӨҡз§ҹжҲ·йҡ”зҰ»
    • и·Ёдё»жңәйҖҡдҝЎ

рҹ‘ү зҪ‘з»ңиҷҡжӢҹеҢ–зҡ„жң¬иҙЁпјҢе°ұжҳҜеңЁвҖңеҚ•дёҖзү©зҗҶзҪ‘з»ңиө„жәҗвҖқд№ӢдёҠпјҢжһ„йҖ вҖңеӨҡдёӘйҖ»иҫ‘зҪ‘з»ңдё–з•ҢвҖқгҖӮ

дәҢгҖҒз¬¬дёҖжҖ§еҺҹзҗҶпјҡLinux зҪ‘з»ңиҷҡжӢҹеҢ–еңЁи§ЈеҶізҡ„дёүдёӘдёҚеҸҳй—®йўҳ

жүҖжңү Linux зҪ‘з»ңиҷҡжӢҹеҢ–жҠҖжңҜпјҢж— и®әеҗҚеӯ—еӨҡд№ҲдёҚеҗҢпјҢжң¬иҙЁйғҪиҗҪеңЁд»ҘдёӢдёүдёӘзЁіе®ҡй—®йўҳз©әй—ҙдёӯпјҡ

1пёҸвғЈ йҖҡдҝЎи·Ҝеҫ„еҰӮдҪ•иў«гҖҢж„ҹзҹҘгҖҒжӢҰжҲӘгҖҒйҮҚеҶҷгҖҚпјҹ

вҖ”вҖ” жҺ§еҲ¶ж•°жҚ®вҖңжҖҺд№Ҳиө°вҖқ

  • еҜ№еә”жҠҪиұЎпјҡж•°жҚ®и·Ҝеҫ„е№Ійў„

  • еҜ№еә”жңәеҲ¶пјҡNetfilter / Hook зӮ№

  • жң¬иҙЁиғҪеҠӣпјҡ

    • жӢҰжҲӘ
    • йҮҚе®ҡеҗ‘
    • дҝ®ж”№
    • дёўејғ

2пёҸвғЈ еҰӮдҪ•еңЁеҗҢдёҖеҶ…ж ёдёӯжһ„йҖ гҖҢеӨҡдёӘзҪ‘з»ңз«ҜзӮ№гҖҚпјҹ

вҖ”вҖ” еҲ¶йҖ вҖңзңӢиө·жқҘеғҸзӢ¬з«ӢзҪ‘еҚЎвҖқзҡ„йҖ»иҫ‘жҺҘеҸЈ

  • еҜ№еә”жҠҪиұЎпјҡиҷҡжӢҹзҪ‘з»ңи®ҫеӨҮ

  • жң¬иҙЁиғҪеҠӣпјҡ

    • жҸҗдҫӣзӢ¬з«Ӣзҡ„еҸ‘йҖҒ/жҺҘж”¶и§Ҷи§’
    • з»‘е®ҡе‘ҪеҗҚз©әй—ҙ
    • йҡ”зҰ»зҪ‘з»ңиә«д»Ҫ

3пёҸвғЈ еҰӮдҪ•и·Ёи¶Ҡзү©зҗҶиҫ№з•ҢпјҢжү©еұ•зҪ‘з»ңиҜӯд№үпјҹ

вҖ”вҖ” и®©йҖ»иҫ‘зҪ‘з»ңзӘҒз ҙзү©зҗҶзҪ‘з»ңйҷҗеҲ¶

  • еҜ№еә”жҠҪиұЎпјҡйҡ§йҒ“дёҺ Overlay

  • жң¬иҙЁиғҪеҠӣпјҡ

    • е°ҒиЈ…
    • и§Је°ҒиЈ…
    • йҖ»иҫ‘зҪ‘з»ңжү©еұ•

вҡ пёҸ еҗҺж–ҮжүҖжңүжҠҖжңҜпјҢйғҪжҳҜиҝҷдёүзұ»й—®йўҳзҡ„дёҚеҗҢе·ҘзЁӢе®һзҺ°гҖӮ

дёүгҖҒLinux зҪ‘з»ңйҖҡдҝЎзҡ„зЁіе®ҡжЁЎеһӢпјҲйқһе®һзҺ°з»ҶиҠӮпјү

еңЁзҗҶи§ЈиҷҡжӢҹеҢ–д№ӢеүҚпјҢеҝ…йЎ»е…ҲжҳҺзЎ® Linux зҪ‘з»ңзҡ„зЁіе®ҡжҠҪиұЎз»“жһ„пјҡ

3.1 зҪ‘з»ңйҖҡдҝЎзҡ„дёҚеҸҜеҸҳеҲҶеұӮжЁЎеһӢ

еә”з”Ёи§Ҷи§’  вҶ“SocketпјҲйҖҡдҝЎжҠҪиұЎпјү  вҶ“дј иҫ“еұӮпјҲTCP / UDPпјү  вҶ“зҪ‘з»ңеұӮпјҲIPпјү  вҶ“зҪ‘з»ңи®ҫеӨҮжҠҪиұЎпјҲNet Deviceпјү  вҶ“й©ұеҠЁ & зү©зҗҶд»ӢиҙЁ

е…ій”®и®ӨзҹҘпјҡ

  • Linux зҪ‘з»ңиҷҡжӢҹеҢ–зҡ„еҲҮе…ҘзӮ№пјҢеҮ д№Һе…ЁйғЁеҸ‘з”ҹеңЁпјҡ

    • Net Device жҠҪиұЎ
    • IP еұӮеҸҠе…¶дёҠдёӢ

еӣӣгҖҒж•°жҚ®и·Ҝеҫ„е№Ійў„пјҡNetfilter зҡ„жҠҪиұЎж„Ҹд№ү

4.1 Netfilter дёҚжҳҜйҳІзҒ«еўҷпјҢиҖҢжҳҜгҖҢи·Ҝеҫ„жҺ§еҲ¶жЎҶжһ¶гҖҚ

Netfilter зҡ„дә”дёӘ Hook е№¶дёҚжҳҜе®һзҺ°з»ҶиҠӮпјҢиҖҢжҳҜдёҖдёӘзЁіе®ҡзҡ„и·Ҝеҫ„еҲҮеҲҶжЁЎеһӢпјҡ

HookжҠҪиұЎиҜӯд№ү
PREROUTINGж•°жҚ®еҲҡиҝӣе…Ҙзі»з»ҹ
INPUTеҸ‘еҫҖжң¬жңә
FORWARDйңҖиҰҒиҪ¬еҸ‘
OUTPUTжң¬жңәеҸ‘еҮә
POSTROUTINGеҚіе°ҶзҰ»ејҖзі»з»ҹ

рҹ‘ү иҝҷжҳҜ Linux зҪ‘з»ңвҖңеҸҜжІ»зҗҶжҖ§вҖқзҡ„еҹәзЎҖгҖӮ

4.2 зЁіе®ҡиғҪеҠӣжҖ»з»“

Netfilter жҸҗдҫӣзҡ„дёҚжҳҜеҠҹиғҪпјҢиҖҢжҳҜиғҪеҠӣпјҡ

  • зҪ‘з»ңжІ»зҗҶиғҪеҠӣ
  • еӨҡз§ҹжҲ·иҫ№з•ҢжҺ§еҲ¶
  • иҷҡжӢҹзҪ‘з»ңеҮәеҸЈжҺ§еҲ¶пјҲNATпјү

дә”гҖҒиҷҡжӢҹзҪ‘з»ңи®ҫеӨҮпјҡеңЁеҶ…ж ёдёӯеҲ¶йҖ гҖҢйҖ»иҫ‘зҪ‘еҚЎгҖҚ

5.1 иҷҡжӢҹи®ҫеӨҮзҡ„з»ҹдёҖжҠҪиұЎ

жүҖжңүиҷҡжӢҹзҪ‘еҚЎпјҢжң¬иҙЁдёҠйғҪжҳҜпјҡ

е®һзҺ° Net Device жҺҘеҸЈзҡ„вҖңиҪҜд»¶еҜ№иұЎвҖқ

5.2 tun / tapпјҡеҲ¶йҖ вҖңиҷҡжӢҹзҪ‘з»ңе…ҘеҸЈвҖқ

и®ҫеӨҮжҠҪиұЎеұӮзә§ж ёеҝғз”ЁйҖ”

  • еёёз”ЁдәҺпјҡ

    • VPN
    • з”ЁжҲ·жҖҒзҪ‘з»ңеҚҸи®®е®һзҺ°

5.3 vethпјҡиҝһжҺҘдёӨдёӘзҪ‘з»ңдё–з•Ңзҡ„вҖңиҷҡжӢҹзҪ‘зәҝвҖқ

veth зҡ„жҠҪиұЎжң¬иҙЁпјҡ

дёҖж №еҶ…ж ёдёӯзҡ„вҖңзӮ№еҜ№зӮ№д»ҘеӨӘзҪ‘зәҝвҖқ

зЁіе®ҡзү№жҖ§пјҡ

  • жҲҗеҜ№еҮәзҺ°

  • еёёз”ЁдәҺпјҡ

    • Namespace вҶ” Namespace
    • Host вҶ” Container

рҹ‘ү veth жҳҜе®№еҷЁзҪ‘з»ңзҡ„еҹәзЎҖиҝһжҺҘеҺҹиҜӯ

5.4 macvlanпјҡиҷҡжӢҹвҖңзҪ‘з»ңиә«д»ҪвҖқ

macvlan зҡ„жң¬иҙЁдёҚжҳҜй“ҫи·ҜпјҢиҖҢжҳҜпјҡ

еңЁеҗҢдёҖзү©зҗҶзҪ‘еҚЎдёҠпјҢеҲ¶йҖ еӨҡдёӘ MAC иә«д»Ҫ

йҖӮз”ЁеңәжҷҜпјҡ

  • еӨҡз§ҹжҲ·дәҢеұӮйҡ”зҰ»
  • еҜ№жҺҘдј з»ҹзҪ‘з»ңи®ҫеӨҮ

е…ӯгҖҒиҷҡжӢҹдәӨжҚўз»“жһ„пјҡжһ„йҖ е№ҝж’ӯеҹҹдёҺдәҢеұӮиҜӯд№ү

VXLAN зҡ„еҮәзҺ°е№¶йқһеҒ¶з„¶пјҢиҖҢжҳҜдёәдәҶи§ЈеҶіпјҡ

  • и·ЁдёүеұӮжү©еұ•дәҢеұӮиҜӯд№ү
  • дә‘и§„жЁЎеӨҡз§ҹжҲ·
