  • дёҚдҝқиҜҒйЎәеәҸ
  • дёҚдҝқиҜҒж—¶е»¶
  • дёҚеҢәеҲҶиҝӣзЁӢпјҢеҸӘеҢәеҲҶдё»жңә

рҹ‘ү з»“и®әпјҡ зҪ‘з»ңеұӮеҸӘи§ЈеҶідәҶвҖңдё»жңәеҲ°дё»жңәзҡ„ж•°жҚ®иҪ¬еҸ‘й—®йўҳвҖқпјҢдҪҶеә”з”ЁзңҹжӯЈйңҖиҰҒзҡ„жҳҜпјҡиҝӣзЁӢеҲ°иҝӣзЁӢзҡ„йҖҡдҝЎиҜӯд№үгҖӮ

1.2 иҝҗиҫ“еұӮзҡ„ж ёеҝғдҪҝе‘Ҫ

иҝҗиҫ“еұӮзҡ„жң¬иҙЁдҪҝе‘ҪеҸҜд»ҘжҠҪиұЎдёәдёҖеҸҘиҜқпјҡ

еңЁдёҚеҸҜйқ гҖҒж— еәҸгҖҒжӢҘеЎһзҡ„зҪ‘з»ңд№ӢдёҠпјҢжһ„йҖ дёҚеҗҢзӯүзә§зҡ„з«ҜеҲ°з«ҜйҖҡдҝЎиҜӯд№үгҖӮ

иҝҷдёҖе®ҡд№үе…·жңүй«ҳеәҰзЁіе®ҡжҖ§пјҢдёҚдҫқиө– TCP / UDP зҡ„е…·дҪ“е®һзҺ°гҖӮ

иҝҗиҫ“еұӮйңҖиҰҒи§ЈеҶізҡ„дёҚеҸҳй—®йўҳеҢ…жӢ¬пјҡ

  1. иҝӣзЁӢзә§еҜ»еқҖпјҲз«ҜеҸЈгҖҒеӨҡи·ҜеӨҚз”Ё / еҲҶи§Јпјү
  2. ж•°жҚ®е®Ңж•ҙжҖ§пјҲжҳҜеҗҰиў«з ҙеқҸпјү
  3. ж•°жҚ®жңүеәҸжҖ§пјҲжҳҜеҗҰжҢүеәҸдәӨд»ҳпјү
  4. еҸ‘йҖҒйҖҹзҺҮжҺ§еҲ¶пјҲйҒҝе…ҚеҺӢеһ®жҺҘж”¶з«Ҝпјү
  5. зҪ‘з»ңиҙҹиҪҪи°ғиҠӮпјҲйҒҝе…ҚеҺӢеһ®зҪ‘з»ңпјү
  6. з”ҹе‘Ҫе‘Ёжңҹз®ЎзҗҶпјҲиҝһжҺҘзҡ„е»әз«ӢдёҺз»Ҳжӯўпјү

1.3 дёӨз§ҚжһҒз«Ҝи®ҫи®Ўе“ІеӯҰпјҡTCP vs UDP

з»ҙеәҰ TCP UDP
и®ҫи®Ўе“ІеӯҰ ејәиҜӯд№үгҖҒејәзәҰжқҹ ејұиҜӯд№үгҖҒејәжҺ§еҲ¶жқғ
жҳҜеҗҰеҸҜйқ  жҳҜ еҗҰ
жҳҜеҗҰжңүиҝһжҺҘ жҳҜ еҗҰ
зҠ¶жҖҒ жңүзҠ¶жҖҒ ж— зҠ¶жҖҒ
еә”з”ЁжҺ§еҲ¶жқғ дҪҺ й«ҳ

рҹ‘ү е…ій”®жҠҪиұЎпјҡ

  • TCP = **зі»з»ҹжӣҝеә”з”ЁжүҝжӢ…еӨҚжқӮжҖ§**
  • UDP = **жҠҠеӨҚжқӮжҖ§е®Ңе…ЁдәӨиҝҳз»ҷеә”з”Ё**

дәҢгҖҒиҝҗиҫ“еұӮзҡ„йҖҡз”ЁжңәеҲ¶жЁЎеһӢпјҲеҸҜжӣҝжҚўзҡ„жңәеҲ¶еұӮпјү

иҝҷдёҖеұӮжҸҸиҝ°зҡ„жҳҜпјҡд»»дҪ•еҸҜйқ дј иҫ“еҚҸи®®йғҪдёҚеҸҜйҒҝе…ҚиҰҒдҪҝз”Ёзҡ„жңәеҲ¶пјҢдёҺ TCP е®һзҺ°ж— е…ігҖӮ

2.1 з«ҜеҲ°з«ҜеҸҜйқ жҖ§зҡ„еӣӣиҰҒзҙ 

дёҖдёӘеҸҜйқ ж•°жҚ®дј иҫ“еҚҸи®®пјҢжңҖе°Ҹеҝ…еӨҮиҰҒзҙ пјҡ

  1. **ж ЎйӘҢжңәеҲ¶**пјҡжЈҖжөӢжҜ”зү№й”ҷиҜҜ
  2. **еәҸеҸ·жңәеҲ¶**пјҡиҜҶеҲ«д№ұеәҸдёҺйҮҚеӨҚ
  3. **зЎ®и®ӨжңәеҲ¶пјҲACK/NACKпјү**пјҡеҪўжҲҗеҸҚйҰҲй—ӯзҺҜ
  4. **е®ҡж—¶еҷЁжңәеҲ¶**пјҡеӨ„зҗҶдёўеӨұдёҺдёҚзЎ®е®ҡжҖ§

иҝҷжҳҜиҝҗиҫ“еұӮзҡ„вҖңжңҖе°ҸеҸҜйқ жҖ§е…¬зҗҶзі»з»ҹвҖқгҖӮ

2.2 д»ҺеҒңзӯүеҲ°жөҒж°ҙзәҝпјҡж•ҲзҺҮзҡ„еҝ…з„¶жј”иҝӣ

еҒңжӯўзӯүеҫ…пјҲStop-and-Waitпјү

  • жҜҸж¬ЎеҸӘе…Ғи®ёдёҖдёӘжңӘзЎ®и®ӨеҲҶз»„
  • з®ҖеҚ•дҪҶеҗһеҗҗжһҒдҪҺ

рҹ‘ү жң¬иҙЁй—®йўҳпјҡй“ҫи·ҜеҲ©з”ЁзҺҮиў« RTT й”Ғжӯ»гҖӮ

жөҒж°ҙзәҝжҖқжғі

  • еҗҢж—¶е…Ғи®ёеӨҡдёӘжңӘзЎ®и®ӨеҲҶз»„еңЁзҪ‘з»ңдёӯ
  • еҸ‘йҖҒж–№ / жҺҘж”¶ж–№йңҖиҰҒзј“еӯҳиғҪеҠӣ

з”ұжӯӨиҮӘз„¶жј”еҢ–еҮәдёӨзұ»й”ҷиҜҜжҒўеӨҚзӯ–з•Ҙпјҡ

зӯ–з•Ҙ ж ёеҝғжҖқжғі д»Јд»·
GBN зҙҜз§ҜзЎ®и®ӨпјҢеҮәй”ҷеӣһйҖҖ йҮҚдј еӨҡ
SR зІҫзЎ®зЎ®и®ӨпјҢйҖүжӢ©йҮҚдј  зҠ¶жҖҒеӨҚжқӮ

рҹ‘ү жҠҪиұЎз»“и®әпјҡ иҝҷжҳҜ**вҖңеёҰе®ҪеҲ©з”ЁзҺҮвҖқдёҺвҖңе®һзҺ°еӨҚжқӮеәҰвҖқд№Ӣй—ҙзҡ„з»Ҹе…ёжқғиЎЎ**гҖӮ

2.3 ж»‘еҠЁзӘ—еҸЈпјҡз»ҹдёҖжөҒжҺ§дёҺеҸҜйқ жҖ§зҡ„з»“жһ„

ж»‘еҠЁзӘ—еҸЈдёҚжҳҜдёҖдёӘвҖңTCP дё“еұһжҠҖе·§вҖқпјҢиҖҢжҳҜпјҡ

еңЁж—¶е»¶еӯҳеңЁзҡ„зі»з»ҹдёӯпјҢеҗҢж—¶жҺ§еҲ¶е№¶еҸ‘еәҰдёҺйЎәеәҸжҖ§зҡ„йҖҡз”Ёз»“жһ„гҖӮ

е®ғеҗҢж—¶жүҝжӢ…дёүз§ҚиҒҢиҙЈпјҡ

  • йҷҗеҲ¶жңӘзЎ®и®Өж•°жҚ®йҮҸпјҲжөҒжҺ§пјү
  • ж”ҜжҢҒжөҒж°ҙзәҝе№¶еҸ‘
  • жҸҗдҫӣйҮҚдј дёҺзЎ®и®Өиҫ№з•Ң

2.4 жөҒйҮҸжҺ§еҲ¶ vs жӢҘеЎһжҺ§еҲ¶пјҲе…ій”®еҢәеҲҶпјү

жҺ§еҲ¶еҜ№иұЎ зӣ®ж Ү еҸҚйҰҲжқҘжәҗ
жөҒйҮҸжҺ§еҲ¶ дёҚеҺӢеһ®жҺҘж”¶з«Ҝ жҺҘж”¶з«ҜзӘ—еҸЈ
жӢҘеЎһжҺ§еҲ¶ дёҚеҺӢеһ®зҪ‘з»ң RTT / дёўеҢ…

рҹ‘ү иҝҷжҳҜиҝҗиҫ“еұӮдёӯжңҖе®№жҳ“иў«ж··ж·ҶгҖҒдҪҶеҝ…йЎ»дёҘж јеҢәеҲҶзҡ„дёҖзӮ№гҖӮ

дёүгҖҒTCPпјҡдёҖз§Қе·ҘзЁӢеҢ–зҡ„еҸҜйқ дј иҫ“е®һзҺ°пјҲе®һзҺ°еұӮпјү

3.1 TCP зҡ„иғҪеҠӣжӢҶи§ЈпјҲиҖҢйқһеҠҹиғҪзҪ—еҲ—пјү

иғҪеҠӣ зі»з»ҹзӣ®ж Ү дё»иҰҒжңәеҲ¶
еҸҜйқ жҖ§ дёҚдёўдёҚд№ұ еәҸеҸ· + ACK + йҮҚдј 
жөҒйҮҸжҺ§еҲ¶ жҺҘж”¶з«ҜдҝқжҠӨ жҺҘж”¶зӘ—еҸЈ
жӢҘеЎһжҺ§еҲ¶ зҪ‘з»ңзЁіе®ҡ cwnd + RTT
иҝһжҺҘз®ЎзҗҶ зҠ¶жҖҒеҗҢжӯҘ дёүж¬ЎжҸЎжүӢ / еӣӣж¬ЎжҢҘжүӢ

3.2 TCP йҰ–йғЁзҡ„з»“жһ„ж„Ҹд№ү

TCP йҰ–йғЁе№¶дёҚжҳҜеӯ—ж®өе Ҷз ҢпјҢиҖҢжҳҜпјҡ

  • еәҸеҸ·з©әй—ҙпјҡеҸҜйқ жҖ§зҡ„еҹәзҹі
  • ACKпјҡеҸҚйҰҲжҺ§еҲ¶еӣһи·Ҝ
  • WindowпјҡжҺҘж”¶иғҪеҠӣеЈ°жҳҺ
  • FlagsпјҡиҝһжҺҘзҠ¶жҖҒжңәдҝЎеҸ·

рҹ‘ү жҜҸдёҖдёӘеӯ—ж®өйғҪжңҚеҠЎдәҺвҖңз«ҜеҲ°з«ҜеҸҚйҰҲжҺ§еҲ¶зі»з»ҹвҖқиҝҷдёҖжҖ»дҪ“и®ҫи®ЎгҖӮ

3.3 RTT дёҺи¶…ж—¶пјҡдёҚзЎ®е®ҡзҪ‘з»ңдёӯзҡ„дј°и®Ўй—®йўҳ

RTT жҳҜдёҖдёӘйҡҸжңәеҸҳйҮҸпјҢиҖҢйқһеёёйҮҸгҖӮ

TCP дҪҝз”ЁжҢҮж•°еҠ жқғз§»еҠЁе№іеқҮпјҡ

EstimatedRTT = (1 - Оұ) * EstimatedRTT + Оұ * SampleRTT

рҹ‘ү жң¬иҙЁпјҡ

  • з”ЁеҺҶеҸІж ·жң¬еҜ№жңӘжқҘиҝӣиЎҢжҰӮзҺҮжҖ§дј°и®Ў
  • и¶…ж—¶ = йЈҺйҷ©жҺ§еҲ¶пјҢиҖҢйқһзІҫзЎ®и®Ўж—¶

3.4 жӢҘеЎһжҺ§еҲ¶пјҡдёҖдёӘеҲҶеёғејҸеҸҚйҰҲжҺ§еҲ¶зі»з»ҹ

TCP жӢҘеЎһжҺ§еҲ¶йҒөеҫӘдёүжқЎдёҚеҸҳеҺҹеҲҷпјҡ

  1. дёўеҢ… / е»¶иҝҹдёҠеҚҮ вҶ’ еҮҸйҖҹ
  2. жҲҗеҠҹзЎ®и®Ө вҶ’ еҠ йҖҹ
  3. е§Ӣз»Ҳд»ҘвҖңиҜ•жҺўвҖқж–№ејҸеҚ з”ЁеёҰе®Ҫ

з»Ҹе…ёйҳ¶ж®өпјҡ

  • ж…ўеҗҜеҠЁпјҲжҢҮж•°жҺўжөӢпјү
  • жӢҘеЎһйҒҝе…ҚпјҲзәҝжҖ§еўһй•ҝпјү
  • еҝ«йҮҚдј  / еҝ«жҒўеӨҚпјҲеҮҸе°‘з©әиҖ—пјү

рҹ‘ү жң¬иҙЁи§Ҷи§’пјҡ TCP жҳҜдёҖдёӘеңЁдёҚе®Ңе…ЁдҝЎжҒҜдёӢиҝҗиЎҢзҡ„иҮӘйҖӮеә”жҺ§еҲ¶зі»з»ҹгҖӮ

3.5 иҝһжҺҘз®ЎзҗҶдёҺ TIME_WAIT зҡ„зі»з»ҹж„Ҹд№ү

TIME_WAIT е№¶йқһвҖңи®ҫи®Ўзјәйҷ·вҖқпјҢиҖҢжҳҜзі»з»ҹжӯЈзЎ®жҖ§зҡ„д»Јд»·гҖӮ

е®ғи§ЈеҶідёӨдёӘж №й—®йўҳпјҡ

  1. зЎ®дҝқе…ЁеҸҢе·ҘиҝһжҺҘеҸҜйқ з»Ҳжӯў
  2. йҳІжӯўж—§иҝһжҺҘзҡ„иҝ·йҖ”жҠҘж–ҮжұЎжҹ“ж–°иҝһжҺҘ

рҹ‘ү иҝҷжҳҜз”Ёж—¶й—ҙжҚўеҸ–е…ЁеұҖдёҖиҮҙжҖ§зҡ„з»Ҹе…ёзі»з»ҹи®ҫи®ЎгҖӮ

еӣӣгҖҒLinux TCPпјҡж“ҚдҪңзі»з»ҹеұӮйқўзҡ„жІ»зҗҶдёҺи°ғдјҳ

иҝҷдёҖеұӮеұһдәҺе·ҘзЁӢе®һзҺ°еұӮпјҢдёҚе…·еӨҮи·Ёзі»з»ҹзЁіе®ҡжҖ§пјҢдҪҶе…·еӨҮе®һи·өд»·еҖјгҖӮ

4.1 еҸӮж•°зҡ„жӯЈзЎ®зҗҶи§Јж–№ејҸ

дёҚиҰҒеӯӨз«ӢзңӢеҸӮж•°пјҢиҖҢиҰҒе»әз«Ӣжҳ е°„е…ізі»пјҡ

еӯҗзі»з»ҹ жҺ§еҲ¶зӣ®ж Ү е…ёеһӢеҸӮж•°
е»әиҝһ жҠ—жҙӘжіӣ syn_backlog / syncookies
ж–ӯиҝһ иө„жәҗеӣһж”¶ fin_timeout / tw_buckets
еҗһеҗҗ еёҰе®ҪеҲ©з”Ё wmem / rmem
е»¶иҝҹ RTT зЁіе®ҡ timestamps

4.2 е·ҘзЁӢеҺҹеҲҷ

  • еҸӮж•°дёҚжҳҜвҖңи°ғеҫ—и¶ҠеӨ§и¶ҠеҘҪвҖқ
  • жүҖжңүи°ғдјҳйғҪжҳҜпјҡ**еҗһеҗҗгҖҒе»¶иҝҹгҖҒиө„жәҗгҖҒйЈҺйҷ©зҡ„жқғиЎЎ**
  • ж•°жҚ®дёӯеҝғеҒҮи®ҫ вү  е…¬зҪ‘еҒҮи®ҫ

дә”гҖҒиҝҗиҫ“еұӮзҡ„жј”иҝӣдёҺжңӘжқҘпјҲж—¶й—ҙз»ҙеәҰпјү

5.1 дёәд»Җд№ҲдјҡеҮәзҺ° QUIC

  • TCP жј”иҝӣеҸ—еҲ¶дәҺеҶ…ж ё
  • иҝһжҺҘиҝҒз§»гҖҒеҝ«йҖҹе»әиҝһйңҖжұӮдёҠеҚҮ
  • з”ЁжҲ·жҖҒеҚҸи®®жӣҙзҒөжҙ»

рҹ‘ү QUIC дёҚжҳҜеҗҰе®ҡ TCPпјҢиҖҢжҳҜиҝҗиҫ“еұӮвҖңдёҠз§»вҖқзҡ„з»“жһңгҖӮ

5.2 жӢҘеЎһжҺ§еҲ¶зҡ„жј”иҝӣж–№еҗ‘

  • RenoпјҡдёўеҢ…й©ұеҠЁ
  • CUBICпјҡй«ҳеёҰе®ҪеҸӢеҘҪ
  • BBRпјҡжЁЎеһӢй©ұеҠЁпјҲеёҰе®Ҫ Г— RTTпјү
