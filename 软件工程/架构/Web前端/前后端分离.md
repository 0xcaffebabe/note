# 前后端分离

## 传统开发

需求-UI-前端-后端-集成-测试-交付

## 分离

需求-UI-前后端约定接口并行开发-集成-测试-交付

## 注意事项

- 前后端项目必须独立开发 独立部署 独立存放 通过接口进行合作

## Mock.js

mock js 数据

- 入门案例

```js
let Mock = require('mockjs')

let data = Mock.mock(
{
        'list|5':[{id:1,name:'kd'}]
}
);
```
