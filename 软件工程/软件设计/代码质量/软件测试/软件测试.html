д»ҘжңүйҷҗжҲҗжң¬пјҢжңҖеӨ§еҢ–жҡҙйңІзі»з»ҹйЈҺйҷ©пјҢе№¶дёәдә§е“ҒдёҺе·ҘзЁӢеҶізӯ–жҸҗдҫӣеҸҜйқ зҡ„иҙЁйҮҸдҝЎеҸ·гҖӮ

з”ұжӯӨеј•з”іеҮәжөӢиҜ•дёҺи°ғиҜ•зҡ„жң¬иҙЁеҢәеҲҶпјҡ

  • **жөӢиҜ•пјҲTestingпјү**пјҡеҸ‘зҺ°й—®йўҳгҖҒжҡҙйңІйЈҺйҷ©
  • **и°ғиҜ•пјҲDebuggingпјү**пјҡе®ҡдҪҚеҺҹеӣ гҖҒдҝ®еӨҚй—®йўҳ

1.2 жөӢиҜ•зҡ„з¬¬дёҖжҖ§еҺҹзҗҶ

  1. дёҚе®Ңе…ЁжҖ§еҺҹзҗҶпјҡ

    • жөӢиҜ•ж°ёиҝңж— жі•з©·е°ҪжүҖжңүиҫ“е…ҘдёҺзҠ¶жҖҒ
    • жөӢиҜ•зҡ„жң¬иҙЁжҳҜвҖңжҠҪж · + йЈҺйҷ©йҖүжӢ©вҖқ

  2. е…Қз–«жҖ§еҺҹзҗҶпјҡ

    • жөӢиҜ•иЎҢдёәжң¬иә«дјҡж”№еҸҳзі»з»ҹзҠ¶жҖҒ
    • зі»з»ҹдјҡеҜ№е·ІиҰҶзӣ–и·Ҝеҫ„дә§з”ҹвҖңе…Қз–«вҖқ

  3. еҸҚйҰҲе‘ЁжңҹеҺҹзҗҶпјҡ

    • жөӢиҜ•и¶Ҡйқ еүҚпјҢеҸҚйҰҲи¶Ҡеҝ«пјҢдҝ®еӨҚжҲҗжң¬и¶ҠдҪҺ

  4. йЈҺйҷ©дјҳе…ҲеҺҹзҗҶпјҡ

    • жөӢиҜ•иө„жәҗеә”дјҳе…ҲжҠ•е…Ҙй«ҳйЈҺйҷ©гҖҒй«ҳеҸҳеҢ–еҢәеҹҹ

  5. еҸҜжөӢжҖ§еҺҹзҗҶпјҡ

    • зі»з»ҹжҳҜеҗҰвҖңеҘҪжөӢвҖқпјҢеҸ–еҶідәҺи®ҫи®ЎдёҺд»Јз ҒиҙЁйҮҸ

дәҢгҖҒжөӢиҜ•зҡ„еҲҶзұ»з»ҙеәҰдёҺзЁіе®ҡи®ӨзҹҘжЁЎеһӢпјҲWhatпјү

2.1 жөӢиҜ•еҲҶзұ»зҡ„еӨҡз»ҙи§Ҷи§’

жөӢиҜ•дёҚжҳҜеҚ•дёҖз»ҙеәҰеҲ’еҲҶпјҢиҖҢжҳҜеӨҡдёӘжӯЈдәӨз»ҙеәҰзҡ„з»„еҗҲпјҡ

  • **жҢүжү§иЎҢж–№ејҸ**пјҡйқҷжҖҒжөӢиҜ• / еҠЁжҖҒжөӢиҜ•
  • **жҢүд»Јз ҒеҸҜи§ҒжҖ§**пјҡй»‘зӣ’ / зҷҪзӣ’
  • **жҢүжү§иЎҢдё»дҪ“**пјҡжүӢе·Ҙ / иҮӘеҠЁеҢ–
  • **жҢүе…іжіЁеҜ№иұЎ**пјҡжҠҖжңҜжөӢиҜ• / дёҡеҠЎжөӢиҜ•

2.2 жөӢиҜ•еұӮзә§жЁЎеһӢпјҡжөӢиҜ•йҮ‘еӯ—еЎ”

жөӢиҜ•йҮ‘еӯ—еЎ”жҸӯзӨәзҡ„жҳҜжҲҗжң¬дёҺеҸҚйҰҲйҖҹеәҰзҡ„з»“жһ„жҖ§и§„еҫӢпјҡ

  • еҚ•е…ғжөӢиҜ•пјҡ

    • ж•°йҮҸжңҖеӨҡ
    • йҡ”зҰ»еәҰй«ҳгҖҒеҸҚйҰҲеҝ«гҖҒжҲҗжң¬дҪҺ

  • жңҚеҠЎ/йӣҶжҲҗжөӢиҜ•пјҡ

    • йӘҢиҜҒз»„д»¶еҚҸдҪң
    • е№іиЎЎзңҹе®һжҖ§дёҺж•ҲзҺҮ

  • UI / з«ҜеҲ°з«ҜжөӢиҜ•пјҡ

    • иҰҶзӣ–зңҹе®һз”ЁжҲ·и·Ҝеҫ„
    • жҲҗжң¬й«ҳгҖҒж•°йҮҸеә”еҸ—жҺ§

ж ёеҝғз»“и®әпјҡ

жөӢиҜ•и¶Ҡйқ дёҠпјҢеҸҚйҰҲи¶Ҡж…ўпјҢж•°йҮҸеә”е‘Ҳж•°йҮҸзә§йҖ’еҮҸгҖӮ

2.3 жөӢиҜ•е…іжіЁйқўжЁЎеһӢпјҡж•ҸжҚ·жөӢиҜ•иұЎйҷҗ

иҜҘжЁЎеһӢд»ҺдёӨдёӘз»ҙеәҰз»„з»ҮжөӢиҜ•и®ӨзҹҘпјҡ

  • йқўеҗ‘дёҡеҠЎ вҶ” йқўеҗ‘жҠҖжңҜ
  • ж”ҜжҢҒејҖеҸ‘ вҶ” иҜ„дј°дә§е“Ғ
иұЎйҷҗ ж ёеҝғд»·еҖј
Q1 ж”ҜжҢҒејҖеҸ‘зҡ„жҠҖжңҜжөӢиҜ•
Q2 дёҡеҠЎйӘҢж”¶дёҺеҠҹиғҪйӘҢиҜҒ
Q3 з”ЁжҲ·дҪ“йӘҢдёҺжҺўзҙў
Q4 йқһеҠҹиғҪдёҺиҙЁйҮҸеұһжҖ§

дёүгҖҒжөӢиҜ•зҡ„з”ҹе‘Ҫе‘ЁжңҹдёҺжөҒзЁӢдҪ“зі»пјҲHowпјү

3.1 жөӢиҜ•е…Ёз”ҹе‘Ҫе‘ЁжңҹжЁЎеһӢ

жөӢиҜ•иҙҜз©ҝж•ҙдёӘиҪҜд»¶з”ҹе‘Ҫе‘ЁжңҹпјҢиҖҢйқһзӢ¬з«Ӣйҳ¶ж®өпјҡ

йҳ¶ж®ө ж ёеҝғе…іжіЁ дё»иҰҒдә§еҮә
йңҖжұӮйҳ¶ж®ө йЈҺйҷ©гҖҒеҸҜжөӢжҖ§ жөӢиҜ•зӯ–з•Ҙ
и®ҫи®Ўйҳ¶ж®ө иҰҶзӣ–жЁЎеһӢ жөӢиҜ•з”ЁдҫӢи®ҫи®Ў
е®һзҺ°йҳ¶ж®ө иҮӘеҠЁеҢ–гҖҒж•°жҚ® жөӢиҜ•иө„дә§
жү§иЎҢйҳ¶ж®ө еҸҚйҰҲгҖҒзӣ‘жҺ§ иҙЁйҮҸдҝЎеҸ·
дёҠзәҝеҗҺ еӣһеҪ’гҖҒжј”иҝӣ иҙЁйҮҸй—ӯзҺҜ

3.2 жөӢиҜ•жөҒзЁӢзҡ„е»әжЁЎи§Ҷи§’

жөӢиҜ•жң¬иҙЁдёҠжҳҜдёҖдёӘвҖңе»әжЁЎвҖ”йӘҢиҜҒвҖ”еҸҚйҰҲвҖқзҡ„й—ӯзҺҜпјҡ

  • **зҺҜеўғ вҶ’ зҺҜеўғжЁЎеһӢ**
  • **иў«жөӢеҜ№иұЎ вҶ’ зі»з»ҹжЁЎеһӢ**
  • **дәәе‘ҳз»ҸйӘҢ вҶ’ й”ҷиҜҜжЁЎеһӢ**

дёүзұ»жЁЎеһӢе…ұеҗҢй©ұеҠЁжөӢиҜ•жү§иЎҢпјҢе№¶йҖҡиҝҮз»“жһңдёҚж–ӯдҝ®жӯЈиҮӘиә«гҖӮ

еӣӣгҖҒжөӢиҜ•и®ҫи®Ўж–№жі•дёҺиҰҶзӣ–дҪ“зі»

4.1 жөӢиҜ•з”ЁдҫӢи®ҫи®ЎеҺҹеҲҷ

  • иҰҶзӣ–йңҖжұӮпјҢиҖҢйқһд»Јз ҒиЎЁйқў
  • зӯүд»·зұ»дёҺиҫ№з•ҢеҖјдјҳе…Ҳ
  • еңәжҷҜй©ұеҠЁиҖҢйқһеӯӨз«ӢеҠҹиғҪ

4.2 з»Ҹе…ёжөӢиҜ•и®ҫи®Ўж–№жі•

  • зӯүд»·зұ»еҲ’еҲҶ
  • иҫ№з•ҢеҖјеҲҶжһҗ
  • еңәжҷҜжі•
  • й”ҷиҜҜжҺЁжөӢжі•

4.3 иҰҶзӣ–зҺҮзҡ„е·ҘзЁӢеҢ–зҗҶи§Ј

  • **йңҖжұӮиҰҶзӣ–зҺҮ**пјҡжҳҜеҗҰйӘҢиҜҒдәҶжүҖжңүдёҡеҠЎжүҝиҜә
  • **д»Јз ҒиҰҶзӣ–зҺҮ**пјҡе“ӘдәӣйҖ»иҫ‘иў«жү§иЎҢиҝҮ

иҰҶзӣ–зҺҮжҳҜдҝЎеҸ·пјҢиҖҢйқһзӣ®ж Үжң¬иә«гҖӮ

дә”гҖҒжөӢиҜ•е·ҘзЁӢеҢ–дёҺиҙЁйҮҸеҹәзЎҖи®ҫж–Ҫ

5.1 жөӢиҜ•еҹәзЎҖжһ¶жһ„

жөӢиҜ•жһ¶жһ„зҡ„зӣ®ж ҮдёҚжҳҜвҖңи·‘жөӢиҜ•вҖқпјҢиҖҢжҳҜпјҡ

  • жҸҗеҚҮжү§иЎҢж•ҲзҺҮ
  • ж”Ҝж’‘е№¶еҸ‘дёҺжү©еұ•
  • дёҺ CI/CD ж·ұеәҰйӣҶжҲҗ

5.2 жөӢиҜ•ж•°жҚ®е·ҘзЁӢ

жөӢиҜ•ж•°жҚ®жҳҜжөӢиҜ•зі»з»ҹзҡ„вҖңзҮғж–ҷвҖқпјҡ

  • On-the-flyпјҡзңҹе®һжҖ§й«ҳпјҢжҲҗжң¬й«ҳ
  • Out-of-boxпјҡж•ҲзҺҮй«ҳпјҢз»ҙжҠӨжҲҗжң¬йңҖжҺ§еҲ¶

жөӢиҜ•ж•°жҚ®йңҖиҰҒпјҡ

  • жңҚеҠЎеҢ–
  • е…ғж•°жҚ®еҢ–
  • иҙЁйҮҸеҸҜжҺ§

е…ӯгҖҒжөӢиҜ•зұ»еһӢдҪ“зі»пјҲSystem Viewпјү

6.1 зі»з»ҹзә§жөӢиҜ•зұ»еһӢ

  • еҠҹиғҪжөӢиҜ•
  • еӣһеҪ’жөӢиҜ•
  • жҺўзҙўејҸжөӢиҜ•
  • дј—жөӢ

6.2 йқһеҠҹиғҪжҖ§жөӢиҜ•

  • жҖ§иғҪ / еҺӢеҠӣ / иҙҹиҪҪ
  • зЁіе®ҡжҖ§
  • е…је®№жҖ§
  • е®үе…ЁжөӢиҜ•

6.3 з§»еҠЁз«Ҝдё“йЎ№жөӢиҜ•

  • е®үиЈ…еҚҮзә§
  • ејұзҪ‘дёҺдёӯж–ӯ
  • жҖ§иғҪдёҺеҠҹиҖ—
  • з”ЁжҲ·дҪ“йӘҢ

дёғгҖҒиҙЁйҮҸе·ҘзЁӢзҡ„жј”иҝӣи·Ҝеҫ„пјҲWhereпјү

7.1 жөӢиҜ•иғҪеҠӣжҲҗзҶҹеәҰжЁЎеһӢ

зӯүзә§ зү№еҫҒ
L1 дәӢеҗҺжүӢе·ҘжөӢиҜ•
L2 йҳ¶ж®өеҢ–жөӢиҜ•жөҒзЁӢ
L3 иҮӘеҠЁеҢ– + е№іеҸ°
L4 ж•°жҚ®й©ұеҠЁ + жңҚеҠЎеҢ–
L5 жҷәиғҪжөӢиҜ• + еҶізӯ–ж”ҜжҢҒ

7.2 е…ҲиҝӣжөӢиҜ•иҢғејҸ

  • зІҫеҮҶжөӢиҜ•пјҲеҹәдәҺиҰҶзӣ–дёҺж•°жҚ®еҲҶжһҗпјү
  • жҷәиғҪеӣһеҪ’пјҲйҖүжӢ©жҖ§жү§иЎҢпјү
  • еҹәдәҺжЁЎеһӢзҡ„жөӢиҜ•пјҲMBTпјү

е…«гҖҒдјҳз§ҖжөӢиҜ•е·ҘзЁӢеёҲзҡ„жҖқз»ҙжЁЎеһӢ

  • зі»з»ҹжҖ§пјҡд»Һж•ҙдҪ“зңӢиҙЁйҮҸ
  • ж•Ҹж„ҹжҖ§пјҡеҝ«йҖҹиҜҶеҲ«ејӮеёёдҝЎеҸ·
  • еҘҪеҘҮеҝғпјҡжҢҒз»ӯжҺўзҙўжңӘзҹҘи·Ҝеҫ„
  • е·ҘзЁӢзҗҶжҖ§пјҡжқғиЎЎжҲҗжң¬дёҺж”¶зӣҠ
